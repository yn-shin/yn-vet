# 🔥 2️⃣ 조직 → 종양 연결 (너가 원한 부분)

```markdown

- stratified squamous → SCC
- glandular epithelium → adenocarcinoma

```

---

## 예시: 간엽

```markdown
## 간엽 → 종양

- bone → osteosarcoma
- cartilage → chondrosarcoma
- muscle → rhabdomyosarcoma
```

## 예시: round cell

```markdown
## round cell tumor

- lymphoma
- mast cell tumor
- plasma cell tumor
- histiocytoma
```
