- 점막 구조가 사실상 외부에 노출된 구조임.

## 3. Defense Mechanisms

#### 🟢 Alimentary System의 Defense Mechanisms

- Mechanical (physical) : 두꺼운 층 → 소화기 내부의 문제가 밖으로 나가지 않게 함. / 조임근 → barrier (위의 문제가 급격히 소장으로 넘어가지 x)
- Biochemical : enzymes → 음식물뿐만 아니라 감염성 원인체를 녹이기도 함 / pH 조절 → 소장과 위의 pH가 달라서 살 수 있는 세균이 다름
- Immunologic : GALT, immune cells 정상세균총은 면역 세포가 건드리지 않는데,

## 4. Portals of Entry

소화기로 들어오는 통로는 크게 4가지.
- 먹거나 (ingestion)
- 기침으로 인한 유입 (cough-induced ingestion)
- 혈중으로 전신성 (systemic spread via bloodstream) : 가장 심각
- 기생충 유입 (parasite migration)



| 소화기 유입 경로                                                                                                 |
| --------------------------------------------------------------------------------------------------------- |
| (1) **Ingestion**                                                                                         |
| - 가장 common 루트, 오염된 음식과 물이 문제.<br>- *E. coli*, *Salmonella spp.* : 입으로 먹어서 소화기 증상 유발<br>- 위산 분비 감소, 효소 감소 |
| (2) **Cough-Induced Ingestion**                                                                           |
| - 호흡기를 통해 들어옴<br>- *Rhodococcus equi* : 말에서 호흡기 증상을 먼저 유발. 기침으로 인해 호흡기균이 소화기로 넘어감.                        |
| (3) **Systemic Spread : Blood-Borne Infections**                                                          |
| - *Clostridium perfringes* : 혈중으로 매우 잘 퍼짐                                                                 |
| (4) **Mucosal Breach**                                                                                    |
| 안전막을 통해 들어옴 > 감염 경로를 특정할 수 없음.                                                                            |
| (5) **Oral Cavity**                                                                                       |
| 치석 관리 중요, 치주염 유발할 수 있음<br>구강 청결                                                                           |

- Tonsil의 역할
	- **Tonsil**은 <u>afferent</u> lymphatic vessels가 없음: tonsil은 들어오는 경로는 없고, 만들어서 내보내기만 한다.
	- Tonsil"혈관을 통해 들어왔다" 가 맞는 설명.
	- Feline Parvovirus : tonsil에 잘 증식.
	- BVDV : tonsil을 통해 전신 혈류로 퍼져나감.

## 5. Response to Injury

어떤 병원체가 주로 염증, 혹은 괴사, 

#### 1️⃣ Inflammation
- Acute inflammation 
	- 급성 염증을 유발하는 대표적인 감염원: *Salmonella*, *E. coli*
	- 부종, 충혈, 혈관 손상 등이 유발된다
- Chronic inflmmation
	- 대표적으로 Johne's disease

![[Pasted image 20251107220652.png|600]]
#### 2️⃣ Necrosis
- 감염성 원인체에 의한 죽음은 apoptosis(계획된 죽음)이 아니라 **necrosis**이다.
- 소장 층에서 안쪽에는 crypt(줄기세포), 위에는 villi가 있음.
	- **Ulcer** : crypt에서 villi까지 전 층이 다 손상된 것.
	- Erosion (미란) : 일부 층만 손상
- 점막이 손상되었다는 건 barrier가 깨졌다는 것.
- lymphoid tissue가 파괴되면 면역 반응이 고장.
- villi가 영양분을 흡수해야 하는데 점막이 손상되면 영양분 흡수가 저해됨.

#### 3️⃣ Lymphangiectasia & PLE
- 관 구조로 된 장기는 폐색이 일어나면
- 림프관 <u>주변</u>의 lymphoma나 chronic inflammation ⇒ 림프관의 물리적 압박 ⇒ 폐색 ⇒ 이후 림프관이 확장된다.
- 갇혀있다 보면 투과성 변화를 유도해서 주변 내강으로 새어나가게 된다.
- 임상 증상 : 설사, 체중 감소, protein-losing enteropathy
- 간부전일 때, 장의 문제가 있을 때 잘 관찰됨.

#### 5️⃣ **Diarrhea**


- 정의 : Abnormally fluid feces accompanied by increased volume and frequency

- **Inflammatory** (염증성) : 직접적으로 cytokine을 분비해 염증을 유발하는 경우
	- *Salmonella*, *Lawsonia* 등
- **Non-inflammatory** (비염증성) : 염증을 유발하지 않되 세포를 직접적으로 파괴, 세포가 가진 것을 이용해서 설사를 유발함

- 원인
	- 감염성 원인체
		- 호흡기 감염 원인체가 기침을 통해 소화기로 재반입될 수도 있음
	- 위산 분비 감소

	- (Toxin이 channel을 통해..) Chloride를 퍼내면서 물이 같이 나감, 능동적 형태

과도한 운동 : 흡수할 시간을 제한 : 삼투성 설사

소화기는 몸의 외부이기 때문에, 천공(식도천공, 위천공, 장천공) ⇒ 이물의 복강내 유입 ⇒ 복막염을 유발할 수 있다.

## 3. Stomach & Abomasum

반추동물의 4위(Abomasum)가 단위동물의 위와 구조적으로도, 기능적으로도 비슷함.

|                🔎 질환별 내용                 | <                                                                                                                                                                                                                                                                                                                                              |
| :--------------------------------------: | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|       **Gastric Ulcers**<br>(위 궤양)       | - 궤양은 미란과 달리 점막의 맨 위층부터 바닥까지 다 손상.<br>- 조직학적으로 점막의 연결성이 끊어진 부분을 볼 수 있다.<br>- 원인<br>　▪ 항생제, NSAIDs 등 약물을 쓰면 위궤양이 잘 생긴다. (위점액을 만드는 prostaglandins를 억제하기 때문)<br>　▪ *Helicobacter* spp. 감염 - 개/고양이에서는 드물다.<br>　▪ 스트레스, 갑작스러운 식이 변화<br>- 궤양, 미란 ⇒ 필연적으로 주변에 <u>염증세포가 동원</u>되어 있다. (없으면 그냥 artifact)<br>- 궤양은 천공, 출혈을 자주 동반한다. 위 내강에 소화물+출혈소가 섞여 있음. |
|        **Abomasal Displacement**         | - RDA (**Right** displaced abomasum) : 오른쪽으로 돌 때 **volvulus** 의 위험이 더 큼.<br>- LDA (Left displaced abomasum) : 더 흔함, gas 축적<br>- 분만 후 과도한 에너지가 필요할 때 더 잘 일어난다.<br>- 운동성 저하(hypomobility), hypocalcemia                                                                                                                                          |
| **GDV**<br>(Gastric Dilation & Volvulus) | - 흉곽이 깊은 종 : Great Danes, German Shepherds 에서 잘 발생<br>- 급격한 운동, 과도한 급이 시 전위 발생 → 음식물이 계속 축적되며 발효, 음식물과 gas, fluid가 쌓여 위가 부풀고, 혈액이 정체된다. (→ 혈전, 괴사, 출혈)<br>⇒ 육안 소견 : <u>위벽이 얇아짐 + displacement (전위) + 혈관의 노장</u><br>- 음식물과 gas가 내려가지 못하고 거꾸로 기도로 넘어와 <u>오연성 폐렴</u>을 일으킬 수 있다.<br>- 무조건 응급 수술                                                    |
|         **Infectious Gastritis**         | [세균성]<br>- *Helicobacter* : 염증을 일으키지만 빈도는 낮음. 만성 위염과 궤양 유발.<br>- *Clostridium perfringens* : 환경 상재균으로 섭취 빈도가 높음. 괴사성 위염, 궤양 유발<br>- Bovine herpesvirus-1 (Infectious Bovine Rhinotracheitis를 유발) : 호흡기 원인체가 소화기 질환도 유발할 수 있음.                                                                                                                  |
|                    ^                     | [기생충성] : 괴사를 유발하는 특징 (necrotizing gastritis)<br>- 두께가 얇아지는 게 아니라 **위벽이 두툼**해지는 것이 특징적.<br>- 반추류 - *Ostertagia ostertagi* (4위에서 염증 유발) / 돼지 - *Hyostrongylus rubidus* <br>- 현미경 소견 : glandular hyperplasia (샘, 림프조직의 증식으로 볼록볼록 결절 형성), eosinophilic inflammation (기생충 감염 → 당연히 호산구!)                                                            |
|           **Uremic Gastritis**           | - 신장이 기능을 제대로 못하면 uremia가 축적되어 mineralization 유발<br>- 신장 배설 능력이 떨어지면 → uremia가 많이 만들어짐 → 대사되면서 질소산화물(NOS) 발생<br>- 질소 산화물은 혈관 공격성이 강함 → 혈관과 샘조직에 mineralization (질소 산화물에 공격받아 죽은 상피세포들의 자리에 칼슘이 침착)                                                                                                                                             |
|           **Pyloric stenosis**           | - Pyloric sphincter muscle (위 → 십이지장 위치)이 기능을 하지 못할 때.<br>- 섬유화(경화) 때문에 탄력을 잃고 이완/수축을 제대로 못함.<br>- 개의 단두종(brachycephalic), 샴고양이(Siamese), 말에서 비교적 흔함.<br>- 이유 후(weaned) 밥을 먹을 때, 반복적인 구토 시 의심해볼 것.<br>- 현미경 소견 : 근육 비대(muscular hypertrophy), edema, 혈관 확장(vascular ectasia), 신경(mesenteric ganglion cell)의 문제                                 |
|          **Gastric Neoplasia**           | - 위에서는 **leiomyoma**가 잘 생김. (평활근 세포에서 생긴 양성 종양)<br> 개에서 leiomyoma, 고양이에서 lymphoma / 소화관 내강을 좁힌다 ⇒ 원발 위치에서 문제를 유발.                                                                                                                                                                                                                              |
