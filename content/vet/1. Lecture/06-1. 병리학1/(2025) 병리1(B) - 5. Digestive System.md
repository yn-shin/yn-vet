## 1. Digestive System Overview

- 점막 구조가 사실상 외부에 노출된 구조임.
- 가장 노출도 많고 손상도 많다.

![[Pasted image 20251104114347.png|Pasted image 20251104114347.png]]

| 점막 조직               |     |              |                       |
| ------------------- | --- | ------------ | --------------------- |
| 구강\~식도              | 위   | 소장(duodenum) | colon                 |
| 편평상피(squamous cell) | 샘조직 | villi 존재     | villi는 없지만<br>샘조직은 있음 |

- 몇 겹의 두꺼운 layer를 뚫고 가야 복강으로 갈 수 있기 때문에
- 대부분의 병변은 점막에 국소적으로 존재한다.
- Brunner's glands : duodenum에만 특징적으로 존재
- Peyer's patch : lymphoid follicle이 많이 모여있고, 면역과 항상성을 조절
- Crypt of Lieberkuhn : crypt는 줄기세포(stem cell)를 많이 가짐. → 점막 villi의 세포들의 재생을 담당




#### 1️⃣ Oral Cavity

- 중층편평상피로 되어있음. = 물리적으로 튼튼해서 잘 손상되지 않기 때문.
- 동물마다 조금씩 다름 : 잇몸, 볼 쪽의 조직층이 얼마나 두꺼운지 비교하여 알아두기.
	- 증식성 병변을 판단할 때 원래 정상적인 형태학적 특징인지 분간하는 데 중요함.
- 동물들의 잇몸에도 minor한 침샘이 존재해서 소화를 도움.


#### 2️⃣ Esophagus
- 구강에서 식도까지는 다 중층편평상피 (stratified squamous epithelium)
- 구강의 소화물/음식물을 위로 내려보내기 위한 연동작용을 위해 근육층이 발달해 있음.

#### 3️⃣ Stomach & Forestomach (Ruminants)
- 포유동물의 위 = 반추동물의 4위(abomasum)
- 위쪽(내강쪽) → 점액 분비 / 안쪽(벽쪽) → 소화액 분비
- 위쪽 세포들의 turnover time이 짧음 (빨리 손상되고 빨리 재생됨)

#### 4️⃣ Small & Large Intestines

#### 5️⃣ Peritoneum, Omentum, Mesentery
- 모두 소화장기들을 덮고있는 구조 : 겹쳐있지만 각각 구분하여 알기
	- Peritoneum : 장막면
	- Omentum : 둘러싸는 지방조직
	- Mesentery : 지방조직에 존재하고 있으나 혈관과 림프관이 지나감

## 3. Defense Mechanisms

#### 🟢 Alimentary System의 Defense Mechanisms

- Mechanical (physical) : 두꺼운 층 → 소화기 내부의 문제가 밖으로 나가지 않게 함. / 조임근 → barrier (위의 문제가 급격히 소장으로 넘어가지 x)
- Biochemical : enzymes → 음식물뿐만 아니라 감염성 원인체를 녹이기도 함 / pH 조절 → 소장과 위의 pH가 달라서 살 수 있는 세균이 다름
- Immunologic : GALT, immune cells 정상세균총은 면역 세포가 건드리지 않는데,

## 4. Portals of Entry

소화기로 들어오는 통로는 크게 4가지.
- 먹거나 (ingestion)
- 기침으로 인한 유입 (cough-induced ingestion)
- 혈중으로 전신성 (systemic spread via bloodstream) : 가장 심각
- 기생충 유입 (parasite migration)

## 6. Diarrhea

- 정의 : Abnormally fluid feces accompanied by increased volume and frequency

- Inflammatory (염증성) : 직접적으로 cytokine을 분비해 염증을 유발하는 경우
	- *Salmonella*, *Lawsonia* 등
- 비염증성 : 세포를 직접 파괴하지 않고..?

장 운동 속도보다 느려져도, 빨라져도 임상 증상으로 설사가 발생할 수 있다.

#### 설사의 병리 기전

1) 삼투성 설사 : 흡수가 잘 안 될 때
	- 혈관내 삼투가 떨어지면 혈관의 fluid가 체외 공간으로 빠져나감 > 체외 공간에 fluid가 많아지면서 설사
2) 배출성 설사 : 배출이 잘 안 될 때
	- (Toxin이 channel을 통해..) Chloride를 퍼내면서 물이 같이 나감, 능동적 형태

과도한 운동 : 흡수할 시간을 제한 : 삼투성 설사


#### Infectious Agents
- ***Lawsonia intracellularis*** : 장벽을 두껍게 하는 것이 특징. 돼지에서 잘 볼 수 있음.
	- Crypt의 증식을 유발. 육안적으로도 울퉁불퉁해짐.

#### 면역 반응과 염증
- 장에서 염증 설사 → IL-8이 활성화되어? 대식세포를 동원함.
- T림프구 → villi 위축을 유발. 
- 염증성 cytokine들이 나오면 세포의 증식을 유발할 수 있음. 원래 cell cycle이 돌아가는 애들에서 과도한 증식을 유발함. 여러 염증성 장질환에서 점막 전체를 흡수세포, ~~
