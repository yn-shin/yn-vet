- 점막 구조가 사실상 외부에 노출된 구조임.

## 3. Defense Mechanisms

#### 🟢 Alimentary System의 Defense Mechanisms

- Mechanical (physical) : 두꺼운 층 → 소화기 내부의 문제가 밖으로 나가지 않게 함. / 조임근 → barrier (위의 문제가 급격히 소장으로 넘어가지 x)
- Biochemical : enzymes → 음식물뿐만 아니라 감염성 원인체를 녹이기도 함 / pH 조절 → 소장과 위의 pH가 달라서 살 수 있는 세균이 다름
- Immunologic : GALT, immune cells 정상세균총은 면역 세포가 건드리지 않는데,

## 4. Portals of Entry

소화기로 들어오는 통로는 크게 4가지.
- 먹거나 (ingestion)
- 기침으로 인한 유입 (cough-induced ingestion)
- 혈중으로 전신성 (systemic spread via bloodstream) : 가장 심각
- 기생충 유입 (parasite migration)



| 소화기 유입 경로                                                                                                 |
| --------------------------------------------------------------------------------------------------------- |
| (1) **Ingestion**                                                                                         |
| - 가장 common 루트, 오염된 음식과 물이 문제.<br>- *E. coli*, *Salmonella spp.* : 입으로 먹어서 소화기 증상 유발<br>- 위산 분비 감소, 효소 감소 |
| (2) **Cough-Induced Ingestion**                                                                           |
| - 호흡기를 통해 들어옴<br>- *Rhodococcus equi* : 말에서 호흡기 증상을 먼저 유발. 기침으로 인해 호흡기균이 소화기로 넘어감.                        |
| (3) **Systemic Spread : Blood-Borne Infections**                                                          |
| - *Clostridium perfringes* : 혈중으로 매우 잘 퍼짐                                                                 |
| (4) **Mucosal Breach**                                                                                    |
| 안전막을 통해 들어옴 > 감염 경로를 특정할 수 없음.                                                                            |
| (5) **Oral Cavity**                                                                                       |
| 치석 관리 중요, 치주염 유발할 수 있음<br>구강 청결                                                                           |

- Tonsil의 역할
- **Tonsil**은 <u>afferent</u> lymphatic vessels가 없음: tonsil은 들어오는 경로는 없고, 만들어서 내보내기만 한다.
- Tonsil"혈관을 통해 들어왔다" 가 맞는 설명.
- Feline Parvovirus : tonsil에 잘 증식.
- BVDV : tonsil을 통해 전신 혈류로 퍼져나감.

## 5. Response to Injury

어떤 병원체가 주로 염증, 혹은 괴사, 

#### 1️⃣ Inflammation
- Acute inflammation 
	- 급성 염증을 유발하는 대표적인 감염원: *Salmonella*, *E. coli*
	- 부종, 충혈, 혈관 손상 등이 유발된다
- Chronic inflmmation
	- 대표적으로 Johne's disease

![[Pasted image 20251107220652.png|600]]
#### 2️⃣ Necrosis
- 감염성 원인체에 의한 죽음은 apoptosis(계획된 죽음)이 아니라 **necrosis**이다.
- 소장 층에서 안쪽에는 crypt(줄기세포), 위에는 villi가 있음.
	- **Ulcer** : crypt에서 villi까지 전 층이 다 손상된 것.
	- Erosion (미란) : 일부 층만 손상
- 점막이 손상되었다는 건 barrier가 깨졌다는 것.
- lymphoid tissue가 파괴되면 면역 반응이 고장.
- villi가 영양분을 흡수해야 하는데 점막이 손상되면 영양분 흡수가 저해됨.

#### 3️⃣ Lymphangiectasia & PLE
- 관 구조로 된 장기는 폐색이 일어나면
- 림프관 <u>주변</u>의 lymphoma나 chronic inflammation ⇒ 림프관의 물리적 압박 ⇒ 폐색 ⇒ 이후 림프관이 확장된다.
- 갇혀있다 보면 투과성 변화를 유도해서 주변 내강으로 새어나가게 된다.
- 임상 증상 : 설사, 체중 감소, protein-losing enteropathy
- 간부전일 때, 장의 문제가 있을 때 잘 관찰됨.

#### 5️⃣ **Diarrhea**


- 정의 : Abnormally fluid feces accompanied by increased volume and frequency

- **Inflammatory** (염증성) : 직접적으로 cytokine을 분비해 염증을 유발하는 경우
	- *Salmonella*, *Lawsonia* 등
- **Non-inflammatory** (비염증성) : 염증을 유발하지 않되 세포를 직접적으로 파괴, 세포가 가진 것을 이용해서 설사를 유발함

- 원인
	- 감염성 원인체
		- 호흡기 감염 원인체가 기침을 통해 소화기로 재반입될 수도 있음
	- 위산 분비 감소
	- 장 운동 능력 : 장 운동 속도보다 느려져도, 빨라져도 임상 증상으로 설사가 발생할 수 있다.

###### 설사의 병리 기전

1) 삼투성 설사 : 흡수가 잘 안 될 때
	- 혈관내 삼투가 떨어지면 혈관의 fluid가 체외 공간으로 빠져나감 > 체외 공간에 fluid가 많아지면서 설사
2) 배출성 설사 : 배출을 너무 많이 할 때
	- (Toxin이 channel을 통해..) Chloride를 퍼내면서 물이 같이 나감, 능동적 형태

과도한 운동 : 흡수할 시간을 제한 : 삼투성 설사


###### Infectious Agents
- ***Lawsonia intracellularis*** : 장벽을 두껍게 하는 것이 특징. 돼지에서 잘 볼 수 있음.
	- Crypt의 증식을 유발. 육안적으로도 울퉁불퉁해짐.

###### 면역 반응과 염증
- 장에서 염증 설사 → IL-8이 활성화되어? 대식세포를 동원함.
- T림프구 → villi 위축을 유발.

- 고양이에서 [[현내2(B) - 3. Hyperthyroidism|Hyperthyroidism]]이 있을 때, ~~

#### 2️⃣ Gastric Dilation & Volvulus (GDV)

- Displacement
