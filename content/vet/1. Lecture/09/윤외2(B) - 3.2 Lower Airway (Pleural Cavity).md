##### Intercostal Thoracotomy

- 3~10번째 늑간만 열 수 있음. (~3 : 아무것도 없음, 10~ : 횡격막)
- 제한적 접근 :
- Trasitional 방식(넓은등근을 포함한 근육 자르면서 접근)으로는 파행이 발생할 수 있지만,
	- Muscle-sparing technique - 근육을 보존하는 방식으로 배우고 있음. 넓은등근 등 자르지 않고 견인해서 하고 있음.

##### Median Sternotomy

- 가운데를 쫙 갈라서 여는 방법
- Indication : 양쪽을 한꺼번에 접근하면서 치료 행위를 할 수 있다. (intercostal은 한쪽 접근만)
- Surgical window가 매우 크다.
- 오해 : sternotomy를 하면 잘 못 걷는다. but 실제로 그렇게 파행이 있지 X
- 생각보다 어렵지 않고 생각보다 많이 함

>[!tldr] Techniques
>- Manubrium(=1번째 흉골)
>- sternum
>- Xiphoid(=마지막 흉골)
>- 피부 좀 열다가 툭 걸리면 바로 흉골 만남
>- perio.. 로 잘 분리해서 흉골 노출
>- sagittal saw로 흉골의 정중을 쭉 갈라줌 → 장기는 쉽게 손상 X
>- Manubrium~xiphoid까지 다 가르면, 나중에 양쪽을 완벽하게 붙이기 쉽지 않다.
>	- 1) manubrium은 살리고 xiphoid를 자른다
>	- 2) xiphoid를 살리고 manubrium을 자른다
>	- 3) 둘다 살린다
>	- 수술 필요한 부위에 따라 뒤나 앞 중 한 쪽을 살려두면 좋음.

>[!tldr] Closure
>- 흉골과 흉골의 연결 부위에 늑골이 있음 → 봉합의 center가 늑골
>- 대각선 봉합 (8자 봉합) ⇒ 늑골에 의해 봉합사가 고정되어 움직이지 않음
>- 예전에는 불안해서 와이어를 썼지만, 뼈를 녹이고 물도 많이 참
>- 비흡수성/흡수성 봉합사 둘다 씀. heavy한 걸로.
