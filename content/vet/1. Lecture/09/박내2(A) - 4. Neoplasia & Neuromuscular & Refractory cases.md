# Neuromuscular junction disorder

- 신경전달물질 (acetylcholine 등) : nerve ending에서 muscle로 신호 전달, 적으면 몸에 힘이 없이 늘어지고, 많으면 근육 강직 등 
- 중증근무력증 → 뛰어다니다가 갑자기 쓰러짐
	- 선천성 중증 근무력증 : acetylcholine 수용체가 부족하게 태어남.
	- Acquired myasthenia gravis (후천성 중증 근무력증) : acetylcholine receptor에 대한 antibody가 형성되어 대신 붙음. → 마비
		- 진단하려면 Ach receptor Ab를 검사하면 됨.
- Acetylcholinesterase inhibitor ([[edrophonium|edrophonium]], [[neostigmine|neostigmine]], [[pyridostigmine|pyridostigmine]]) → Ach 분해를 억제, 중증근무력증 치료 
	- 부작용 : 소화기계의 과도한 운동으로 설사를 유발할 수 있음
- Tensilon (edrophonium) test 를 통해 중증근무력증을 테스트함. (치료적 진단)
- 국소적 또는 전신적으로 오는 motility disorder. 

### Megaesophagus를 유발하는 원인 

- Myasthenia gravis
- Esophagitis
- Endocrine disease (내분비 질환)
	- hypothyroidism, Addison's disease
- SLE, polymyopathies (다발성 근염)
- Idiopathic
- Thymoma (thymus에 생긴 cancer)
- Botulism (음식 잘못 먹어서 독소 발생)

>[!check] Further examination
> - **Ach receptor antibody test**
> - ANA test : SLE 감별
> - Thyroid profile (T4, fT4, TSH, Thyroid Ab)
> - ACTH stimulation test : Cushing, Addison 감별

### MG 발생 기전

✔️ 전신적 MG : 막 운동하다가 갑자기 픽 쓰러지는 애들, edrophonium 주면 20초 만에 벌떡 일어남.<br>✔️ Focal MG : 식도/위/소장/대장에만 국소적으로 오는 경우, 소화기 운동성이 떨어짐. → 증상은 소화기지만 원인은 신경학적. 수술이 아니라 약물로 치료해야 함.

| 신경 전달 과정                                                                                                                                                    |                                                                                                                                                                        |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ![[Pasted image 20250419150406.png\|250]]                                                                                                                   | choline acetyl transferase에 의해 acetyl CoA가 acetylcholine으로 바뀜, 이 과정에 작용하는 미네랄이 Ca2+.<br><br>즉, <u>Ca이 있어야만</u> acetylcholine이 보자기처럼 담겨, <u>post-synapse로 이동할 수 있음</u>. |


- Hypocalcemia : Ach이 release되지 못해 <u>근육 이완</u> (장 운동 저하 등)
- Hypercalcemia : 강직증, 심장 수축력 증가(빈맥)<br>  → Calcium channel blocker - Ach 적게 분비, 탈분극 지연, 심장을 천천히 뛰게 함 / 이완기 문제인 HCM 등에 적용 ([[amlodipine|amlodipine]], [[verapamil|verapamil]], [[diltiazem|diltiazem]])

- Anticholinesterase therapy
- 후천성(자가면역성) → Immune suppressive therapy : thymoma에 의해 항체가 형성, 항체를 차단하는 면역억제제 필요 ([[cyclosporine|cyclosporine]], [[MMF|MMF]], [[leflunomide|leflunomide]])

### Gastrointestinal motility 관련 Receptor subtypes

Neurotransmitter의 문제에 의해 생기는 소화기 질병이 많음.

| Receptor            | Agonist                                                           | Antagonist                                    |
| ------------------- | ----------------------------------------------------------------- | --------------------------------------------- |
| Muscarinic          | [[pyridostigmine|pyridostigmine]] (+)                                            | [[atropine|atropine]] (-)                              |
| Alpha 2 adrenergic  | [[clonidine|clonidine]] (-)                                                 |                                               |
| Dopamine D2         | [[dopamine|dopamine]] (-)                                                  | [[metoclopramide|metoclopramide]] (+)<br>[[domperidone|domperidone]] (+) |
| Serotonin 3 (5-HT₃) |                                                                   | [[metoclopramide|metoclopramide]] (+)<br>[[ondansetron|ondansetron]] (+) |
| Serotonin 4 (5-HT₄) | [[cisapride|cisapride]] (+) : nonselective<br>[[mosapride|mosapride]] (+) : selective |                                               |


(+) : excitatory effect on motility, (-) : inhibitory effect on motility


> [!info] Muscarinic receptor 
> - **pyridostigmine** : 장 운동 촉진, MG에서 사용하는 AchE inhibitor


>[!info] Dopamine D2 receptor
>- 십이지장 쪽에 분포함
>- 뇌에서 도파민이 너무 많이 분비되면 vomiting : 운동 능력이 촉진되어 과도한 장 운동
>- 도파민 차단제 : **metoclopramide**, **domeperidone**
>	- dopamine과 metoclopramide 동시에 쓰면 소용 없겠지~?
>- Metoclpramide는 <u>항구토제</u>로 매우 많이 쓰임 : 도파민 D2 수용체를 차단해서 구토가 이루어지지 않게 하는 것

>[!info] Serotonin 3 (5-HT₃) / Serotonin 4 (5-HT₄) receptor
>- Serotonin = 5-hydroxytryptamine, 5-HT
>- Serotonin 3 & 4 수용체 : 위장관 쪽 분포, 특히 4는 식도 쪽에 많이 분포
>	- Serotonin 3 차단제 = <u>항구토제</u> : **ondansetron**, **metoclopramide** (세로토닌과 도파민 수용체 둘 다 차단)
>	- Serotonin 4 차단제 = 식도 운동성 촉진 : **cisapride**, **mosapride**
>
>---
>
>![[Pasted image 20250419174636.png|350]]
>
>- Brain의 중요한 neurotransmitter, 95%는 위장관 계통에 작용함.
>	- cell membrane의 activity를 줄여주는 효과 
>	- 뇌에서 세로토닌 분비가 증가하면 위장관 계통에 문제 (설사, 구토 등)
>- Serotonin이 너무 많이 분비되어 나타나는 증상 → <u>serotonin syndrome</u>
>- duodenum 쪽에 도파민 수용체, 세로토닌3 수용체가 많아 구토가 잘 생김. ⇒ 여기를 차단하여 구토를 억제
>	- [[maropitant|maropitant]] [cerenia] : 뇌의 vomiting center에 작용, 중추성 구토를 억제 (다른 애들은 다 말초성) 
>	- [[apomorphine|apomorphine]] : 도파민 수용체에 붙어서 구토 유발.


>[!info] Mosapride, Cisapride (Serotonin 4 receptor)
>**Mosapride** : <u>selective</u> 5-HT4 receptor agonist
>- 식도의 운동성을 촉진 (Prokinetics)
>- 위장관 운동의 효과에 대해 in vivo/vitro study  했더니⇒ mosaspride > cisapride, 0.3-3mg/kg IV 투여했더니 식도의 운동성이 상당히 촉진
>- Affinity : metocloprimide보다 2-12배 높다.
>- Lower보다 Upper GI tract에 훨씬 친화도가 높다, (⇒ lower GI tract에는 metocloprimide)
>- 부작용 : 사람에서는 두통(headache) 알려져 있다 (Headache 즉 Migraine(편두통)이라고 하심; 편측으로 나타나는 박동성 구통)
>
>**Cisapride** : <u>nonselective</u> serotonin receptor agonist 
>- Serotonin뿐 아니라 dopamine receptor, muscarinic receptor에도 붙음. 
>- 전체 GI tract에 붙는다 → 부작용이 더 많다.
>	- 빈맥, 심실 빈맥, QT 간격 증가, Extrapyramidal syndrome (척수의 추체가 아니라 추체 옆 신경을 따라가서 빠르게 전달됨, 강직증이 잘 발생)

>[!check] Dopamine 관련 pathway
>- 뇌에서 도파민이 부족하면
>	- 인지 기능 저하, 운동 기능 저하
>	- 손을 덜덜 떤다 (파킨슨 초기 증상)
>- 도파민은 monoamine oxidase(MAO) 에 분해됨
>	- 도파민이 과도하면 강직증이 오니까, 몸에서 적당히 쓰고 분해시킴
>- L-selegiline : MAO 억제제, 도파민을 항진시켜서 인지기능 촉진 (파킨슨병 보조 치료에 사용)
>	- 너무 많으면 부작용으로 도파민이 많아져 vomiting 유발

- 타액선 cytlogy → 세포 정상, 종양이나 염증 아님.
- CT, Brain MRI → NRF
- 흉강 X-ray → NRF
- 삼안검 부종, 호흡곤란, 간헐적 구토, 침흘림은 계속됨..
- 일주일 동안 소화기 약을 써도 증상이 개선되지 않음.

> ##### A & P


검사는 잔뜩 했는데 진단이 안 나오는 상황.

이스라엘의 한 논문에 똑같은 케이스가 실림, 치료 방법으로 phenobarbital 저용량으로 써 보는 것이 제시됨.
- Phenobarbital sodium (1.5mg/kg PO q12h) 적용하자 하루 만에 증상이 dramatic하게 사라짐.

⇒ <u><b>Idiopathic phenobarbital-responsive sialadenosis</b></u>  (전 세계에서 3번째로 보고된 케이스)



>[!check] Sialadenosis (🔗 [[김내2(A) - 1.2 Sialoadenosis (타액선증)|김내2(A) - 1.2 Sialoadenosis (타액선증)]] - PRS)
>- limbic epilepsy로 분류.
>- limbic에서는 호흡, 골격근 운동, 소화작용, 호흡, 배뇨, 배변에 관여함.
>	- 인지 기능, 감정, 행동, 동기 부여, 기억, 후각 등 여러 기능을 담당함.
>	- 여기 문제가 생기면 구토, 근육 떨림, 삼안검 돌출, 아무데나 배뇨, 타액선 비대 & 분비 많아짐, 거품.
>- Clinical signs
>	- retching (헛구역질, sudden)
>	- gulping (꿀꺽꿀꺽 삼키는 행동)
>	- salivary gland bilateral enlargement (양측성 침샘 비대)
>	- gagging (켁켁거림)
>	- lip-smacking (입술을 핥음음)
>	- weight loss, hyporexia, vomitting
>	- 마치 호흡기/소화기 질병처럼 나타남
>- 매우 rare
>- Generalized, partial 말고 매우 드물게 limbic epilepsy도 있다는 것.


⇒ phenobarbital(항경련제) 쓰면 dramatic하게 개선, 평생 약 먹어야 함.
