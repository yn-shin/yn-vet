>[!check] Case 1
>- 12y, toy poodle, male, 4.84kg
>- Dx - TCC, hematuria, melena, DIC
>- PT 지연, APTT 지연, TEG 비정상 ⇒ Consumptive coagulopathy (소모성 응고병증)
>- 지속적인 염증으로 DIC 발생

수액 처치만 하면 → dilution이 가장 문제. 혈액이 희석됨.
이 환자에서 필요한 Resuscitation = 수혈

## 수혈의 원칙 : 3R
- **Right Timing** (언제?)
- **Right Material** (무엇을? 얼마나?)
- **Right technique** (어떻게?)

## 1. Right Timing
- No transfusion trigger : 수혈의 타이밍을 판단할 수 있는 명확한 trigger는 없음.
- tissue hypoxia(shock)일 때 조직으로 산소를 운반해주기 위해 수혈 필요.

| 수혈의 판단 기준                                                                                                                                                                            |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Clinical signs**                                                                                                                                                                   |
| Shock와 관련된 증상이 있을 때 수혈하 (주관적, 경험적 영역..)<br>: tachycardia, pallor, weakness/exercise intolerant, elevated lactate, tachypnea, +/- hypotensive                                         |
| **PCV**                                                                                                                                                                              |
| - <u>Chronic</u> → PCV < 15% 시 (재생불량성 빈혈, 종양)<br>   : 빈혈의 상황에 적응할 수 있는 충분한 시간이 있었음<br>- <u>Acute</u> → 25-45% (suspect > 25-30% loss)<br>   : 급성은 적응할 시간이 없음. 혈액량의 30% loss 일어나면 수혈. |
| **Hemoglobin**                                                                                                                                                                       |
| - < 7g/dL : hypoperfusion<br>- < 5-6g/dL : high risk of death                                                                                                                        |

의학적으로는 Hgb가 훨씬 sensitive한 marker이지만,
PCV가 훨씬 기계의 영향을 덜 받고 일관적인 수치여서 PCV로 기준 잡음.


## 2. Right Material

#### 1️⃣ RBC Transfusion

> - Wholed blood
> 	- Fresh WB
> 	- Stored WB
> - Packed RBC (pRBC)


##### Fresh whole blood (FWB)
- 채혈 후 **4-6시간 이내**, <u>혈소판이 살아있는</u> 혈액
- RBC, WBC, platelet, coagulation factor 모두 있음
- Indication
	- Acute hemorrhage (surgery, trauma, organ rupture)
	- Evans syndrome (IMHA+IMT)


##### Stored whold blood (WB)
- 채혈 후 6시간이 지나면(=혈소판 파괴) 저장해서 사용 
- Stored 1-6℃ for 28 days (냉장고에 <u>1달까지</u> 저장)
- RBC, WBC, coagulation factor / ~~Platelet~~ (혈소판은 깨졌지만 응고인자는 살아 있음)
- Indication
	- Acute hemorrhage
	- Anemia with <u>coagulopathy</u>

##### Packed RBCs (pRBC)
- Whole blood에서 RBC 분리
- Stored 1-6℃ for 35 days (보관 기간이 좀 더 길다)
- RBC, WBC (+with minimal plasma) : 오직 적혈구 수혈만 가능, 응고인자/albumin 등은 없음
- Indication 
	- Anemia without coagulopathy
	- FFP(Fresh plasma)와 섞어 전혈처럼 사용할 수 있음

##### Leukoreduced pRBCs
- Leukoreduction : 백혈구를 거르는 필터 → 백혈구+혈소판 제거하고 적혈구만 남음
	- post-storage보다 pre-storage가 좋음 (저장 전에 필터링)
- Leukocyte는 cytokine을 분비하고 transfusion reaction(FNHTRs)을 유발 - 사실 수혈에 있어서는 이로울 게 하나도 없음.
- 이런 반응이 문제가 될 수 있는 환자에서는 leukoreduction을 거친 혈액을 수혈 (과민 반응, 면역에 문제 있는 환자)

##### Stored WB
- 1-6℃ refrigerator에서 28-35일간 보관.
- 2일에 1번 (EOD) 뒤집어주지 않으면 혈전 생김.

>[!warning] Storage lesion
>- 저장 기간 중 glucose가 소모되어 ATP가 생성되고, 세균이 살기 좋은 환경이 됨. 
>- 적혈구 기능 저하 및 합병증 유발.
>- First In, First Out으로 사용!
>- 단 특수 상황에서는 우선으로 신선한 혈액 사용
>- IMHA는 자가면역 질환이므로 혈전 가능성



### 2️⃣ Plasma Transfusion
- **응고인자 보충**에 가장 강력한 효과!

| Plasma 종류                 |                                                         |
| ------------------------- | ------------------------------------------------------- |
| FFP (Fresh Frozen Plasma) | Fresh WB에서 바로 혈장을 분리해서 만든 것                             |
| FP (Frozen Plasma)        | 8시간 이후, 24시간 이내 바로 분리해서 만듦<br>약간의 응고인자들이 파괴되었지만 큰 문제 없음 |
| Thawed plasma             | 37도에서 녹여서, 5일간 냉장 보관하며 사용                               |


### 3️⃣ Platelet Transfusion
- 혈소판은 채혈 6-8시간 내에 파괴됨.
- 이런 제제들에서 가능
	- **Fresh Whole blood**
	- Plastelet Rich Plasma

### 그래서? 혈액 제제 결정
아래 설명은 객관식에서 중요한 내용이었고..

결정할 때는
1) 적혈구 수혈이 필요한가? → RBC 중 하나
2) 혈소판, 응고인자는 필요한가? 
	- 필요하면 Fresh WB !!! (없으면 pRBC+FFP 가능)
	- 필요하지 않으면 pRBC
	- 혈소판은 필요없지만 응고인자는 필요하면 Stored WB
3) 적혈구 정상이고 Coagulopathy가 있다면 → Plasma (FFP)
 

| 종류                  |                                  |
| ------------------- | -------------------------------- |
| **RBC**             | **PCV가 떨어진 환자**                  |
| Fresh Whole Blood   | RBC + WBC + 응고인자 + **platelet**  |
|                     | : 채혈 4-6시간 이내이므로 donor에서 얻을 수 있음 |
| Stored Whole Blood  | RBC + WBC + 응고인자                 |
| Packed RBC          | RBC + WBC (+minimal plasma)      |
| Leukoreduced pRBC   | RBC (WBC, plasma 제거)             |
| **Plasma**          | **PCV가 정상인 환자**                  |
| Fresh Frozen Plasma | Hemostatic protein, 단백질 등        |
| Frozen plasma       |                                  |
|                     |                                  |

## How, Where

> 혈액형 검사 → Cross-matching → 수혈량 계산 → 수혈 → 모니터링 순!


Canine
- DEA : 1, 3, 4, 5, 6, 7, 8, Dal
- DEA **1**, 4, Dal 주의! 특히 1형이 가장 무서운 반응을 일으킴.
- AHTR (Acute Hemolytic Transfusion Reaction)을 일으킬 수 있음.

같은 애들끼리는 당연히 가능하고
개는 자연발생항체가 없어서 1차 수혈에서는 문제를 일으키지 않음.
2차 수혈부터는 주의!


|     |     |
| --- | --- |
|  DEA 1- → DEA 1+ (1차)  |   가능  |
| DEA 1+ → DEA 1- (1차) | 가능 |
| DEA 1+ → DEA 1- (2차) | 불가능 |

DEA1-가 1+의 항원에 반응을 일으킴 (1차 수혈에서 항체 생겨서 2차 수혈부터 반응)
