# 1. Etiology & Epidemiology

- 2가지 질병 유발 : 1) Feline <u>infectious peritonitis</u> virus (FIPV) 2) Feline <u>enteric corona</u>virus (FECV)
	- FIPV : 고양이 전염성 복막염 유발
	- FECV : 고양이 장염 유발
- serotype(혈청형) : 항원성에 따른 구분
	- Serotype 1이든 2든 장염/복막염 유발 가능 (serotype과 임상 증상은 직접적 연관성이 없음)
- Biotype(생물형) : 임상 증상(장염/복막염 등)에 따라 구분
- FECV에 감염된 고양이에서 바이러스의 돌연변이(특히 3c, spike protein 유전자 등)가 발생하면 FIPV로 전환될 수 있음

# 2. Clinical Findings
#### FECV

- 장에 감염을 일으킴.
- 감염 후 3일 정도면 분변에서 바이러스가 검출될 수 있고, 전염성이 매우 높음.
	- 집단 생활하는 고양이들은 거의 다 감염된다는 연구도 있음.
- 열, 구토, 점액성 설사 등 경미한 소화기 증상
- 면역이 괜찮으면 대부분 며칠 내로 회복함(self-limiting). Supportive care만 해도 잘 낫는 편.
- 바이러스 배출 형태가 다양 - 계속 배출하는 경우도 있고, 간헐적으로 나오거나, 나았다가 다시 배출되는 경우도 있음.

#### FIP

- FECV에 감염된 고양이에서 돌연변이가 생기면 FIPV로 바뀔 수 있음. FIP는 주로 다묘 가정이나 사육장에서 많이 발생함.
- FIP 임상형은 크게 두 가지로 나뉨:

| Effusive form (wet)                      | Noneffusive form (dry)                                                                                                          |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| 배가 빵빵하게 참<br>(복강, 흉강 등에 단백질이 풍부한 액체가 고임) | 부분적인 세포 매개성 면역이 있는 환자에서 발생.<br>덩어리진 <u>육아종성 병변이 다양한 장기에</u> 발생<br>(pyogranulomatous or granulomatous)<br>주로 눈, 뇌 > 신장, 장간막, 간 등 |

- 두 가지 형태가 혼합된 경우도 있음.
- RT-PCR로 진단 가능 (특히 effusion에서 바이러스 검출 시 임상적으로 유용)

>[!tldr] FIP 발병에 영향을 주는 요인들(Factors):
> 1. 바이러스의 병원성(virulence)
> 2. 감염된 바이러스의 양(dose)
> 3. 감염 경로(route)
> 4. 고양이의 면역 상태(immune status)
> 	- 6개월령 이후에는 저항성이 증가함
> 	- FeLV, 호흡기 감염 시 FIP 위험 증가
> 	- FIV 있는 고양이에서 FECV shedding이 훨씬 증가함
> 5. 유전적 요인(genetically)
> 	- predisposed breeds : 브리티시 숏헤어, 데본 렉스, 아비시니안 등 (정확한 기전은 아직 불명확)
> 6. 동시 감염(concurrent infection) 여부
> 7. 이전에 non-FIP inducing feline coronaviruses에 노출된 적이 있는지 여부
> 	- 실험적으로는 antibody-dependent enhancement가 있지만
> 	- 실제 자연 감염에서는 드물게 관찰됨

FIP의 Clinical Findings

- 5살 미만의 어린 개체에서 (특히 <1살)
- intact male에서 많이 걸림
- FECV와 달리 FIP는 <u>집단 감염 심하지 않음</u>.
- 비특이적 임상 증상: 식욕부진, 체중감소, 우울 전신적인 증상이 흔함.
- 눈, 뇌로 진행: 포도막염, **맥락막염** / 신경학적 증상 (특히 dry FIP에서)

# 2. Diagnosis

- 증상에 따라 다양한 hematology, S-chem, urinalysis.. 등이 나타남
- 확진은 오직 **histopathology(조직검사)** 로만 가능!!
- 실제로는 임상적 특징 + 여러 검사를 종합하여 가진단 (잠정진단) 

---

- α1-acid glycoprotein : 염증 확인, FIP를 확진하는 것 아님.
- Feline coronavirus 키트
	- 위양성 가능, 위음성은 거의 드묾 (이론적으로 가능)
	- 길냥이는 웬만하면 양성으로 많이 뜸..
	- Cross-reacting antibodies를 형성하기 때문에, 바이러스에 노출된 적이 있을 뿐 FIP를 감별할 수 없음.
	- 어린 고양이에서 모체이행항체가 detect될 수 있음
- ICC (Immuno<u>cyto</u>chemistry) | 위양성 가능성
- IHC (Immuno<u>histo</u>chemistry) | 좀 더 진단.. (노란색 복수, 퉁퉁 부은 장
- RT-qPCR | 
	- 혈액, 분변 샘플 X
	- Effusion : 검체로 유용함
	- 신경/눈 문제 환자의 CSF도 민감도 높음

| 정리                                         |                                                                                                                                                                   |
| ------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Hematologic                                | nonregenerative anemia (normocytic, normochromic)<br>Neutrophilic leukocytosis, lymphopenia<br>Thrombocytopenia (DIC 있을 때)<br>드물게 hemolytic anemia                |
| S-chem                                     | **Hyperglobulinemia** <br>    ✓ < 0.4 이면 supportive<br>    ✓ > 0.8 이면 excluded<br>α1-acid glycoprotein (APP; acute phase protein)                                 |
| CSF                                        | Increased [Protein]<br>Increased nucleated cell counts (염증세포)                                                                                                     |
| Radiographs                                | Pleural/Pericardial/Peritoneal **effusions**<br>(Fissure line, 폐가 둥둥 떠다님)<br>Decreased serosal detail<br>Hepatomegaly<br>Renomegaly<br>Mesenteric lymphadenopathy |
| Ultrasonography                            | Abdominal fluids                                                                                                                                                  |
| **키트** \| Feline coronavirus antibody test | Antibody-positive                                                                                                                                                 |
| **Effusions**                              | Sterile, colorless to straw-colored<br>[Protein] 높음<br>Albumin/globulin ratio (AGR) <br>    ✓ AGR 0.5 = 양성<br>	✓ AGR 1.0 = 음성                                     |
| 현미경                                        | 다양한 염증 세포<br>대부분 Neutrophils > Matrophages<br>일부는 Macrophages 우세                                                                                                  |
| Antibody titiers?                          | Viral RNA가 검출되고, effusion을 유발할 다른 질병이 없으면<br>분자 검사를 꼭 수행                                                                                                          |

>[!warning] Definitive Diagnosis
>1) characteristic histopathologic findings
>2) presence of coronavirus by virus isolation

### Zoonotic Aspects


- 고양이 코로나바이러스는 건조한 환경에서 최대 7주까지 생존 가능.
- 소독약으로 쉽게 불활성화됨.
- FIPV에 노출된 모든 고양이가 FIP에 걸리는 것은 아님.
- FIP에 대한 면역은 영구적이지 않음.
- 유전적으로 FIP에 취약한 고양이는 번식에 주의 필요.
- 백신(비강 접종, Primucell FIP)은 효과가 불확실하여 비권장(noncore) 백신으로 분류됨.
- 사람에게 전염된 사례는 없음(인수공통감염 위험 없음).
