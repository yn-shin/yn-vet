# 1. 환자 평가

응급 상황이므로 신속, 정확하게 판단한다.

| Physical Examination |                                                                                                                                                                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Respiratory pattern  | - 정상 호흡수는 <u>30 이하</u><br>- Noise 평가<br>　✓ **Stridor** : 상부 호흡기, 좁아진 기도<br>　✓ **Stertor** : 상부 호흡기, 기도 막힘<br>　✓ **Goose-honking** : 하부 호흡기, 기관 허탈<br>- Panting                                                                             |
| MMC, CRT             | - **Cyanotic** : 매우 심한 저산소증, 혀에서 잘 관찰<br>- **Pale** : 관류 저하 (용혈성 빈혈, 비소상성 쇼크)<br>- **Brick red** : 보상성 쇼크<font color="#7f7f7f">, carboxyhemoglobin</font><br><font color="#7f7f7f">- Dark Blue : 청색증보다 더 어두운 푸른색, methemoglobinemia</font> |
| Temperature          | - **고체온증** : 상부 호흡기 폐색(호흡이 안 되어 흥분), 감염 (폐렴, 농흉)<br>- **저체온증** : 비보상성 쇼크, cardiogenic shock (cats)                                                                                                                                         |
| Asculatation         | - Lung sound<br>　✓ Crackles : 폐부종, 페출혈, 폐렴, 종양 등 뭐가 찬 것<br>　✓ Wheezes : 천식, 호흡기 폐색 (좁은 길을 빨리 지나가다 나는 소리)<br> - Heart sound<br>　✓  Murmur : 판막이 제대로 닫히지 않아 혈액 역류<br>　✓  Gallop : 심장 근육이 제대로 이완되지 않아 혈액과 부딪힘                                 |
| Pulse quality        | Bounding / weak puse 주의                                                                                                                                                                                                                    |
| Blood pressure       | Systolip BP < 100 Hg                                                                                                                                                                                                                       |
|                      | ⇒ 쇼크!!                                                                                                                                                                                                                                     |
