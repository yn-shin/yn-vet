## 1️⃣ GME (육아종성 뇌수막염)

#### Etiology

- GME (Granulomatous meningoencephalitis) : CNS의 염증성 특발성 질병
- 어린 연령에서 호발 / Poodles, Toy breeds, Pekingese, Terriers (우리나라에서 흔한 소형견종)
- Occasionally large-breed / 2-6살도 걸림 (대부분은 6개월 정도)
- 개에서 female에 좀 더 호발한다고 함.
- 고양이에서는 정확한 통계 없음. 훨씬 드문 편.

| Three distinct form   |                                                                                                                                                            |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Ocular form**       | - 가장 드문 형태.<br>- 급성 시력상실, 동공 산대 및 반응 없음.  <br>- 안저검사에서 충혈, 부종, 출혈 등.<br>- 나중에 dissemianated GME까지 이어지기도. (빠른 진단 중요)                                        |
| **Focal form**        | - caudal cranial fossa → vestibulocerebellar signs<br>- cerebrum → 전신 발작 등 forebrain signs<br>- 서서히 진행 (4-6개월)                                             |
| **Disseminated form** | - 병변의 위치와 기원에 따라 증상이 다양함<br>- 급속 진행, 다발성/광범위 병변 (cerebrum, brainstem, cerebellum, cervical spinal cord, meninges 등)  <br>- 발작, 서맥, 발열, 목 통증 등 다양한 신경학적 증상. |

#### 2. Clinical Features


- 임상 증상은 병변의 위치와 성격에 따라 다양함.
	- 발작, 원형보행, 행동 변화: 전체 환자의 약 20%에서 관찰
	- 뇌줄기 증상: 안진, 머리 기울임, 균형 상실, 뇌신경 결손
	- 경부 통증: 약 10%에서 관찰, 수막염, 척수 침범, 두개내압 상승 시
- 척수 침범: 전체의 약 10%에서 관찰
- Disseminated form: 발열, 말초 호중구 증가(50%에서), 급성~아급성 진행(수일~수주~수개월), 25%는 1주 이내, 대부분은 21일 이내 사망
- Focal form: 더 서서히 진행, 4-6개월에 걸쳐 악화

#### 3. Diagnosis

|                                    |                                                                                                                                                                                                 |
| ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Imaging                            | - MRI: Focal GME는 단일, 불규칙 경계의 mass lesion,<br>　T2/FLAIR에서 고강도, T1에서는 등강도 또는 저강도, 조영증강은 다양함.  <br>- Disseminated GME는 다수의 불명확한 병변이 뇌실질 및 수막에 존재.  <br>- CT는 MRI보다 민감도가 낮으나, 조영증강은 흔히 관찰됨(염증 반영). |
| CSF analysis                       | - Protein 증가 : 단핵구가 특히 많아짐 (mononuclear pleocytosis)<br>- 소림프구, 단핵구, 드물게 형질세포  <br>- Anaplastic mononuclear cells(드물게)<br>- Neutrophils : 20% 미만으로 보임                                           |
| CSF electrophoresis<br>(뇌하수체 전기영동) | -  혈액뇌장벽(BBB) 손상, 만성 환자에서 감마글로불린의 뇌내 생성 증가<br>-  사실 이 정도 용량의 샘플을 뽑기 어려움                                                                                                                         |
| Infectious disease exclusion       | - 감염성 원인 배제: 세균배양, 혈청 및 CSF 항체가, 항원검사, PCR<br>(Neospora, Toxoplasma, tick-borne, West Nile 등)                                                                                                   |
| Definitive diagnosis               | 뇌조직 검사를 하기 어려움 <br>⇒ typical MRI, CSF findings, elimination of diseases로 잠정 진단.                                                                                                                 |


#### 4. Treatment & Prognosis

| Treatment                                               |                                                                                                                                                       |
| ------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Glucocorticoids**                                     | - 일시적으로 호전 또는 역전 가능(특히 focal form에서).  <br>- 하지만 증상 재발이 흔함. Median survival: 7~114일(질병 형태와 위치에 따라 다양).                                                |
| **Cytosine Arabinoside**<br>= [Cytosar]<br>= Cytarabine | - BBB를 잘 통과함.<br>- 이 약 덕분에 GME의 예후가 훨씬 좋아짐.<br>- 더 공격적인 화학요법 병용 시 임상 증상 및 생존기간이 더 연장됨.  <br>- Focal form에서 면역억제제 병합치료 시 median survival > 12개월 기대 가능. |
| Cyclosporine, Azathioprine                              | - 추가 면역억제제 병용 시 효과적일 수 있음.                                                                                                                            |
| Radiation therapy                                       | - 일부 focal intracranial mass에서 효과 보고됨.<br>- 화학요법과 유사한 결과.                                                                                             |

- 영구 회복은 되지 않음.
- 전반적으로 예후가 좋다 보기는 어려움.
- 정확한 protocol을 확립하기는 어렵다 :
    - 저자는 PDS + 4 cycles cytosine arabinoside + cyclosporine or azathioprine
- MRI, CSF cytology로 치료 모니터링은 드물게 시행됨.
- Forebrain sign이 있는 경우 multifocal/brainstem sign보다 생존기간이 더 긍정적임.
- 발작 또는 cerebral edema(MRI상) 동반 시 생존기간이 제한적임.

## 2️⃣ NME (Necrotizing Meningoencephalitis, 괴사성 뇌수막염)

- 품종 특이적, 특발성 염증성 질환
- 주로 퍼그(Pug), 말티즈, 웨스트하이랜드화이트테리어, 치와와, 페키니즈, 시추, 파피용, 코통드툴레아, 라사압소 등에서 발생
- 발병 연령: 9개월~7세(퍼그 평균 18개월, 기타 품종 평균 29개월)
- 암컷 퍼그에서 더 흔함
- 급성 발작 및 뇌/수막 관련 신경증상(보행 곤란, 약화, 협응장애, 원형보행, 머리누르기, 행동 변화, 피질성 실명, 목 통증 등)
- 신경학적 악화가 매우 빠름: 치료하지 않으면 5~7일 내에 혼수, 기립불능, 사망
- 일부 퍼그는 더 느리게 진행하는 형태도 있음(간헐적 발작 후 점진적 악화, 생존기간 6개월 미만)

#### Diagnosis & Treatment

- 신경학적 증상, 품종, 영상학적 소견(MRI/CT: 뇌실 옆 백질, 회색-백질 경계 부위의 고단백 액체로 채워진 공동, T2W 고강도 침윤성 병변)으로 의심
- CSF: 단백질 증가, 세포수 증가(주로 소림프구, 일부 대형 단핵구)
- 감염성 질환(Toxoplasma, Neospora, CDV 등) 배제 필요
- 확진은 부검/생검에서 괴사성 병변 및 공동 확인
- 치료: GME와 유사한 면역억제 요법, 항경련제(phenobarbital)로 발작 조절
- 예후: 장기적 호전 및 생존은 매우 불량

## 3️⃣ NLE (Necrotizing Leukoencephalitis, 괴사성 백질뇌염)

- 품종 특이적, 다발성 괴사성 비화농성 뇌염
- 주로 요크셔테리어, 프렌치불독, 드물게 말티즈에서 발생
- 발병 연령: 1~10세(평균 4.5세), 암수 모두 발생
- 병변: 대뇌, 시상, 뇌간의 백질(white matter)에 주로 분포
- 증상: 의식 변화, 발작, 시각장애, 머리 기울임, 안진, 뇌신경 이상, 고유감각 이상 등
- 신경학적 악화가 매우 빠름: 대부분 5~7일 내에 기립불능 또는 사망

#### Diagnosis & Treatment

- 신경학적 증상, 품종, 급속 진행하는 대뇌/뇌간 증상으로 의심
- MRI: 대뇌, 시상, 뇌간 백질에 다발성 비대칭성 T2W 고강도 병변, 낭성 괴사 영역
- CSF: 단백질 증가, 혼합성 염증세포 증가(대식세포, 단핵구, 림프구, 형질세포)
- 치료: GME와 유사한 면역억제 요법 권장, 예후는 매우 불량
