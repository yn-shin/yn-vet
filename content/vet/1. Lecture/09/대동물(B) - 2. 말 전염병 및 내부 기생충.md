#### 1. 말의 주요 전염병
- **제1종 가축전염병** | 아프리카마역 (중요하지만 국내 발생X), 수포성 구내염
- **제2종 가축전염병** | 말 전염성 빈혈, <u>전염성 자궁염</u>, 바이러스성 동맥염, 마비저, .. <u>일본뇌염</u>, 서부 나일열
- **비법정 전염병** | <u>선역</u>, 비강폐렴, <u>말 인플루엔자</u>, 파이로플라즈마증 등
- 인플루엔자, 선역, 일본뇌염 3가지는 꼭 백신? 맞아야 함.
- 그 중에서도 **말 인플루엔자** 가장 중요.

WOAH (세계동물보건기구)
- List A (국제적 문제) : 아프리카마역, 수포성 구내염 등
- List B (국가적 문제) : 일본뇌염, 비강폐렴, 말인플루엔자 등

| Equine Infectious Diseases                                                             |                                                                                                                                                                                                                                                                                                                |
| -------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **말 인플루엔자**<br>(Equine Influenza)                                                      | - 전세계 공통 문제, 국내 법정 전염병은 아니지만 WOAH에 보고 의무 있음<br>- 기침과 고열 등의 증상, 폭발적 감염력(이환율 95%, 지역적 전파)<br>- 주로 비말 감염, 어린 말에서 많이 발생하나 전 연령에 이환<br>- 백신 맞아도 항체 보유기간이 6-8개월 ((연 2회)<br>- 폐사율은 3%로 낮지만 망아지에서는 폐사↑<br>- 주로 대증치료, 항생제나 수액 요법으로 2차적 폐렴 예방, 백신 중요!                                                                    |
| **선역**<br>(Strangle)                                                                   | - *Streptococcus equi. equi.* - 세균성 호흡기 질병<br>- 선역 증상인데 이 균이 검출되지 않으면 **의사선역**이라고 통칭.<br>(*S. zooepidemicus* : 상재균이나 기회감염을 일으켜 유산 유발)<br>- 연 1회 백신 접종 (겨울철에 많이 발병)<br>- 특징적인 증상 : 터져나오는 농양 형성<br>- 치료는 농양 배액 및 세척, 치료 경과는 좋으나 확산이 빨라서 주의.                                                                      |
| **아프리카마역 (AHS)**<br>(Africa Horse Sickness)                                            | - 심급성 또는 아급성, 폐사율 95% 이상인 치명적인 질병<br>- 아프리카 지역에서 주로 발생. (아프리카 말 이동 금지 → 다른 지역 전파X)<br>(\*희한하게 2020년 태국/말레이시아 발생 사례  - 아마 흡혈 곤충에 의해)<br>- **폐장형**(심급성) / 심장형(아급성) / 혼합형(급성) / 발열형(가벼움)<br>- 불활화 예방 백신이 있으나, 발생 시 살처분이 최우선.                                                                                      |
| **마전염성빈혈** (EIA)<br>(Equine Infectoius Anemia)                                         | - 한 번 감염되면 평생 carrier 상태, 살처분 권장<br>- 흡혈곤충, 기계적 전파, 태반 감염<br>- 잠복기 1-3주 → 고열, 빈혈, 사지부종 / 급성으로 전신쇠약과 운동실조 시 폐사 가능                                                                                                                                                                                               |
| **말 바이러스 동맥염** (EVA)<br>(Equine Viral Arthritis)                                       | - 급성 접촉성 전염병, 50% 정도 유산 발생시킴<br>- 불현성 감염 비율 높음<br>- 특별 치료법 없음<br>- 미국/유럽에서는 상재균, 엄격한 검역 체계 존재 (수출국에서 + 수입국에서)                                                                                                                                                                                                  |
| **말 허피스 바이러스 감염증**<br>(Equine Herpesvirus infection)<br>= **비강 폐렴** (Rhinopneumonitis) | - 상부 호흡기 병변, 신경증상 주 증상<br>- *Equine Herpes virus* 1, 4 <br>(EHV-1 ; 유산, 호흡기, 신경증상 / EHV-4 ; 주로 호흡기)<br>- 특징적 증상 : 고열, 습성 기침, 폐렴 / 뇌염 및 마비 증상 [페사] / 임신 말기 유사산<br>- 임신 5/7/9개월째 예방 접종, 대증 치료                                                                                                                    |
| **수포성 구내염**<br>(Vesicular Stomatitis)                                                  | - **인수공통전염병** : 사람에서도 빈혈 등 유발<br>- 흡혈 파리와 곤충에 의해 전파, 오염 사료나 장비의 접촉 감염<br>- 특별한 치료법 없이 1-2주 안에 회복, 접촉 주의                                                                                                                                                                                                        |
| **말 바이러스성 뇌척수염**<br>(Equine Viral Encephalitis)                                        | - 중추신경계 손상으로 신경 증상<br>- **인수공통전염병** (접촉으로는 감염 X), 국내 침입 아직 없음<br>　✓ <u>동부말뇌척수염</u> (EEE) : 가장 치사율 높음, 타동물 전파 x<br>　✓ <u>서부말뇌척수염</u> (WEE) : 독성 낮음, 타동물 전파 x<br>　✓ <u>베네주엘라말뇌척수염</u> (VEE) : 치사율 50-70%, 말이 중간숙주로 타동물 전파!<br>- 고열, 급성 침울, 실명, 선회운동, 양쪽 다리를 모아 서 있거나 벽에 머리를 박음<br>- 기립불능 상태에서 발길질 → 이후 예후는 극히 불량 |
| **웨스트나일 말 뇌염**<br>(West Nile Virus Infection)                                          | - **인수공통전염병**, 사람에서 주목도 높음 (국내 미유입 but 온난화로 침입 위험 ↑)<br>- 모기를 통해 감염, 뇌염 및 신경증상 유발<br>- 고열, 침울, 근육 쇠약, 과흥분, 심하면 뇌염 증상<br>- 효과적 치료법 없음                                                                                                                                                                           |
| **말 전염성 자궁내막염**<br>(Contagious Equine Metritis)                                        | - 법정 2종 + OIE 리스트 질병<br>- 급성 화농성 자궁내막염, 수태불능(불임) 유발, 수태율 급격히 저하, 임신 초기 유산<br>- 말 수출에 영향이 있어 청정화 위해 노력<br>- 교배 시작(2월) 전 번식마 전 두수에 대해 검사(11월), 음성만 교배 가능                                                                                                                                                         |
| **말 파이로플라즈마증**<br>(Equine Piroplasmosis)                                               | - 진드기 매개 (대형 *Babesia caballi*, 소형 *Babesia equi*)<br>- 빈혈, 황달, 열 등 (잠복기 *equi* 12-19일, *caballi* 10-30일)<br>- 전국 말 대상 추적 관리 but 발병 보고 없음. 혈액검사상 존재 확인은 있음<br>- 특별한 치료법은 없으며 진드기 예방                                                                                                                            |
| **비저** (Glanders)                                                                      | - 수 주 이내 폐사 가능 / 만성형 → 피부와 폐에 결절과 궤양<br>- 법정 제2종 + OIE 리스트 질병, **인수공통전염병** (치료받지 않으면 95% 치사율)<br>- 호흡기와 피부의 삼출물에 감염력 존재<br>- 고열, 기침, 비루 등, 폐비저 / 비강비저 / 피부비저 3가지 형태<br>- 회복해도 보균 상태, 양성 말 색출 도태 조치                                                                                                             |
| **원충성 말 뇌척수염** (EPM)<br>(Equine Protozoal Myeloencephalitis)                           | - 원충에 의한 뇌척수염, 국내 미유입 but 미국에서는 크게 문제<br>- 오염된 사료나 물을 말이 섭취했을 때 체내 감염 → 원충이 뇌로 이동<br>- 후구의 근위축, 후구마비<br>- 말은 1차 숙주가 아님, 말에서 증폭되지 않음.<br>- 갑자기 보행 실조, 기립 불능, 비대칭적 후지 발병<br>- 뇌척수액 검사로 분리해야 됨 (어려움), 효과적 치료법 없음, 대부분 안락사                                                                                         |
| **일본 뇌염** (Japanese Encephalitis)                                                      | - 모기에 의해 전파, **인수공통전염병**<br>- 말이 유럽으로 이동할 때 백신 접종 의무                                                                                                                                                                                                                                                           |
