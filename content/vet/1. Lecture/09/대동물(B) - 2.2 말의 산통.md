#### 산통(Colic)

- 복강내 장기에서 통증을 수반하는 질환의 통칭 ("배앓이")
- 말 폐사의 주요 원인, 이환율과 치사율 매우 높음.

#### 산통 발생의 원인

- 해부학적으로 장 내용물이 정체하기 쉬운 구조 (위 유문부, 맹장 말단, 대결장 만곡부 및 말단)
- 긴 장간막이 소장을 복강에 연결함 → 염전 쉽게 발생
- 대결장이 큰 double U 형태, 복강 내 지지 구조 없음 → 복강 내 유리, 장 변이가 쉬움
- 구토가 어려움 

| 사양 관리                                                                           | 장 내부                                                                                             | 기타                                                                         |
| ------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------- |
| - 과식 (특히 농후사료)<br>- 부패한 사료 급식<br>- 급격한 사료 변경<br>- 음수 부족 (수분부족)<br>- 모래 섞인 목초 섭취 | - 장 내용물 정체<br>- 위장 내 발효 (가스 저류)<br>- 장결석, 식괴, 헤어볼<br>- 장내 기생충 감염<br>- 장 협착, 경련<br>- 장 변위, 염전, 중첩 | - **기후 급변** (多)<br>- 체온 하강<br>- 스트레스<br>- 치아 이상<br>- 장기간 항생제 투여<br>- 원인 불명 |

Colic의 기전
- 장 내용물 축적으로 장벽 확장/신장 → 장벽내 예민한 신경 자극
- 장 변위 발생 → 장간막에 팽팽한 긴장 → 통증
- 장 염전 → 혈관 폐색
- 장 내외측 장벽에서 발생하는 염증

#### 산통의 종류

| 종류            |                                                                                                                                                                                                |
| ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **장폐색**       | - 교액 : <u>장염전, 장중첩</u> 등에 의해 장의 혈액공급이 막힌 것. <br>　(장중첩 - 회맹부 연결부 중첩이 가장 일반적)<br>- <u>비교액성 경색</u> : 원선충의 유충에 의한 동맥 폐색<br>- <u>매복</u> : 장관의 기계적/물리적 폐색<br>- <u>장결석, 이물질 폐색</u><br>- <u>대결장 변위</u> |
| **가스산 (고창산)** | - 원발성 : 장내 탄수화물의 발효로 가스 과다<br>- 속발성 : 맹장/결장의 장폐색에 의해 2차 발생 (급성, 개복수술)                                                                                                                          |
| **경련산**       | - 예민한 말, 공포/불안 자극<br>- 심한 운동 후 다량의 찬물을 급격히 마셨을 때<br>- 목 부위 일부분에 땀이 남                                                                                                                           |

#### 증상

- 식욕감퇴, 침울, 불안, 배뇨 자세/견좌 자세, 심한 전신 땀흘림
- 복부팽만, 호흡수/맥박수/체온 증가, 장운동 감소
- 통증 시 → Pawing(앞발 긁기), 윗입술을 말아올림, 몸뻗치기(: 소장중첩), 등을 대고 뒹굴기(: 위궤양)
- 아픈 쪽을 바라봄, 몸을 뒤로 길게 늘어뜨림, 누우려 함

#### 진단

| 진단 항목                                     |                                                                                                                                                                                              |
| ----------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 병력 청취                                     | - 최근 변동 : 급식, 급수, 환경, 운동량<br>- 급수 부족, 거친 등겨 급식 → 장폐색<br>- 모래 마당, 흙 바닥 사육 → 모래산<br>- 어린 말의 이물 섭식 → 소결장 폐색                                                                                     |
| 마체임상검사 (P/E)                              | - 전신 상태와 통증 정도 : 땀, 앞발, 머리, 다리, rolling, stretching<br>- 체온 : 정상이거나 통증으로 인해 약간 상승 (\* 직장검사 전 체온 측정!)<br>- 심박수 : 정상 40회/분 → **급격한 상승 (80-90회/분**) 시 완전 폐색, 염전 시 40-70회                        |
| 장음 청진                                     | - 좌측 상부에서 소장/결장, 하부에서 대결장 ⇒ mixing, progressive sound<br>- 우측 상부에서 맹장, 하부에서 대결장 맹장 ⇒ 파이프 통과하는 물 소리<br>- 맹장 부위에 가스가 가득 차면 ping sound                                                          |
| 직장 검사                                     | 비장 → 신장 → 십이지장 → 맹장 → 소결장 → 대결장 만곡부 검사<br>복강 후반 1/3 검사 가능                                                                                                                                    |
| 위관 삽입                                     | - 식도, 상부 소화기 폐색 여부 진단 및 약물 투약<br>- 식도는 튜브 진입에 저항이 있는 반면, 기관은 저항 없이 들어가니 주의<br>- 위관은 산통 시 [위확장 완화 / 진단 목적 / 약물 투여] 위해 반드시 실시!                                                                 |
