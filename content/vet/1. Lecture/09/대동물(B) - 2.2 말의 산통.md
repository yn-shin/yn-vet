# 1. 산통
#### 산통(Colic)

- 복강내 장기에서 통증을 수반하는 질환의 통칭 ("배앓이")
- 말 폐사의 주요 원인, 이환율과 치사율 매우 높음.

#### 산통 발생의 원인

- 해부학적으로 장 내용물이 정체하기 쉬운 구조 (위 유문부, 맹장 말단, 대결장 만곡부 및 말단)
- 긴 장간막이 소장을 복강에 연결함 → 염전 쉽게 발생
- 대결장이 큰 double U 형태, 복강 내 지지 구조 없음 → 복강 내 유리, 장 변이가 쉬움
- 구토가 어려움 

| 사양 관리                                                                           | 장 내부                                                                                             | 기타                                                                         |
| ------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------- |
| - 과식 (특히 농후사료)<br>- 부패한 사료 급식<br>- 급격한 사료 변경<br>- 음수 부족 (수분부족)<br>- 모래 섞인 목초 섭취 | - 장 내용물 정체<br>- 위장 내 발효 (가스 저류)<br>- 장결석, 식괴, 헤어볼<br>- 장내 기생충 감염<br>- 장 협착, 경련<br>- 장 변위, 염전, 중첩 | - **기후 급변** (多)<br>- 체온 하강<br>- 스트레스<br>- 치아 이상<br>- 장기간 항생제 투여<br>- 원인 불명 |

Colic의 기전
- 장 내용물 축적으로 장벽 확장/신장 → 장벽내 예민한 신경 자극
- 장 변위 발생 → 장간막에 팽팽한 긴장 → 통증
- 장 염전 → 혈관 폐색
- 장 내외측 장벽에서 발생하는 염증

#### 산통의 종류

| 종류            |                                                                                                                                                                                                |
| ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **장폐색**       | - 교액 : <u>장염전, 장중첩</u> 등에 의해 장의 혈액공급이 막힌 것. <br>　(장중첩 - 회맹부 연결부 중첩이 가장 일반적)<br>- <u>비교액성 경색</u> : 원선충의 유충에 의한 동맥 폐색<br>- <u>매복</u> : 장관의 기계적/물리적 폐색<br>- <u>장결석, 이물질 폐색</u><br>- <u>대결장 변위</u> |
| **가스산 (고창산)** | - 원발성 : 장내 탄수화물의 발효로 가스 과다<br>- 속발성 : 맹장/결장의 장폐색에 의해 2차 발생 (급성, 개복수술)                                                                                                                          |
| **경련산**       | - 예민한 말, 공포/불안 자극<br>- 심한 운동 후 다량의 찬물을 급격히 마셨을 때<br>- 목 부위 일부분에 땀이 남                                                                                                                           |

#### 증상

- 식욕감퇴, 침울, 불안, 배뇨 자세/견좌 자세, 심한 전신 땀흘림
- 복부팽만, 호흡수/맥박수/체온 증가, 장운동 감소
- 통증 시 → Pawing(앞발 긁기), 윗입술을 말아올림, 몸뻗치기(: 소장중첩), 등을 대고 뒹굴기(: 위궤양)
- 아픈 쪽을 바라봄, 몸을 뒤로 길게 늘어뜨림, 누우려 함

# 2. 산통 진단

# 3. 산통 처치
- 대부분 내과적 처치로 개선 : 진통소염제 + 수액 + 항생제 + 윤활제 투여
- 구조적 폐색인 경우 외과적 처치

| 내과적 처치    |                                                                                                 |
| --------- | ----------------------------------------------------------------------------------------------- |
| 수액 처치     | 탈수 예방, 혈액 공급 원활<br>14-16G 카테터로 8-10L/1-2Hr<br>- 순환기 쇼크 시 20L/hr, 필요 시 30-100L/day               |
| 윤활제/사하제   | 치료(장 윤활+분변 연화) + 진단(장관 폐색 확인) 목적<br>- Fluid, **Mineral oil**                                    |
| 진통제       | - NSAIDs : **Flunixin meglumine** (가장 많이)<br>- 진정 마취 : **Detomidine**<br>- Opoids : Butorphanol |
| 위장관 운동 항진 | - Neostigmine                                                                                   |

| 외과적 처치                                     |                                                                                                                                                                                                                                |
| ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **수술 지표**                                  | - uncontrollable / severe pain<br>- depressed<br>- <u>alkaline yellow fluid > 5L</u> (위 후방 폐색 가능성)<br>- 직장 검사 → 소장 확장, 대결장 확장 변위, 이물 촉진<br>- <u>심박 > 75회/분</u>, 약한 맥박, CRT > 2.5s, 사지 말단 차가움 (shock)<br>- 내과적 치료에 반응 없음, 통증 지속 |
| **개복 수술**                                  | 전신 흡입 마취, 정중 절개<br>- 복강 장기를 최대한 노출할 수 있고, 수술 시간이 빠름<br>\* 대장 및 소장 탐색 : 맹장의 lateral band가 cecocolic fold로 이어짐<br>\* 소장 탐색 : 맹장을 뒤집으면 dorsal band → 따라 내려가면 ileocecal fold                                                       |
| **Enterotomy<br>Resection<br>Anastomosis** | - 장 운동성과 생존성을 판단해서 절제 결정<br>- 대결장의 경우 pelvic flexure의 antimesenteric surface 절개<br>- 누출, 협착, 유착의 합병증이 있을 수 있으니,<br>- 새지 않게 봉합 / fibrous 형성 최소화 / 적절한 내강 확보                                                                     |
| **술후 처치**                                  | 최초 3일 집중 처치 : 드레싱, 수액/소염제/항생제 투여<br>- 1일간 절수, 2일간 절식, 3일째부터 부드러운 건초 공급<br>　(너무 일찍 급여하면 ileus 유발)<br> - 12-14일 후 복부 봉합사 제거                                                                                                      |
| **술후 합병증**                                 | ileus (원인불명) / peritonitis (오염, 내용물 누출) /<br>**myopathy** (장시간 전신마취 및 압박) / Laminitis (endotoxin ↑) / Pneumonia                                                                                                                |

# 4. 산통 종류 및 수술적 교정

<font color="#7f7f7f">대강 정리는 했으나 1도 안 중요할듯</font>

1. 장중첩 (Intussusception)
    - 장의 일부가 다른 장 속으로 들어가는 상태로, 주로 어린 말에서 발생.
    - 증상은 소장의 완전 폐색과 유사하며, 중첩된 장을 절제해야 함.
2. 염전 (Volvulus)
    - 장이 360도 이상 회전하여 꼬인 상태로, 급성 산통을 유발.
	- 절제해야 하는 부위가 소장의 50% 이상이면 안락사
3. 교액 (Strangulating Obstruction)
    - 노령마(10세 이상)에서 발생
    - 장간막의 지방종이 소장을 감싸 폐색을 유발하는 경우.
    - 지방종 제거 후 소장의 운동성을 확인하고, 필요시 절제 후 문합.
4. 내부 허니아 (Internal Hernias)
    - 복강 내 정상 또는 병변 구멍을 통해 소장이 감돈.
	- Epiploic foramen incarceration : 초기 확진이 어려워 수술 단계에서 예후 불량
	- Mesenteric incarceration : 절제 후 문합
5. 대결장 변위 (Displacement of Large Colon)
    - 대결장이 정상 위치에서 벗어난 상태로, 혈액 공급은 유지됨.
    - 수술적 교정이 필요하며, 대결장을 정상 위치로 복원.
6. 좌측 대결장 변위 혹은 끼임 (Left dorsal displacement or nephrosplenic entrapment of large colon)
	- Left large colon이 복벽과 비장 사이에 끼인 상태
