#### Etiology


-

- MALT lymphoma : 점막에만 분포한, 느리게 자라는 암. 


	- Helicobacter의 심각도는 사람에서 훨씬 큼.



|                                                                                                                                   |                                                                                                                                          |
| --------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| 사람 치료                                                                                                                             | 개/고양이 치료                                                                                                                                 |
| 	- Helicobacter를 치료하면 MALT도 치료된다. MALT로 진행될 수 있는데, 그럼 제균치료를 해야 함.<br>	- 위염이 심하므로 제산제를 써서 산을 억제, <br>	- PPi + Clarithromycin + AMC | - 증상이 없으면 치료하지 않아도 됨.<br>	- Helicobacter만 나왔으면 항생제<br>		- 고양이에서 저등급 lymphoma가 발생 → 항암제. (연관성 떨어짐)<br>	- PPi + <u>metronidazole </u>+ AMC |

#### Diagnosis
- Silver stain → Helicobacter가 명확하게 검정으로 보임. (but 요즘 잘 안 함)
- 조직검사


- 위에 사는 이상한 세균
	- 위산에서 알칼리성 
	- Urease
- Phenol red : 원래 노란색, 중성 이상, 산성에서 핑크색으로 바뀜. 여기에 Helicobacter에 감염된 점막을 넣으면, urease에 의해 붉은색으로 바뀜. 
Campilobacter-like organism. : helicobacter 진단용.

#### Treatment
- PPi + [[metronidazole|metronidazole]] + 아모크라 ([[Amoxicillin+Clavulanic acid|Amoxicillin+Clavulanic acid]])
