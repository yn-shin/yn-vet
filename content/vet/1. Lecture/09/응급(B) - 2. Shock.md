# 1. What is shock?

- **몸에 산소가 부족한 상태 (tissue hypoxia) = Shock** 
- Oxygen delivery (DO2) ≠ Cellular oxygen consumption (VO2) ⇒ 산소의 밸런스가 깨짐. 조직에서 필요로 하는 만큼 산소를 공급하지 못함.
- 보통 consumption보다는 delivery의 문제

#### Perfusion triangle

- Perfusion을 일으키는 세 가지 요소를 perfusion triangle이라고 함.
- **Pump** = 심장 (엔진)
- **Content** = 혈액 (엔진을 움직여줄 휘발유)
- **Pipe** = 혈관 (둘을 연결하는 파이프)

#### Lactate

>[!note] Lactate
>![[Pasted image 20250405210254.png|250]]
>- Aerobic (호기성 대사) → 36ATP 생성
>- Anaerobic (혐기성 대사) → 2ATP + <u>lactate 생성</u>

- 가장 유용한 parameter, 조직의 저산소증이나 관류 상태 개선을 확인하기에 유용. Shock 환자에서 반드시 체크해야 함.
- 쇼크 환자에서 산소 공급이 부족해서 혐기성 대사가 이루어지면서 **Lactate 증가** ⇒ **Hyperlactermia**.
- 쇼크 환자의 resuscitation으로 산소가 공급되기 시작하면 호기 대사로 전환되면서 **Latctate 감소**.

# 4. Shock Evaluation

>[!note] Perfusion parameter
>1) TPR
>2) BP
>3) Pulse quality
>4) MM color / CRT
>5) Consciousness (Mental)
>6) Dehydration
>7) Other gross abnormalities (Ambulation, External injuries)

#### 1) Ineffective perfusion

- 가장 중요한 건 <u>TPR & BP</u>
- (1) **TPR** : 체온 하강(hypothermia), 빈맥→서맥(Tachy/bradycardia), 호흡 패턴 비정상(Abnormal breathing pattern)
- (2) **BP** : 저혈압(hypotension) - sys BP 100 이하로 떨어지면 소생 필요 (정상 120)

>[!note] BP measurement
>- Oscillometry : 대형견에서는 정확하게 나옴. But 고양이, 소형견, 서맥, 관류가 안 좋은 환자에서는 부정확한 측정. 정상적인 대형견 아니면 어려움.
>- Doppler : 훨씬 정확하고, 작고 서맥인 환자에서도 측정 가능. Technique의 영향을 받고, MAP(diastolic)은 잴 수 없음. 
>	- 고양이는 실제보다 낮게 측정되는 경향이 있어 +14mmHg 해서 보정
>- Cuff size : 둘레의 40-60%를 채우는 크기. 너무 작은 커프를 쓰면 혈압이 높게 측정됨. Serial monitoring이 필요하므로 일관성이 중요 (사용한 기계, 커프 사이즈, 수축기or이완기, 측정 부위 등)
>- Cuff location : 도플러는 앞다리(Median a.)에서 주로 측정. 혹은 뒷다리(Metatarsal a.)나 꼬리(Coccygeal a.)에서도 가능.

- (3) **Pulse quality** : bounding, weak
	- Femoral, inguinal 부위에 손가락을 대서 평가 가능.
	- 평소 정상 상태일 때 많이 느껴봐야 정확한 평가 가능. 
	- Bounding : 
- (4) **Dehydration** : 5~12%까지 평가, 주관적, 점막의 수분도 확인.
	- skin turgor는 종에 따라 차이가 커서 신뢰도 낮음.
	- 12% 이상 탈수 : 후기 비보상기에 해당할 만큼 진행됨, 사망 위험도 높음.

>[!note] 탈수 평가 및 탈수의 위험도
>![[Pasted image 20250405215212.png|250]]
>- 세포 안에 fluid가 가장 많음(크고 강력한 물탱크, reservoir 역할).
>- Intravascular fluid는 심장, 뇌, 폐 혈액을 운반하는 애들로, 항상 정상 볼륨이 유지되어야 함.
>- 체내에 fluid가 부족하면 세포 → 혈관으로 fluid shift가 발생, 나중에 다시 채워짐. 계속 퍼주기만 하면 점점 세포 안의 물이 비어 가고, 혈관 쪽으로도 수분 보충 불가. ⇒ <u>dehydration</u>
>- **8%** 탈수까지는 계속 세포가 물을 보내서 intravascular volume을 채워줌 ⇒ dehydration은 있으나 hypovolemic하지는 않음.
>- **8-10%** 에서는 탈수는 있으나 fluid resuscitation까진 필요 없고, 유지용량 정도로도 괜찮다
>- **10%** 부터는 hypovolemic shock, 쇼크 증상 시작, 응급 처치 필요 (50cc 가득 15분 동안 쭉)
>- **12%** 부터는 심각한 hypovolemic 상황으로 목숨이 위험.

#### 2) Resuscitation-required situation

![[Pasted image 20250405222454.png|700]]
- 쇼크 환자가 오면 일단 혈압을 잰다.
- 혈압이 낮으면 tachycardia(보상성) or bradycardia(비보상성)를 확인한다. 
- 다른 parameter들도 확인해서, hypotension, hypoperfusion 상황임을 파악하면 resuscition 바로 돌입.
- 단, cardiogenic shock인지 rule-out 필요. (일반적인 fluid resuscitation을 못하기 때문)

# 5. Other formulas

이외에 추가적으로 볼 것들 
- Shock index : trauma history 등 
- 전신에 심한 염증이 있는 상황(⇒ **Distributive** shock)에서는 **승압제**를 고려해야 함.
	- SIRS (전신 염증 증후군, systemic inflammatory response syndrome)
	- MODS (다발성 장기 부전 증후군, multiple organ dysfunction syndrome)
	- Sepsis : SIRS+전신 감염
