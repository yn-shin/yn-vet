# 1. 말의 소화기 구조

#### 말의 소화기관 (Equine gastrointestinal track)


![[Pasted image 20250423081210.png|600]]
#### 입

- 견치가 수말에게만 존재함
- 제1전구치 = 낭치(wolf lteeth) : 저작 기능이 없음, 매우 작게 존재함
- 재갈을 어금니 끝부분에 물림. 낭치에 이상이 생기면 통증을 유발해서 재갈 물리기 어려울 수 있음. 불편을 주면 제거 필요
- 영구치는 4년에 걸쳐 생성됨.

- 단위 동물
- 위 용적이 체구에 비해 작음

#### 위

- 중앙에서 약간 좌측
- J자 구조, 강력한 괄약근 ⇒ 구토가 어려움.
	- 스스로 해소하기 어려워 위확장, 위파열 발생.

#### 입안에서 발생하는 질병

- 치아 검사 : mouth gag 장착해서 입 안을 열
	- 말은 소와 달리 날카롭게 물어 끊어버림. 물리면 위험.

| **위확장** (gastric dilatation)   | - 위에 사료가 가득 축적 → 내려가는 통로가 막혀 위가 확장됨.<br>- 과량의 물 흡수, 공기 흡수에 의해서도 발생 가능<br>- 심한 급성 통증을 유발, 위 압력을 완화하기 위한 견좌 자세<br>- 복통 → 심박수 80 이상<br>- 위폐색 : 수술 필요                                                                                                   |

| 위 종양 (gastric neoplasia)       | pass                                                                                                                                                                                                                                                |
| **소장 장염**                      | - *Salmonella*, *Clostridium* spp 등에 의해 발생<br>- 3세 이상에서 다발<br>- 소장 폐색, 염전 등과 감별<br>- 80% 치유됨, 치료 기간 통상 3-4일<br>- 위관을 상시 장착하여 수시로 감압(gastric decompression)<br>- 합병증 예방 제엽염 : 혈관 독소들이 제엽에 염증을 유발해서 심한 파행과 통증, 제벽의 탈락을 가져오는 질환. 위장관에 문제가 생겼을 때 발생 가능. |
| **소장운동 항진** (SI hypermotility) | - 전형적인 심한 복통<br>- 복부 청진 시 장 운동음을 격렬하게 청취                                                                                                                                                                                                            |
| **급성 설사** (acute diarrhea)     | - 심한 설사 미치 복통, 빠르게 증상 악화, 쇼크까지 이어질 수 있음<br>- 특정 항생제나 NSAIDs 장기간 사용에 의해 발생하기도 함                                                                                                                                                                      |
| **만성 설사** (chronic diarrhea)   | - 원선충(strongylus spp)이 가장 문제 많이 일으킴.<br>- 장내 모래 축적되어 장 점막을 자극하기도 함.<br>- 가장 특징적인 증상 : 체중 감소, 피부 거칠어짐 <br>- 치료 : 구충제, 수액 및 전해질 공급                                                                                                                    |
| **변비** (constipation)          | - 망아지에서 대변 배출 능력이 없어 태변 정체 → 산통 발생<br>- 치아 질환, 기생충 감염, 잘게 분쇄된 곡물 사료 다량 급여 시 발생<br>- 물이 얼어버리거나 해서 음수 불량 시에도 발새애<br>- 진단 : 복부 만져보면 단단한 분변들이 만져짐. 병력 청취도 중요한 힌트<br>- 치료 : 사하제 투여, 대량 수액 정맥내 투입                                                         |
| 복막염 (peritonitis)              | - 다른 대동물과 비슷, pass                                                                                                                                                                                                                                  |
