# 4주차 (3/28) : `I`

# Inflammatory CNS disease (염증성 뇌질환)

- Granuloma : 감염이 없는 상태에서 <u>macrophage & lymphocyte가 혼재</u>된 상태로 존재.
- 염증세포가 많이 존재 → 비중(SG) 높음, protein 농도 높음.
- Mononuclear cell(단핵세포) 증가 - 주로 monocyte, lymphocyte 증가
	- GME는 대부분 lymphocyte 침윤 (**T림프구** > B림프구)

| Inflammatory CNS disease                                                                                            |                                        |
| ------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| Infectious                                                                                                          | Non-infectious                         |
| - Fungal : 우리나라는 rare.<br>- Bacterial : uncommon<br>- Protozoal : uncommon - *Babesiosis*<br>- Rickettia : uncommon | - Immune mediated, autoimmune : common |
### CSF

>[!note] CSF : cerebrospinal fluid (뇌척수액)
>- 맑고(clear) 투명(colorless)
>- Protein content : < 20mg/ml
>- WBC : < 6cells/ul 이하로 존재
>- small lymphocyte가 많음
>- 뇌압 : < 173 mmH2O


| CSF 소견           |                                                                                                                                                 |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
|                  | **Increased cells & porteins** 는 모두 공통 !!<br>                                                                                                   |
| **Inflammatory** | - 염증, 바이러스성 질환 :<br>✔️ cell count : > 6cells/ul<br>✔️ 단백질 농도 : > 25mg/ml<br>✔️ Mononuclear cells 증가 (lymphocytes, reactive macrophages)         |
| **Fungal**       | -  Cryptococcosis : <br>✔️ Mixed cell population<br>(lymphocyte, reactive macrophage, neutrophils 등이 혼재)<br>✔️ Capsulated, budding yeast가 볼록 보임 |
| **Bacterial**    | - Septic spinal tap<br>✔️ Neutrophil이 훨씬 많음<br>✔️ culture해서 세균 확인                                                                               |
| **Neoplasia**    | - Lymphosarcoma가 많음<br>✔️ neoplastic cells 증가<br>(choroid plexus papillomas, ependymomas, lymphosarcoma에서 나타남)                                  |

- 혈청에서의 titer와 뇌척수액의 titer를 비교 평가 (<u>paired serum and CSF</u>)

#### SRMA (CRTS) : 치료 방향 반대

>[!note] CRTS (Corticoid-responsive tremor syndrome) 
> - Tremor 증상이 있었는데, 스테로이드 약물에 반응하여 떨지 않음.
> 	- MRI 결과에도 나타나지 않지만, 스테로이드 먹이면 좋아지더라
> - 과거에는 털이 하얀 애들한테만 생겨서 white shaker syndrome이라고 했으나,
> - 요크셔테리어처럼 털이 하얗지 않은 개에서도 나타나서 "CRTS"로 바뀜.

>[!note] SRMA (Steroid responsive meningitis-arteritis / Suppurative meningitis & arteritis)
>(Neutrophil이 많아서 suppurative하게 나타남)
>- Steroid 쓰면 dramatic하게 좋아짐. but 세균 감염은 아닌지 확실히 구별해야 함!
>- MRI상 나타나는 이상은 없지만, 뇌척수액에 **neutrophil 증가** (pleocytosis) ⇒ 뇌수막에만 염증이 온 것으로 여겨짐 (meningitis)
>- <u>CRTS = SRMA</u> 라고 생각됨.

# 5주차 (녹강) : `I`
### GME (Granulomatous meningoencephalitis)

육아종성 뇌수막염.
- 비감염성 뇌수막염을 유발.
- Small breed dog에서 많이 발생 - 푸들, 말티즈, 퍼그, 요크셔테리어 등
- 젊은 나이에 많이 발생하며 매우 치명적, 오랜 기간 치료 필요.

### 다양한 meningoencephalitis

- **GME** (Granulomatous meningoencephalitis) : 주로 림프구가 병변에 침윤하여 비감염성으로 발생한 뇌수막염.
	- Granulomatous = lymphocytes, macrophages, monocytes, neutrophils가 혼재되어 나타난 구조.
	- 이런 육아종성 염증이 CNS에 분포함.
- **NME** (Necrotisizing meningoencephalitis) : 림프구가 침윤하여 뇌 일부에 괴사가 진행, GME보다 악화된 소견.
	- Pug meningoencephalitis : pug에 호발하여 붙은 이름, 과거에 사용
- CDV-meningoencephalitis : Distemper virus에 의한 viral meningoencephalitis
	- Mucoencephalitis : 주로 white matter에 점액성 염증이 나타나서 붙은 이름
- 원인을 찾지 못한 경우, MUE (Meningoencephalitis or unknown etiology)
- Pons-medullary GME : seizure는 발생하지 않고 호흡 곤란, 자율신경계 이상, cranial nerve 이상, 빈맥성 부정맥이 발생함.

### Types of GME


| Types             |                                                                |
| ----------------- | -------------------------------------------------------------- |
| Ocular form       | Optic nerve의 염증이 optic neuritis를 유발하여 시각 장애                    |

| Disseminated form | 뇌뿐만 아니라 눈, optic nerve, brain stem, spinal cord 등에 다양하게 발생한 경우 |

### Therapy & Prognosis of GME


| 약물                                 |                                                                                                                                                                                                                                                                            |

| **Steroid (Prednisolone)**         | - 항염 효과로 좋음<br>- 장기간 사용 시 의인성 Cushing 위험<br>- 점차 감량하면서 최소 용량으로 유지                                                                                                                                                                                                          |
