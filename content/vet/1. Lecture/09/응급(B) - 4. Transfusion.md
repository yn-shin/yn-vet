>[!check] Case 1
>- 12y, toy poodle, male, 4.84kg
>- Dx - TCC, hematuria, melena, DIC
>- PT 지연, APTT 지연, TEG 비정상 ⇒ Consumptive coagulopathy (소모성 응고병증)
>- 지속적인 염증으로 DIC 발생

수액 처치만 하면 → dilution이 가장 문제. 혈액이 희석됨.
이 환자에서 필요한 Resuscitation = 수혈

## 수혈의 원칙 : 3R
- **Right Timing** (언제?)
- **Right Material** (무엇을? 얼마나?)
- **Right technique** (어떻게?)

## 1. Right Timing
- No transfusion trigger : 수혈의 타이밍을 판단할 수 있는 명확한 trigger는 없음.
- tissue hypoxia(shock)일 때 조직으로 산소를 운반해주기 위해 수혈 필요.

| 수혈의 판단 기준                                                                                                                                                                            |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Clinical signs**                                                                                                                                                                   |
| Shock와 관련된 증상이 있을 때 수혈하 (주관적, 경험적 영역..)<br>: tachycardia, pallor, weakness/exercise intolerant, elevated lactate, tachypnea, +/- hypotensive                                         |
| **PCV**                                                                                                                                                                              |
| - <u>Chronic</u> → PCV < 15% 시 (재생불량성 빈혈, 종양)<br>   : 빈혈의 상황에 적응할 수 있는 충분한 시간이 있었음<br>- <u>Acute</u> → 25-45% (suspect > 25-30% loss)<br>   : 급성은 적응할 시간이 없음. 혈액량의 30% loss 일어나면 수혈. |
| **Hemoglobin**                                                                                                                                                                       |
| - < 7g/dL : hypoperfusion<br>- < 5-6g/dL : high risk of death                                                                                                                        |
의학적으로는 Hgb가 훨씬 sensitive한 marker이지만,
PCV가 훨씬 기계의 영향을 덜 받고 일관적인 수치여서 PCV로 기준 잡음.


## 2. Right Material

#### 1️⃣ RBC Transfusion

> - Wholed blood
> 	- Fresh WB
> 	- Stored WB
> - Packed RBC (pRBC)

##### Fresh whole blood (FWB)
- 채혈 후 **4-6시간 이내**, <u>혈소판이 살아있는</u> 혈액
- RBC, WBC, platelet, coagulation factor 모두 있음
- Indication
	- Acute hemorrhage (surgery, trauma, organ rupture)
	- Evans syndrome (IMHA+IMT)
##### Stored whold blood (WB)
- 채혈 후 6시간이 지나면(=혈소판 파괴) 저장해서 사용 
- Stored 1-6℃ for 28 days (냉장고에 <u>1달까지</u> 저장)
- RBC, WBC, coagulation factor / ~~Platelet~~ (혈소판은 깨졌지만 응고인자는 살아 있음)
- Indication
	- Acute hemorrhage
	- Anemia with <u>coagulopathy</u>
##### Packed RBCs (pRBC)
- Whole blood에서 RBC 분리
- Stored 1-6℃ for 35 days (보관 기간이 좀 더 길다)
- RBC, WBC (+with minimal plasma) : 오직 적혈구 수혈만 가능, 응고인자/albumin 등은 없음
- Indication 
	- Anemia without coagulopathy
	- FFP(Fresh plasma)와 섞어 전혈처럼 사용할 수 있음


- Leukoreduction : 백혈구를 걸리는 필터, storage
	- pre-storage가 나
