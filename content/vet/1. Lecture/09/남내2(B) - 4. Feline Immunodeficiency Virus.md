# 1. Etiology & Epidemiology

>- Family *Retroviridae*
>	- Subfamily *Orthoetrovirinae*

- Exogenous, ssRNA virus
- HIV와 형태적 유사성은 있으나, 항원성이 달라 교차감염 없음.
- 다양한 바이러스 계통 존재, 계통별 병원성 차이 있음.
- 전 세계 분포, Older, male, outdoor cats에서 호발함.

#### Transmission

- **Aggressive biting behavior** (가장 흔함) : 구강 림프조직에서 매우 빠르게 증식

- 인공수정, 태반, 주산기 감염은 드물게 발생함.
- 절지동물 매개 전파는 거의 없음.
- 혈중 바이러스 농도 상승 시 질병 심화, 해당 기간은 짧음.

| Phase                          |                                                                                                                                                                                                                                                                                     |
| ------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Primary phase**              | - 바이러스가 몸 전체로 퍼지며, 경미한 열, 호중구 감소, 전신 림프절 종대가 나타남<br>- 백신 접종 후 나타나는 증상보다 더 뚜렷할 수 있음                                                                                                                                                                                                  |
| **Subclinical, latent period** | - 건강한데 감염된 중년령 고양이: 평균 3년령<br>- 자연적으로 감염되어 임상 증상이 나타난 고양이: 평균 10년령<br>- 잠복기가 매우 길 수 있음 (수년~7년 이상)<br>- 전반적으로 면역 기능이 점진적으로 감소함<br>　: CD4+ 림프구 감소 / mitogen(분열)에 반응 / 세포매개 면역 cytokine(IL-2, IL-10 등) 생산 감소 /  호중구, NK cell 기능 저하<br>- 반면, 체액성 면역(항체 생성)은 비교적 잘 유지되며, 다양한 면역글로불린이 증가함 |
| **Immune deficiency stage**    | - 수개월 - 수년 동안 면역결핍이 진행되어, 2차 감염에 취약해짐<br>- Coinfection<br>	- FeLV 동반 감염 가능 : FIV의 primary/deficiency phase를 더 악화시킴<br>	- FHV (*Felis catus gammaherpesvirus*) 도 FIV-infected cats에서 관찰된 바 있음<br>	- FIV도 T-cell lymphoma 발생 가능성이 있으나, FeLV에 비해 드묾 <br>                               |

# 2. Clinical Features

- 증상은 다음 2가지 원인으로 나타남
	- Direct viral effects (바이러스 자체)
	- Secondary infection (면역 결핍에 의한 기회 감염)

- Primary phase에서는 fever & generalized lymphadenopathy
- 비특이적 증상으로 내원 (체중 감소, 기력 부진, 우울, 장기 관련 이상들)
![[Pasted image 20250523163011.png|900]]

- 엄청 broad하고, 대단히 특이적인 증상도 없음.


- FIV antibody test : 양성을 prove하지 않음
- 다른 검사로 potential cause를 배제해야 함.

# 3. Diagnosis

|                                             |                                                                                                                                                                                 |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Hematologic**                             | ![[Pasted image 20250523163358.png\|250]]<br>- 정상 면역 상태에서는 CD4/CD8 > 1 상태를 유지하고 있음.<br>- Acute하면 CD4가 급격히 떨어지고 CD8이 급격히 증가해서 <1<br>⇒ <u>Ratio가 역전됨!</u>                         |
| Bone marrow                                 | 종양성 변화<br>(Maturation arrest (myelodysplasia), lymphoma, leukemia)                                                                                                              |
| Imaging                                     | FIV 자체와는 특징적인 증상 X                                                                                                                                                              |
| **Antibodies**<br>[Triple Feline Kit]       | - 혈청으로 검사 / 또는 대안으로 침(saliva) 가능<br>- **위양성** \| 백신 접종에 의한 항체 / 모체이행항체 (<6개월령)<br>- **위음성** \| 진행이 너무 빨라서 항체가 아직 다 형성되지 않음<br>      / 영원히 항체가 안 생기는 고양이도 있음 (never seroconvert) |
| Virus isolation / **RT-PCR** / Western blot | - PCR이 가장 접근성이 좋고 빠름.<br>- 한계 : 실험실마다 표준화가 부족해 위양성, 위음성 가능성 있음<br>- 6개월 미만 고양이에서 모체이행항체로 인한 위양성이 의심될 경우, 60일 간격으로 재검사 권장.                                                       |

- 백신을 맞은 적이 없는데 항체가 검출되었다면?
	- FIV에 노출되었거나, 감염이 지속(persistent infection)되는 것
	- 반드시 FIV로 인한 질병이라는 건 아님 (2차 감염원일일 수도)
- FIV와 관련된 임상 증상들은 기회감염 등 다른 원인에 의해 나타날 수 있으므로, 추가적인 진단 절차가 필요함.

# 4. Treatment

- Potential한 원인을 잘 찾아 치료해야 함.
- FIV 때문만이 아닐 수 있음 ⇒ 원인 확인

- FIV에 대한 치료?
	- 1) 감염 방지 & 빠른 치료를 위해 <u>항생제를 고용량(upper end)으로</u> : 면역결핍 상태이므로 장기간 필요할 수도 있음.
	- 2) Anti-lentiviral drugs : 효과를 명확히 말하긴 어려움
	- 3) Interferon
		- **Feline recombinant IFN-omega** : 5일 동안 적용, 고가
	- 4) Bovine lactoferrin : 면역력 증강 목적, 구내염 환자에 적용
	- 5) Others^[사람 HIV가 난치성이듯 고양이도 그러함.]
		- **AZT** (reverse transcriptase inhibitor azidothymidine) : 효과 검증된 바는 없음
		- Melittin, Phosphonate 등
	- 6) Human recombinant EPO(erythropoietin) 
		- RBC, WBC 증가 + 바이러스 증식이나 부작용 관찰되지 않음 ⇒ 빈혈 관리로 고려
		- Contraindicated : Human recombinant [[GM-CSF|GM-CSF]]  (발열, 항체 생성, 바이러스 증식)
	- 7) FIV 고양이에서 비타민 부족 관찰 → Vitamin 투여, 햇빛 쐬기? 

# 5. Zoonotic Aspects and Prevention

- should be housed indoors
    - FIV 감염 고양이는 반드시 실내에서 사육해야 함.
    - FIV-naïve 고양이에게 노출을 막고, 다른 고양이에게 전파를 방지함.
    - 실내 사육은 감염된 고양이의 기회감염 위험도 줄여줌.
- 1-2마리만 키우는 가정에서 더 예후가 좋았음
- Killed vaccine : 많이 사용하지 않음. (일부 국가에서 허가되어 있으나, 효능이 명확히 입증되지 않음)
- HIV and FIV
    - HIV와 FIV는 형태적으로 유사하지만, 항원성이 달라 교차감염 없음.
    - FIV에 감염된 고양이에서 사람에게 항체가 검출된 사례는 없음.
    - FIV 감염 고양이는 면역결핍으로 인해 2차 감염(기회감염)에 취약 → 임상적으로 아픈 경우에는 추가적인 진단과 관리가 필요함.
