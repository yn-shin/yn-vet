#### 2. Patellar Luxation 

대퇴골의 활차구에서 슬개골이 안쪽/바깥쪽으로 탈구되는 것.

- 선천적인 경우가 대부분 (84%), 외상으로도 발생 가능 (후천 16%)

- 주로 내측 슬개골 탈구 (Medial patellar luxation)
- 소(중)형견 : 대형견 = 10:1
- 양측성 탈구는 20-25% 정도
- 대형견에서는 Lateral patellar luxation이 소형견에 비해 많다
- Female : Male = 1.5:1
- Clinical signs : Lameness

- QA : Acetabulum의 cranial edge에서 부터 trochlea의 중심까지 이은 선 ~ Trochlea의 중심에서 Tibial tuberosity를 이은 선의 각도.
	- 일반적으로는 10.5도임
	- 슬개골 탈구의 grade가 올라갈수록 QA가 올라감.
	- 수술할 때 QA를 정상으로 만들도록 고려

| Lameness grade |                                             |
| -------------- | ------------------------------------------- |
| Grade 0        | No lameness                                 |
| Grade 1        | Subtle, intermittent weight-bearing         |
| Grade 2        | Subtle, consistent weight-bearing           |
| Grade 3        | Obvious, weight-bearing                     |
| Grade 4        | Intermittent, non weight-bearing            |
| Grade 5        | Consistent, non weight-bearing<br>(3다리로 걸음) |

| Patellar luxation grade |                                                      |
| ----------------------- | ---------------------------------------------------- |
| 1기                      | 슬개골을 손가락으로 밀었을 때 빠졌다가, 손가락을 떼면 다시 돌아옴                |
| 2기                      | 다리의 폄과 구부림에 따라 patella가 원위치됨. (extension하면 원래대로 들어감) |
| 3기                      | 슬개골이 빠져있으나, 밀어서 활차구 안으로 넣을 수 있음.                     |
| 4기                      | 슬개골이 빠진 상태에서 움직이지 않아 손가락으로 활차구 안으로 이동시킬 수 없음.        |

- 골재건술: 경골결절변위술, 활차구성형술, 비골두변위술, 슬개골절제술, 대퇴골 혹은 경골절단술

>[!tldr] Block recession - 활차구성형술의 하나
>- A, Use a thin saw blade to make two parallel cuts just axial to both **trochlear ridges**.
>	- 15번 blade로 양쪽 condyle에 칼집을 낸 뒤, sawing하여 block처럼 조각을 낸다.
>- B, Use an **osteotome** from proximal and distal to elevate an **osteochondral block** from the patellar groove.
>	- Osteotome을 사용해서 block을 분리시킨 뒤, 
>- C, Remove bone from the bottom of the incised block to deepen the sulcus.
>	- Osteotome으로 block의 subchondral bone을 날려서 cartilage만 남게 한다.
>- D, Replace the osteochondral block
>	- Rasp를 사용해 bone block을 원래 위치로 정복시키면 활차구가 깊어진 것을 확인.

>[!tldr] TTT (Tibial Tuberosity/Crest Transplantation) - 경골결절 변위술
>- Tibial tuberosity was chopped by the osteotome.
>- The tibial tuberosity was being transplanted and fixed by Kirschner wire using wire twisters

- 수술의 적용 : 슬개골 탈구 + 파행증을 동시에 보이면 수술 필요.
- 수술 부위 종창이 가라앉는 시기는 대부분 1주일 이내.
- 수술 30일 이내에는 대부분 회복이 된다. 수술 다음날에는 통증으로 인해 파행 정도가 높음.

##### DFO (Distal Femoral Osteotomy)

- Varus 등 증상이 심할 경우, plate를 통해 고정하기도 한다.
	- 종류 : femur 내측 carus / tibia varus / femur varus & tibial valgus)

- aLDFA(anatomic Lateral Distal Femoral Angle) : 품종마다 차이가 있는데, 이를 고려해 절제할 뼈의 분량을 결정한다.<br>The **anatomic lateral distal femoral angle (aLDFA)** is the angle formed between the<u> proximal anatomic axis (PAA)</u> and the <u>distal femoral reference line (DFL)</u> to the lateral aspect of the joint. The proximal anatomic axis was defined as the line that bisects the femoral diaphysis one third and one half (blue circles) the femoral length measured from the proximal aspect of the greater trochanter.

##### 녹강

>[!tldr] 접근
>1. Patella, tibial tuberosity의 외측으로 접근함. Patella lateral 쪽으로 0.5cm 떨어져서 skin incision.
>2. Patella ligament, tibial tuberosity, distal femur가 이루는 삼각형 모양의 영역을 살짝 든 다음 puncture를 내어 관절을 엶.
>- 관절낭을 열게 되면 정상 상태일 경우 관절액이 0.5ml 정도 존재. 질병 상태에서는 관절액은 묽어지고, 양은 더 증가함. 관절낭 또한 얇은 것이 정상이나 질병 상태에서는 비후되어 두꺼워짐.
>- 관절을 열고 관절 안을 검사해 십자인대가 끊어져 있는지 확인.

>[!tldr] 1 - 활차구 성형술
>- 활차구의 깊이, 연골 상태 등을 평가. 연골이 정상일 경우 연골을 살리는 trochlear block recession을 통해 연골을 살리면서 활차구를 깊게 함.
>- Trochlear의 medial & lateral 에 ridges 존재. 
>- Ridge보다 약간 안쪽으로, 메스를 이용해 직사각형 모양으로 block을 냄.
>- 톱을 사용해 2mm 정도 파 줌.
>- Trochlear보다 약간 작은 크기의 폭을 지닌 osteotome과 mallet(망치)를 이용해서 distal 쪽, proximal 쪽에서 각각 접근해 파 줌.
>- 직사각형 모양의 연골조각을 분리해 내는데, trochlear의 양쪽 ridge가 망가지지 않도록 주의함.
>- 잘라낸 연골은 hyaline cartilage만 남길 수 있도록 rasp를 이용해 최대한 갈아냄. Subchondral bone을 제거하는 과정.
>- Rasp를 이용해 활차구를 깊게 만들어줌.
>- Block(직사각형 연골조각)을 다시 덮어주고, 거즈를 대고 살짝 눌러줌.
>- 슬개골을 다시 고랑 안으로 넣어주면, 이전보다 안정된 상태를 유지하는 것을 볼 수 있음.

>[!tldr] 2 - 경골결절변위술, TTT (Tibial Tuberosity Transposition)
>- Patellar ligament는 tibial crest(경골능)에 insertion.
>- Tibial crest의 가쪽에 존재하는 cranial tibial muscle을 분리함.
>- Tibial tuberosity에 구멍을 낸 후, tuberosity를 다른 조직과 분리하여 가쪽으로 견인함. 
>- 구멍을 뚫은 곳에 pin을 박아 고정하고, 반대쪽 cortex에도 핀이 살짝 겹칠 수 있게 함.
>- Bender를 이용해 철사를 구부린 후, cutter로 잘라줌.
>- 드릴로 구멍을 뚫고, tension band wire를 장착함. 
>- 수술 후 관절낭과 fascia lata를 봉합. 장력 봉합이 필요하므로 horizontal mattress suture.
>- 피하조직을 닫고, 가쪽에서부터 mattress로 봉합. 피부까지 봉합.


#### 3. Cranial Cruciate Ligament Rupture

: 전십자인대 파열 (진단 검사, 치료 옵션), **LFS**/**TPLO**/TWO/TTA/TTO
##### Test
- Cranial drawer movement (LFS 성공 시 양성에서 음성)
	- distal femur: 한 손으로 고정. 검지는 슬개골을 정상 위치로 넣은 상태로 고정
	- tibia: 다른 손의 엄지로 proximal tibia 뒤쪽을, 검지로 경골능을 잡고 앞쪽으로 전진시킴
- Tibia compression test (TPLO 성공 시 양성에서 음성)
	- Stifle joint 편 상태에서 한 손은 경골능, 다른 손은 발을 구부림 ; 전방 십자인대 끊어지면 구부렸을때 proximal tibia가 앞으로 밀림
##### Surgical Treatment

1 - Intracapsular reconstruction techniques

- The most commonly used material is autogenous fascia lata
- An advantage of intracapsular technique is that they most closely mimic the original position and biology of the original CCL.
- Disadvantages of these techniques are their invasiveness and the tendency of the graft to stretch or fail.

2 - Extracapsular reconstruction techniques

>[!tldr] Lateral Fabellar Suture (LFS)
>끊어진 전방십자인대 제거 후 관절낭을 닫고, 관절낭 밖에 인공 인대를 다는 방법.



>[!tldr] Capsular Fascial Imbrication
>관절낭을 슬개골 인대 아럐쪽에 묶고, fascia를 슬개골 인대에 묶음.



3 - Corrective osteotomy

>[!tldr] Tibial Plateau Leveling Osteotomy (TPLO)
>- 수술 전 TPA(Tibial Plateau Angle) / Saw radius / Plate size / Magnitude of rotation
>- 수술 전후 TPA 측정 시 5-7도가 정상.
>- 톱의 직경은 tibial width의 25% 이내
>- prefer TPLO for treating larger, active dogs in which long term rehabilitation and postoperative control are difficult
