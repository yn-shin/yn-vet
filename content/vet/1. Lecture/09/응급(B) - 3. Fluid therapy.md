> 수축기 혈압 감소, 체온 낮음, 약간 tachycardic. ⇒ **Shock의 초기 비보상 단계**
>- 계속 토했으니 > hypovolemic shock
>- SIRS 2/4 > distributic shock

- 수액 치료는 resuscitation의 기본.
- 내원한 환자의 90-95%는 수액부터 꽂고 시작. (but 50% 이상이 부적절한 조치)

>[!warning] Common mistakes
>Too much하게 줄 때가 많다! 
> - 수액이 필요 없는데 주는 경우
> - 너무 많은 수액을 넣는 경우
> - 부적절한 유형을 주는 경우
> - 불필요하게 길게 주는 경우
> - (혹은 필요한데 주지 않는 경우, 충분한 수액을 주지 않는 경우)
## 1. Fluid therapy = Drug therapy

- 용량에 따른 효과 차이도 크고, 처방(prescription)이 필요함.
- 수액을 주는 근거(reason)이 있어야 함
- 환자에 따른 개체 차이 고려해야 함

>[!check] 수액을 왜 줘야 할까? (Reason for giving)
>- **Fluid deficit**을 채워주는 것. (dehydration, hypovolemic)
>- **Continuous loss**를 메워주는 것. (excessive loss, maintenance; 유지용량을 메우지 못할 때)
>- 약물을 수액에 타서 **Vehicle**로서 사용 (bolous, CRI) - 과하게 수액 들어갈 수 있음을 고려. (other medication/electrolytes)

But 현실적으로 수액을 주는 흔한 이유..?
- 무엇을 해야 할지 몰라서, 교수님이 시켜서, 내 마음의 안정..
- 열을 내리기 위해 : 부작용이 더 클 수 있음
- 보호자의 요청

>[!quote]- 필요한 수액일까? 불필요한 수액일까?<br>[1] 잘 먹고 잘 마시는 설사 환자<br>[2] 3일 구토, 치료받지 않은 환자<br>[3] 마취에서 회복 중인 환자<br>[4] HBC로 인한 출혈이 있는 환자<br>[5] 열이 있는 stable 환자<br>[6] Renal azotemia 있는 환자 <br>[7] 중독 물질을 섭취한 환자
>1) 필요 X | 구강을 통한 수분 공급 충분
>2) 필요할 가능성 ↑ | 탈수, hypovolemic 상태 확인.
>3) 필요 X | 건강한 환자라면 괜찮다, 상태 안 좋은 환자면 필요할 수 있음.
>4) 필요 O | 일반 수액 처치든, 수혈이든 필요
>5) 필요 X | vital 문제 없으면 굳이 필요하지 않음
>6) **Prerenal azotemia면 필요** | prerenal은 신장으로 가는 혈류량이 부족해 발생하는 급성 신부전, 수액 필요(치료는 아님). 배출이 잘 안 되면 fluid overload 될 수 있으니 주의.
>7) 종류에 따라 다름 | 신장으로 배출되는 독소면 필요할 수도 있지만, 아니면 크게 의미 없을 수도.


#### ✳️ Fluid distribution (60:40:20 rule)

- 우리 몸의 **60%** = 물
	- 물(60) 중 **40%** = ICF (세포내액) : 세포는 가장 큰 물탱크 (<u>혈액 쪽으로 계속 shifting</u>)
	- 나머지 **20%** = ECF (세포외액)
		- ECF의 75%는 interstital fluid, 25%는 plasma


**수액을 어디로 줘야 할까? (Where is the deficit?)**

- 지속적인 구토 설사로 **탈수**가 심하다면 → 계속 혈액 쪽으로 shifting이 일어나 세포 내부 fluid 부족 ⇒ **세포** 채워야
- 교통사고로 심한 **출혈** → 탈수 없음, 빠른 속도로 혈액이 빠진 것, absolute hypovolemia ⇒ **혈액** 채워야

|              Resuscitation              |                            Rehydration                             | Maintenance |
| :-------------------------------------: | :----------------------------------------------------------------: | :---------: |
| **Hypovolemia** 교정<br>: 혈관내 (혈액) target | **Dehydration** 교정 (Extravascular fluid의 재배치)<br>: 세포내 (간질) target | 정상 수화 상태 유지 |
|               \*쇼크 증상 확인                |                             \*탈수 정도 평가                             |             |

## 2. Types of Fluids (수액 종류 결정)
>[!check] 4Ds : 수액 쓸 때 고려할 4가지
>- Drug
>- Dose
>- Duration
>- De-escalation

#### ✳️ Fluid의 종류

수액의 종류는 크게 2가지(+혈액)로 나눌 수 있다!

- **crystalloid** : 혈관막을 쉽게 이동함 ⇒ <u>간질 보충</u>
- **colloid** : 분자량이 커서 혈관을 통과하지 못함 → 혈관 내 삼투압을 높여 물을 끌어옴 ⇒ <u>혈액량 보충</u> (수액의 절대량 + 끌어온 물 = 용량 뻥튀기)
- Blood transfusion : 적혈구를 넣어줘서 빈혈 및 출혈 개선

| Crystalloid                                                                                                                                  |
| -------------------------------------------------------------------------------------------------------------------------------------------- |
| water, sodium, glucose base +@                                                                                                               |
| 분자량이 작음. <font color="#d83931">세포막을 왔다갔다 자유롭게 움직임</font>. (fluid shift)<br>빠르게 redistribution 됨. (30분 안에 75%가 간질이나 세포로 재분배.) → **탈수 교정**에 유용 |
| 장기부종 유발 가능 (특히 폐부종)                                                                                                                          |
| **Hypotonic** (0.45%) / **Isotonic** (0.9%) / **Hypertonic** (3-7%)                                                                          |

| Colloid                                                                                                                                                                                                                        |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 혈관 안에 머물며 Intravascular <font color="#d83931">삼투를 유지 또는 늘림</font>.<br>**Volume expansion** 효과가 아주 큼. (주변에서 끌어오는 양까지, 투여량보다 2-3배 정도)<br>Intravascular 안에서 하루까지 머무름 (**long-lasting**)                                           |
| 📉 **LFVR** (Limited Fluid Volume Resuscitation) : Small volume으로 최적의 resuscitation<br>(원래는 80-90ml/kg를 maximum으로 보고 1/4씩 4번까지도 줌. → Expansion 효과로 인해 혈액 및 응고 인자 희석, 혈관층 무너짐, 산증 등 부작용. ⇒ 이젠 <u>Maximum 30ml/kg</u>을 권장량으로 봄.) |
| First choice는 아님. 합병증이 발생하면 치명적 (AKI, coagulopathy응고병증)                                                                                                                                                                        |
| Crystalloid를 먼저 쓰되, 계속 빠지거나 합병증(조직부종)이 의심될 때 Colloid를 add-on.                                                                                                                                                                  |

> - Ex. 10% 탈수 + hypovolemic 환자가 있다면,
> 	- 등장성 crystalloid를 먼저 쫙 → 세포 수화 될 때쯤 maintenance 용으로 저장성 crystalloid → 균형이 잡히면 colloid 
> 	- 간질/세포에 fluid가 다 빠진 상태에서 colloid를 먼저 쓰면 → (물을 계속 끌어오니까) 탈수 악화 → 폐, 뇌에도 영향.
> 	- Colloid는 탈수를 다 교정한 상태에서!


- **Hypovolemia w. dehydration** (ex. 12% 탈수 + BP 60) | <u>Isotonic crystalloid</u>
	- 둘 다 target, 일단 등장성 crystalloid로 충분히 resuscitation
- **Hypovolemia w/o dehydration** (ex. 외상 & 심한 출혈) | crystalloid (1-2 aliquots) + <u>Colloid</u>
	- 혈관 안을 target하되 (colloid)
	- 간질이나 세포도 충분히 수화될 수 있도록 crystalloid 조금 써줘도 좋음 (10ml 1번 정도)
- **Hypovolemia w. cerebral, pulmonary damage** | <u>LFVR</u>로 수액 용량 조절
	- 폐, 뇌는 부종에 가장 취약, 조금만 overload 걸려도 부종 발생
	- 폐, 뇌 문제가 있으면 혈관 투과성도 증가 → fluid 많이 빠져나감
	- Aggressive하게 쓰면 절대 안 됨! LFVR 적용.
