> 수축기 혈압 감소, 체온 낮음, 약간 tachycardic. ⇒ **Shock의 초기 비보상 단계**
>- 계속 토했으니 > hypovolemic shock
>- SIRS 2/4 > distributic shock

- 수액 치료는 resuscitation의 기본.
- 내원한 환자의 90-95%는 수액부터 꽂고 시작. (but 50% 이상이 부적절한 조치)

>[!warning] Common mistakes
>Too much하게 줄 때가 많다! 
> - 수액이 필요 없는데 주거나, 너무 많은 수액을 넣거나
> - 부적절한 유형을 주거나, 불필요하게 길게 주거나
> 
> (혹은 필요한데 주지 않는 경우, 충분한 수액을 주지 않는 경우)
## 1. Fluid therapy = Drug therapy

- 용량에 따른 효과 차이도 크고, 처방(prescription)이 필요함.
- 수액을 주는 근거(reason)이 있어야 함
- 환자에 따른 개체 차이 고려해야 함

>[!check] 수액을 왜 줘야 할까? (Reason for giving)
>- **Fluid deficit**을 채워주는 것. (dehydration, hypovolemic)
>- **Continuous loss**를 메워주는 것. (excessive loss, maintenance; 유지용량을 메우지 못할 때)
>- 약물을 수액에 타서 **Vehicle**로서 사용 (bolous, CRI) - 과하게 수액 들어갈 수 있음을 고려. (other medication/electrolytes)

But 현실적으로 수액을 주는 흔한 이유..?
- 무엇을 해야 할지 몰라서, 교수님이 시켜서, 내 마음의 안정..
- 열을 내리기 위해 : 부작용이 더 클 수 있음
- 보호자의 요청

>[!quote]- 필요한 수액일까? 불필요한 수액일까?<br>[1] 잘 먹고 잘 마시는 설사 환자<br>[2] 3일 구토, 치료받지 않은 환자<br>[3] 마취에서 회복 중인 환자<br>[4] HBC로 인한 출혈이 있는 환자<br>[5] 열이 있는 stable 환자<br>[6] Renal azotemia 있는 환자 <br>[7] 중독 물질을 섭취한 환자
>1) 필요 X | 구강을 통한 수분 공급 충분
>2) 필요할 가능성 ↑ | 탈수, hypovolemic 상태 확인.
>3) 필요 X | 건강한 환자라면 괜찮다, 상태 안 좋은 환자면 필요할 수 있음.
>4) 필요 O | 일반 수액 처치든, 수혈이든 필요
>5) 필요 X | vital 문제 없으면 굳이 필요하지 않음
>6) **Prerenal azotemia면 필요** | prerenal은 신장으로 가는 혈류량이 부족해 발생하는 급성 신부전, 수액 필요(치료는 아님). 배출이 잘 안 되면 fluid overload 될 수 있으니 주의.
>7) 종류에 따라 다름 | 신장으로 배출되는 독소면 필요할 수도 있지만, 아니면 크게 의미 없을 수도.


#### ✳️ Fluid distribution (60:40:20 rule)

- 우리 몸의 **60%** = 물
	- 물(60) 중 **40%** = ICF (세포내액) : 세포는 가장 큰 물탱크 (<u>혈액 쪽으로 계속 shifting</u>)
	- 나머지 **20%** = ECF (세포외액)
		- ECF의 75%는 interstital fluid, 25%는 plasma


**수액을 어디로 줘야 할까? (Where is the deficit?)**

- 지속적인 구토 설사로 **탈수**가 심하다면 → 계속 혈액 쪽으로 shifting이 일어나 세포 내부 fluid 부족 ⇒ **세포** 채워야
- 교통사고로 심한 **출혈** → 탈수 없음, 빠른 속도로 혈액이 빠진 것, absolute hypovolemia ⇒ **혈액** 채워야

|              Resuscitation              |                              Rehydration                              | Maintenance |
| :-------------------------------------: | :-------------------------------------------------------------------: | :---------: |
| **Hypovolemia** 교정<br>: 혈관내 (혈액) target | **Dehydration** 교정<br>(Extravascular fluid의 재배치)<br>: 세포내 (간질) target | 정상 수화 상태 유지 |
|               \*쇼크 증상 확인                |                              \*탈수 정도 평가                               |             |

## 2. Types of Fluids (수액 종류 결정)
>[!check] 4Ds : 수액 쓸 때 고려할 4가지
>Drug / Dose / Duration / De-escalation

#### ✳️ Fluid의 종류

수액의 종류는 크게 2가지(+혈액)로 나눌 수 있다!

- **crystalloid** : 혈관막을 쉽게 이동함 ⇒ <u>간질 보충</u>
- **colloid** : 분자량이 커서 혈관을 통과하지 못함 → 혈관 내 삼투압을 높여 물을 끌어옴 ⇒ <u>혈액량 보충</u> (수액의 절대량 + 끌어온 물 = 용량 뻥튀기)
- Blood transfusion : 적혈구를 넣어줘서 빈혈 및 출혈 개선

| water, sodium, glucose base +@                                                                                                               |
| 분자량이 작음. <font color="#d83931">세포막을 왔다갔다 자유롭게 움직임</font>. (fluid shift)<br>빠르게 redistribution 됨. (30분 안에 75%가 간질이나 세포로 재분배.) → **탈수 교정**에 유용 |
| 장기부종 유발 가능 (특히 폐부종)                                                                                                                          |

##### 수액 선택 - Hypovolemia? Dehydration?

>[!quote] Ex. 10% 탈수 + hypovolemic 환자가 있다면,
> - **등장성 crystalloid**를 먼저 쫙 → 세포 수화 될 때쯤 maintenance 용으로 **저장성 crystalloid** → 균형이 잡히면 **colloid** 
> - 만약 간질/세포에 fluid가 다 빠진 상태에서 colloid를 먼저 쓰면,
> 	- (물을 계속 끌어오니까) 탈수 악화
> 	- 폐, 뇌에도 영향
> ⇒ Colloid는 탈수를 다 교정한 상태에서!

| Hypovolemia w/ dehydration (ex. 12% 탈수 + BP 60)       |
| ----------------------------------------------------- |
| <u>Isotonic crystalloid</u>                           |
| : 둘 다 target하되, 일단 등장성 crystalloid로 충분히 resuscitation |


| Hypovolemia w/o dehydration (ex. 외상 & 심한 출혈)                                              |
| ----------------------------------------------------------------------------------------- |
| crystalloid (1-2 aliquots) + <u>Colloid</u>                                               |
| : 혈관 안을 target하되 (colloid)<br>: 간질이나 세포도 충분히 수화될 수 있도록 crystalloid 조금 써줘도 좋음 (10ml 1번 정도) |


| Hypovolemia w/ cerebral, pulmonary damage                                                                                    |
| ---------------------------------------------------------------------------------------------------------------------------- |
| <u>LFVR</u>로 수액 용량 조절                                                                                                        |
| : 폐, 뇌는 부종에 가장 취약, 조금만 overload 걸려도 부종 발생<br>: 폐, 뇌 문제가 있으면 혈관 투과성도 증가 → fluid 많이 빠져나감<br>: Aggressive하게 쓰면 절대 안 됨! LFVR 적용. |

#### 1️⃣ Types of Crystalloid

혈장의 osmolality = 290-310mOsm/L (dog) 정도.

|                       Isotonic (등장성)                        |                     Hypotonic (저장성)                      |                         Hypertonic (고장성)                          |
| :---------------------------------------------------------: | :------------------------------------------------------: | :---------------------------------------------------------------: |
| 혈관내 30분 머물고 간질로 퍼짐 <br>: **Resuscitation** + Maintenance 목적 | 혈관에 거의 머물지 않고<br>다 세포 수준으로 빠짐<br>: 완전 **Maintenance** 목적 | 혈관 내 삼투압을 일시적으로 증가시켜<br>주변 fluid를 끌어옴 <br>: **Resuscitation** 효과적 |
|                     혈장 삼투와 비슷 (290 전후)                      |                    혈장 삼투의 절반 정도 (150)                    |                          엄청나게 높음 (2500)                           |


##### Isotonic

등장성 수액은 혈관내에 30분 머물고 간질로 퍼짐. ⇒ **Resuscitation** + Maintenance 목적으로 사용됨.
<br>\* 혈장의 pH는 7.3-7.4 정도의 중성, CPA 발생 시 pH 6.7 정도.<br>\* Balanced solution = Plasma-Lyte, Hartman, Normosol 등은 혈장 pH와 비슷하게 중성인 수액.
   : 중환자 의학에서 선호됨. 중탄산을 만들어내서 추가적으로 <u>알칼리화</u> 효과.<br>\* Normal saline은 사실 pH 5.0으로 강산성이고, 체내를 <u>산성화</u>시킴. (산증 환자에서 위험)
   

- Lactated Ringer's solution (= Hartman) : 중성에 가까움 (pH6.5)
- Normosol : 중성에 가까움 (pH6.4)
- <font color="#d83931">Plasma-Lyte</font> A : 가장 중성에 가까움, 중환자에서 가장 많이 씀.
- <font color="#4f81bd">Normal Saline</font> (0.9% NaCl) : Chloride가 많아 강산성 (pH5.0) (*사실 normal 아님* )
	- 혈장 전해질 구성과 다름 & 산증인 환자에서 쓰면 안 됨 & AKI 유발 가능
	- [indication] 강산성이니까 ⇒ metabolic alkalosis, hypochloremia 일 때 / potassium, calcium 없으니까 ⇒ hypercalcemia, w. citrate blood products 일 때


##### Hypotonic

저장성 수액은 혈관 내에 거의 머물지 않고, 세포 수준으로 쫙 빠짐. ⇒ 완전 **Maintenance** 목적


- 0.45% NaCl : sodium 수치 낮음 = fluid overload 덜함
	- [use] Hypernatremia 치료 / CHF, renal disease 환자 (sodium load의 부담 있는 환자)
	- [avoid] Hyponatremia 환자 /  [[SIADH|SIADH]], positive-pressure ventilation, RAAS 항진 시 (sodium 이미 낮은 환자)
- D5W



##### Hypertonic

고장성 수액은 혈관 안에 머무르지는 않지만, 혈관내 삼투압을 일시적으로 증가시켜 주변 fluid를 끌어옴. ⇒ **Resuscitation**에 효과적. 
<br>\* Colloid와 효과는 비슷하지만, 전해질 base인 만큼 지속 시간이 30분 정도로 짧음.<br>   ⇒ 병용하여 쓰기도 함. (hypertonic + colloid = <u>Turbo starch</u>)<br>\* LFVR로 써야 함 (용량이 많이 뻥튀기되니까)

- 7.5% NaCl 등



※ Potassium 부족할 시 KCl 앰플을 수액에 타서 추가 공급한다. Hyperkalemia를 유발하지 않도록 maximum 속도보다 아래로 주입.

#### 2️⃣ Types of Colloid

- Colloid는 교질 농도를 높여 혈관 밖의 fluid를 안쪽으로 쫙 끌어당긴다. 
- 삼투압의 유지에는 albumin이 80% + RBC 또한 중요한 역할을 한다.


##### Natural

= blood product. 많은 양이 필요함에도 불구하고, 합성 제제보다 안정성이 높아 최근 권장되는 추세.

- Whole blood (전혈)
- Plasma
- Albumin :
	- CSA(Canine serum albumin)보다 HSA(Human ~)가 구하기 쉽고, 충분히 효과적.
	- hypersensitivity 주의 (용량 2g/kg/day로 제한)

##### Synthetic

- Dextran
- Gelatin 
- **HES** (Hydroxyethyl starch) : 전분을 정제, 원래 많이 썼으나 단점이 부각되고 있음. (6%, 130달톤)
	- 삼투를 유지해서 volume resuscitation 가능 → 그 외 알부민 기능은 대체 불가.
	- 충분히 수화(rehydration)시킨 뒤 사용 !!
	- 20ml/kg/day 초과 X → 부작용 AKI, coagulopathy, volume overload, allergic reaction

#### ✳️ Fluid responsiveness
- Fluid responsive : cardiac output, stroke volume 충분히 증가함
- Fluid Non-responders : 둘 다 증가하지 않고 오히려 lactate만 계속 오름. ⇒ 보다 근원적인 원인 교정 필요!
	- SIRS, sepsis, 급성 전신염증 ⇒ 승압제 고려
	- Hypoalbuminemia ⇒ 혈압이 올랐다가도 계속 떨어짐. 알부민 수혈 고려

>[!note] 확인 방법
>임상적으로 **혈압 측정** 이 제일. (반응성 좋으면 혈압이 상승)
>##### Non-responsive 지표
>- low BP 지속
>- lactate 증가
>- persistent LV filling, No collapse in vena cava (정수압 충분)
>- VTI < 10-15%, PPV < 11-16%
>
>⇒ 승압제 사용 고려(distributive shock), 교질 삼투압 교정

## 4. Maintenance & Monitoring

#### ✳️ 유지용량 공식

**Maintenance + Dehydration + Ongoing loss** 

- (1) Maintenance : 자연적으로 내 몸에서 빠지는 수분량 ⇒ 보통 2.5mL/kg/h (60ml/kg/day)
	- sensible loss : 계산 가능, 배뇨량, 건강한 개체에서는 1.5ml/kg 정도.
	- insensible loss : 호흡이나 땀 등 계산 불가, 대략 1ml/kg 정도로 잡음
	- 즉, <u>urine output + 1</u> 로 계산
- (2) Dehydration : BW (kg) × % dehydration × 1000 (ml/day)
- (3) Ongoing loss : 구토, 설사, 출혈 등 (정확한 출혈은 어려움)

(1) + (2) + (3) 의 용량을 6-24시간 동안 교정하되, **Maintenace의 2배를 넘지 않는다**. 

#### ✳️ Fluid creep (변수)

수액 처치뿐만 아니라 delivery를 위해 함께 들어가는 용량이 무시 못할 만큼 크다는 것. 
(주사제, 혈액 제제, 무언가를 희석해서 넣어주는 것 등)

- 이런 creep을 알고 있을 것.
- 약물을 적게 희석하고, 가능한 DW에 희석할 것.

#### ✳️ Fluid overload 평가 & 조치 & 예방

**Preventing Fluid Overload!** : 과한 fluid로 문제될 때가 90% 이상. 예방을 잘하자!

- BW & Urine ouput을 지속적으로 모니터링 & 재평가해서 fluid가 과하게 들어가지 않는지 확인해야 함.
	- 체외로 배출되지 못한 fluid는 간질에 쌓여 부종을 유발할 수 있기 때문.
	- 간질에 쌓이면 체중이 가장 먼저 증가함.

##### Indicator
- **BW 증가** (> 5~10%) : 가장 중요한 indicator
	- 체중이 5-10% 이상 증가하면 바로.. ⇒ 1) 소변 유도 (방광 비우기), 2) 수액 용량 확인
- **Hypertension** : 정수압 상승 → 가장 먼저 혈압 증가
- <u>resting respiratory rate</u> 증가
- Pulmonary edema : 가장 먼저 나타나는 편
- Pitting edema : 눌렀을 때 자국이 오래감
- Chemosis(결막 부종), nasal dischage, HR 증가, shivering

1% overload 될 때 사망률은 8% 증가함!

- 그 외 초음파에서의 sign
	- B lines (폐실질에 액체가 쌓여 나타나는 hyperechoic line; 3-4개 이상)
	- LA:Ao ratio 증가
	- Pleural effusion, Pericardial effusion
	- Non-collapsible vena cava

##### Therapy
- 수액 바로 중단!
- Diuretics ([[furosemide|furosemide]]) : 간질 부종, 폐부종에서 엄청난 효과 (0.2-0.5mg/kg, up to 1mg/kg)
	- 단, 충분히 수화되지 않은 상태에서 쓰면 신장 관류량이 감소해 AKI 가능성 ↑ 
- Increase mobility : 계속 움직이게 해서 관류를 증가시킴
- 흉수, 복수 제거

##### Prevention
애초에 잘 예방하는 게 훨씬 좋다!!
- low-volume strategy (LFVR)
- Fluid creep을 정확하게 계산하진 못하더라도, 항상 인지하고 주의하기!
- 체중 모니터링을 꾸준히, 자주 (q6-12h)


>[!quote]- 30kg, 6y, 래브라도 리트리버 / consistent vomiting & anorexia / HR 112, RR 30, T 39.1, SBP 135, mental-alert, 7% dehydration ⇒ Fluid therapy plan?
>- Perfusion 모두 정상이므로 굳이 resuscitation이 필요하진 않다.
>- 쇼크로 진행되지는 않은 상태지만 탈수와 구토에 대한 교정이 필요하다. ⇒ Fliud loss가 많을 테니, 세포/간질로 빠지는 **crystalloid** 수액 사용.
>- 탈수 교정이 안 된 상태에서 colloid 사용하면 안 됨.
>- 수액 용량?
>	- (1) Maintenance : 30 x 2.5 = 75ml/h
>	- (2) Dehydration : 0.07 x 30 x 1000 = 2100ml/day (87ml/h)
>	- (3) Ongoing loss : 100ml/day (4.1ml/h) 
>- ⇒ 다 더해주면 166.1ml/h 인데, 유지용량의 2배를 넘지 않도록 150ml/h만 준다.

>[!quote]- 4.7kg, 2y, toy poodle / HBC / HR 148, RR 32, T 37.1, SBP 90, Normal CRT, Pale MMC, Mental depressed, no dehydration, severe skin bruising ⇒ Fluid therapy plan?
>- 바이탈을 보고 초기 비보상 쇼크 단계인 걸 알아야 한다! (심박 증가, 혈압 감소, 다른 지표들)
>- Resuscitation? Yes!
>	- Plasmalyte 4.7 x 10ml bolus IV for 10min, 2 times (94ml)
>	- Turbostarch 4ml bolus IV for 10min
>	- Vasopressor (Norepi+vasopressiN) : 염증이 심하거나 SIRS 가능한 상황으로 승압제 고려
>- Stabilization
>	- Maintenance 2.5 x 4.7 = 11.75 ml/hr
>	- Transfusion 필요한지 결정

>[!quote]- 11.3kg, 7y, 웰시코기 / Falling / HR 112, RR 25, T 37.1, SBP 80, weak pulse, delayed CRT, pale MMC, mental - stuporous to comatose, ascultation - pulmonary crackles, no dehydration ⇒ Fluid therapy plan?

>	- Turbostarch 4ml bolus IV for 10min
