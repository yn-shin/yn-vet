# Noninfectious inflammatory encephalitis, meningitis (1) : SRMA

스테로이드 반응성 수막염-동맥염 : [[SRMA|SRMA]]<br>⇒ 예후 엄청 좋음.

# 1. Etiology

- SRMA = aseptic meningitis(세균성 아님), steroid-responsive suppurative meningitis, necrotizing vasculitis, juvenile polyarteritis(어린 연령에서 호발), Beagle pain syndrome(비글에서 호발)
- 수막의 혈관염 → 척수와 뇌줄기에 광범위한 병변을 유발
- Juvenile~young adults (6-18개월) 에서 호발 > middle, older ages에도 생김.
- Large-breed dog에서 더 흔하다.^[그래서 교재에서는 SRMA가 더 호발한다 나오지만, 소형견>대형견인 국내에서는 SRMA < GME, MUE인 느낌.]
- 소인 : Beagles, Bernese Mountain dogs, Boxers 등

# 2. Clinical Features

- <u>Fever</u> (50-90%), reluctance to move, <u>neck pain</u>, vertebral pain : [[IMPA|IMPA]], [[Polymyositis, PM|다발성 근염(polymyositis)]]과 유사
- Neurologic examination은 보통 정상
- 초기에는 wax and wane (좋아졌다 안 좋아졌다) 함
- 보호자가 보기에 "갖다 대주지 않으면 먹거나 마시기 힘들어해요 / 고개 숙였다가 드는 걸 어려워해요"
- 활발하고 어린 비글이 갑자기 우울해지는데 특별한 신경증상은 없다 ⇒ SRMA 의심

# 3. Diagnosis

- Spinal fluid analysis
	- Neutrophilic pleocytosis (>100cells/μL, 75% 이상이 neutrophils)
		- 건강한 척수액에서는 세포 거의 안 나옴. 진짜 많은 것.
	- Increased protein concentration
- CSF는 <u>스테로이드 투약 전 반드시 sampling</u>
- IMPA와 병발할 수 있음

# 4. Treatment and Prognosis

| Treatment       |                      |
| --------------- | -------------------- |
| Glucocorticoids | 면역억제 용량 → tapering   |
| Azathioprine    | PDS만으로 반응이 없으면 추가 가능 |


- 영구적 손상 시 치료가 어려우니 빠르게 치료 들어가야 함!!
- 빨리 치료된다면 재발 가능성 낮음
- 예후는 좋음 (60-80%는 재발 없이 완치)
- 고령+품종 소인 → 예후 나쁠 수 있음
