- 국내 유병률은 FeLV > FIV

## 1. Etiology & Epidemiology

>- family *Retroviridae*
>	- subfamily *Oncovirinae* → *Orthoretrovirinae*
>		- genus *Gammaretrovirus*
>\* 종양 유발 능력 때문에 과거 Oncovirinae로 분류되었으나, 현재는 Orthoretrovirinae로 분류.

- *Retroviridae* - *gammaretrovirus*에 속하는 single-strand RNA virus 이다.
- 구조는 <u>core</u> proteins + <u>envelope</u> proteins 등으로 구성.
    - Core protein **p27** : 감염 세포, 혈액, 침, 눈물 등에서 검출 → 항원검사의 표적.
    - Envelope protein **gp70** : subgroup에 따라 감염력, 병원성, 질병 다름. (subgrup A/B/C)
	    - 일부 고양이에서 gp70 노출 이후 중화항체 생성.
    - Envelope protein **p15e** : 면역억제 유도.

---

- 바이러스는 **역전사효소(reverse transcriptase)** 를 생성, 감염된 세포의 세포질에서 RNA → DNA(provirus)로 전환시킴.
	- 이 provirus는 숙주 세포의 유전체에 삽입되어, 세포 분열 시 새로운 바이러스 입자의 주형이 됨.
	- 숙주세포의 세포막에서 budding 방식으로 전파됨.
- 처음에는 **oropharynx** 에서 복제 → 이후 전신으로 확산되어 **골수** (WBCs), 상피조직 (침샘, 눈물샘) 등 감염.

- 그루밍이나 식기 공유를 통해 <u>감염 고양이의 침, 비강 분비물과의 장기간 접촉하는 것</u>이 주된 감염 경로.
    - 태반, 수유, 성적 접촉 통한 전파 / 환경을 통한 (feces, urine, fomite, aerosol) 전파 : 흔하지 않다.
- Distibution : worldwide / 실외 생활, 1-6세 수컷 고양이에서 흔히 감염.

PCR과 ELISA 결과에 의해 다음과 같이 분류할 수 있다.

| 4 classes of FeLV infection |                                                                                                     |
| --------------------------- | --------------------------------------------------------------------------------------------------- |
| **Abortive**                | 바이러스 완전 제거, 불현성                                                                                     |
| **Progressive**             | 임상 질환으로 진행, 지속적인 viremia<br>- 암컷에서 더 흔함 (수컷은 abortive)<br>- FeLV-B에서 바이러스 load가 높음                  |
| **Regressive**              | 항원 음성, 일시적 PCR 양성<br>- 바이러스 DNA가 최대 12년까지 검출될 수 있기 때문<br>- regressive한 고양이의 혈액이 naïve 고양이를 감염시킬 있음. |
| **Focal**                   | 일시적 항원 양성, 지속적인 PCR 양성<br>- glucocorticoids나 면역 억제제에 의해 일시적으로 활성화될 수 있음.                            |

- 항원 양성이라도 임상증상이 없을 수 있으며, 불현성 감염도 존재

---

- FeLV에 의해 아래와 같이 다양한 syndrome이 유발될 수 있다.
	1. 림프종(**Lymphoma**) : oncogenes 활성화, lymphoid precursor에 provirus 삽입
	2. 재생불량성 빈혈(**Aplastic anemia**, subtype C) : TNF-α 분비증가
	3. 면역결핍(**Immunodeficiency**) : T-림프구(CD4, CD8)의 감소 또는 기능장애
	4. 호중구 감소증(**Neutropenia**)
	5. 호중구 기능장애(**Neutrophil function disorders**)
	6. 악성 변형(**Malignant transformation**) : 정상 세포가 악성 세포로 전환
	7. 골수증식성 질환(**Myeloproliferative diseases**) : 골수 성장 촉진 물질 → 증식성 골수 병변

- 다른 retroviruses (FIV, foamy virus 등) 와 co-infection 될 수 있다.
	- FIV 단독 감염 생존 기간 > FeLV 또는 FeLV+FIV 동시 감염 생존 기간
	- FFV의 proviral load가 높으면 FeLV, FeLV-B, FCoV의 proviral load도 높아지는 경향
	- male에서 feline gammaherpesvirus 동시 감염이 더 흔함

# 2. Clinical Features

- FeLV 감염 환자는 주로 식욕부진, 체중감소, 우울 등 <u>비특이적 증상</u>으로 내원하는 경우가 많다.

|                                   |                                                                                                                                                                                                                                                                                                                                 |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Specific clinical syndrome**    | (1) 바이러스 자체의 직접적 영향, (2) 면역억제에 의한 2차 감염 (**기회감염**) 에 의해 나타남.<br>- 구내염, 구토, 설사, 황달, 비염, 폐렴 등 다양하게 나타날 수 있음.<br>- 임상 증상만으로 FeLV 감염을 진단하기 어려움 → 감별진단 필요                                                                                                                                                                            |
|                                   | [Opporotunistic infections와 그 원인]<br>- Bacterial or calicivirus-induced stomatitis : 면역 억제<br>- Vomiting, diarrhea : 장염/면역억제로 인한 2차 감염 등 <br>- Icterus<br>    - Prehepatic: FeLV에 의한 면역매개 적혈구 파괴 또는 2차 감염(M. haemofelis 등)<br>    - Hepatic: 간 림프종, 지방간, 국소 간 괴사<br>    - Posthepatic: 장 림프종<br>- Rhinitis or pneumonia : 2차 감염 |
| **Neoplasms**                     | FeLV에 의해 직접적으로 / 또는 FeLV로 인한 면역억제에 2차적으로 발생<br>- Lymphoma : mediastinal, multicentric, alimentary 림프종이 흔함.<br>- Leukemia : erythroleukemia(적혈구 전구세포 증식), acute myelomonocytic leukemia<br>- Fibrosarcomas : feline sarcoma virus와 동시 감염 시 드물게 발생<br>림프종이 specific하게 FeLV 감염을 시사하지 않음 !                                          |
| **Renal failure**<br>(신부전)        | Renal lymphoma, glomerulonephritis에 의해 발생 가능<br>- PU/PD, 체중 감소, 식욕 부진                                                                                                                                                                                                                                                           |
| **Urinary incontinence**<br>(요실금) | 괄약근 기능부전, 배뇨근 과민반응 등에 의해 발생<br>- 방광 용량이 적고, 야간에 요실금 (가장 흔한 형태)                                                                                                                                                                                                                                                                  |
| **Ocular lymphoma**                   | Miosis(동공축소), blepharospasm(눈꺼풀 경련), cloudy eyes 유발<br>- 전방 삼출, 종괴, 각막 ㅊ미착, 수정체 탈구, 녹내장 등이 관찰됨<br>- lymphoma 없이 단독으로 포도막염(uveitis)을 일으키지는 않음.                                                                                                                                                                                   |
| **Neurologic Abnormalities**          | 다발성 신경병증(polyneuropathy), 림프종의 결과로 발생<br>: 동공부동(anisocoria), 운동실조, 약화, 사지마비, 행동 변화, 요실금 등                                                                                                                                                                                                                                       |
| **임신한 고양이**                           | - 유산(abortion), 사산(stillbirth), 불임(infertility)<br>- 자궁에 감염된 경우 진행이 빠르고, 새끼가 사망할 수 있음 (kitten mortality complex)                                                                                                                                                                                                                |
| **Lameness, weakness**                | - Immune complex 침착에 의한 화농성 비감염성 polyarthritis에 의해<br>- Multiple cartilaginous exotoses에 의해 (뼈 겉면에 비정상적으로 뼈조직 증식)                                                                                                                                                                                                               |

*너무 복잡하고.. 다양하고.. 비특이적이다.*
- <u>종양</u>(특히 림프종, 백혈병)과 <u>혈액/골수 질환</u>이 흔하고, 다양한 장기 침범
- 면역억제에 의한 2차 감염 → 구내염, 호흡기/소화기 감염, 황달 등
- 신장, 신경, 안구, 생식기 이상도 동반
