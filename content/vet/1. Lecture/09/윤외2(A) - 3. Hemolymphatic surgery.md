# 1. Surgery of Lymphatic System

그렇게 복잡하고 정교한 수술은 아니지만, 진료를 쉽게 하고 진단을 명확하게 하는 데 매우 중요한 파트.
간과하기 쉽고, 안 해봤기 때문에 잘 안 하려고 함.
진료의 심도나 정확도를 향상시킬 수 있는(데 안 하는) 파트.

#### Definitions
- **Lymphadenectomy** : lymph node의 전체를 절제.
- **Lymphangiomas** (benigns) vs. **Lymphangiosarcomas** (malignant)
- **Lymphedema** : interstitial space에 fluid가 저류됨. Drainage 하기는 어려운 형태.

| Lymphangiomas (림프관종)                                                             | Lymphangiosarcomas (림프관 육종)                                 |
| -------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| 말초 림프계(peripheral lymphatics)에 나타난 양성 종양성 변화                                     | 말초 림프계에 나타난 악성 종양성 변화                                       |
| Fluid가 안에 들어서, 파동감이 있는(fluctuant) cystic 구조                                      | Metastasis : 전이가 진행된 것으로 판단<br>→ 완치 어려움, 종종 안락사 (but 완치되기도) |
| 치료 :<br>- complete excision 또는<br>- marsupialization(조대낭술; cystic 구조에 구멍을 크게 뚫음) | - complete excision<br>- 적절한 항암제 선택 → 치료 기대해볼 만             |

#### Lymph node 관련된 오해

>[!question] 1 - Lymph node의 size가 크면 병적인 변화, size가 정상이면 문제 없음?

- LN size가 질병과 반드시 상관 관계가 있다고 보기는 어려움.
	- lymph node의 size가 정상임에도 조직 검사에서 종양 세포가 발견되기도 하고,
	- enlarge 되어 있어서 검사해보면 antigenic reaction (항원 반응) 정도로만 나오기도.
- **분명히 연관은 있지만, 사이즈가 100% 대변할 거라고 생각하면 안 됨**.

>[!question] 2 - Lymph node가 enlarge되어 있으면 다 biopsy ?

- Mandibular, mesentric LN 등 구강이나 GI 쪽은 항상 antigen에 노출이 많기 때문에 reactive hyperplasia의 가능성이 높음. 
- Mandibular LN이 비대되어 있다고 바로 biopsy를 고려하지 X.
- **여러 림프절이 비대되어 있다면 굳이 구강 쪽 LN을 biopsy하기보다는, 다른 곳을 우선적으로**.

#### Surgical Technique


|                    FNA                    |                  Trucut                   |                         Incisional biopsy                          |
| :---------------------------------------: | :---------------------------------------: | :----------------------------------------------------------------: |
| ![[Pasted image 20250416110815.png\|200]] | ![[Pasted image 20250416110825.png\|200]] | ![[Pasted image 20250416110837.png\|250]]<br>(horizontal mattress) |

- 림프절 이상이 발견되었다면 조직 생검용 채취 필요
- **FNA** : 18-25 gauge needle 사용, 음압에 의해 빨려나오는 조직 이용
	- 조직 검사가 아니라 cytology. 진단률은 50% 정도
- **Trucut biopsy** : 14-16 gauge Trucut needle 사용
	- 작은 림프절에 적용하기는 쉽지 않음. 충분히 큰 림프절에 대해 적용.
	- 병변의 core 쪽 생검이 가능하다는 장점이 있음.
	- 진단률 90% 이상
- **Incisional biopsy** : Wedge로 파서 조직의 양을 충분히 확보, 조직 검사뿐 아니라 cytology, culture 등 다양하게 진단 검사
	- Trucut으로도 양이 충분하지 않을 때 이용
	- 림프절 생검의 목적은 (1) <u>진단용</u>, (2) 전이된 종양을 제거하는 <u>치료</u>의 효과. 치료적 목적을 고려하여 **Excisional biopsy** (전체를 다 떼어내는 것)이 권장됨. 하지만 림프절의 size가 너무 크거나, 바로 옆에 major 혈관/신경이 지나가서 고려하여 전제거를 시도하기 위험이 있을 때, incisional biopsy 시행
- Lymphadenectomy (=Excisonal biopsy) : 둘러싼 capsule을 잘 분리해서 들어냄.
	- Bacterial/fungal cultures, histopathology, cytology를 위한 sampling 가능.
	- 혈관이 지나가는 것에 주의. Afferent, Efferent 혈관 다 확인해서 결찰/소락해야 함.
	- 작은 혈관은 electrocautery로도 지혈(대부분), 간혹 큰 혈관은 봉합사를 이용해 ligation.
	- LN의 size가 크면 dead space가 생김 → 봉합하는 것을 권장.
# 2. Surgery of Spleen

### Definitions
- **Splenomegaly** : 어떤 이유에서든 비장이 커져 있는 상태 (focal or diffuse)
- **Splenectomy** : 비장을 제거하는 것
- **Splenosis** : 정상 비장 조직이 어떤 이유로 떨어져 나가, 떨어진 자리에서 다시 자란 것 (선천적/외상성/수술 과정 중) (\*난소 등에서도 발생 가능)
	- 병적인 변화 아님, 정상적으로 발생할 수 있는 상황
- **Siderotic plaques** : 철분, calcium 등이 비장 가장자리에 침착되어 갈색 또는 녹슨 색(rust-colored)으로 보임
	- 병적 변화보다는 노령성 변화로 봄
- **Splenorrhaphy** : Trauma 등으로 찢어진 비장을 봉합해주는 것, 흔한 경우는 아님. (\*-rrhaphy : 덮어주고, 봉합해주는 것. Hernia를 봉합하면 Herniorrhaphy라고 함.)

>[!note] Anatomy of Spleen
>- Capsule(피막)과 Parenchyma(실질)
>- Capsule : elastic smooth muscle fiber로 구성 → 탄력이 아주 좋음, 두꺼운 고무 풍선처럼 잘 늘어남, 크기 차이가 엄청
>- Parenchyma
>	- White pulp : 면역 관련, lymphoid tissue로 구성
>	- Red pulp : 혈액 관련, venous sinuses
