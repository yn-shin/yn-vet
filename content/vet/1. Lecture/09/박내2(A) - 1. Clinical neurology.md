### ✔️ 부위에 따른 다양한 cerebrovascular disease

#### Cerebrovascular disease

- 원인 : <u><b>Embolism</b></u> (혈전; major cause), thrombosis, hemorrhage
	- 혈액을 끈적끈적하게 하는 모든 factor가 유발 가능하다.
	- **D-dimer** : 색전증의 biomarker (고양이에서는 확립되지 않음.)
- 뇌혈관 질환의 특징
	- Peracute or **acute** onset(빠르게 나타나는 편)
	- focal, **asymmetrical** (한쪽에서 나타남)
	- **non-progressive** brain dysfunction (문제가 된 한쪽 혈관/신경 부위에서만 나타나고, 다른 부위로 진행되진 않음)
- 개에서는 사람에서만큼 흔하지 않음(uncommon). - 지금까지 진단 기술이나 데이터가 부족해서 잡힌 게 적음

#### Cerebellar infarction

- Cerebellar infarction(소뇌 경색) → 혈액이 막혀 뒤쪽에 산소가 공급되지 않으면서 허혈성 괴사(ischemic necrosis)가 생김.
- 소뇌에 문제 → balance의 문제 → **vestibular sign** (균형감각의 이상; <u>nystagmus, head tilt, tetraparesis, circling, growling, falling, ataxia</u> 등)
	- 균형이 안 맞으니 <u>vomiting</u>도 나타남. - 소화기 질병만 생각하면 안됨. 중이염, 외이염 등에 의해서도 가능.
	- vestibular nucleus(전정핵, 안뜰핵) : 연수 부분(medulla oblongata)에 있음. 8번 신경(vestibulocochlear n.)의 핵. 이쪽에 염증 있으면 증상 나타남.
	- flocculonodular lobe, cochlear, interna - 세 부위와 관련. (대뇌가 아닌 뇌줄기, 내이 부분)

>[!note]- Flocculonodular lobe | 소뇌(Cerebellum)의 위치와 구조
>![[Pasted image 20250326132559.png|400]]
>- 소뇌의 엽은 다음과 같이 Anterior lobe, Posterior lobe, Flocculonodular lobe 3가지
>	- **Flocculonodular lobe** : 아래에 연수가 붙어 있어 소뇌와 연수를 연결시키는 부위.
>	- Flocculonodular lobe에 문제가 있으면 vestibular sign이 나타날 수 있음.
 
#### Spinal cord infarction

- Spinal cord infarction : FCE(연골섬유성 물질, disc로부터 들어오는 것으로 생각됨)이 색전증을 일으킴. → 편측성으로 다리를 쓰지 못함.

<br>

- 이외에 부위에 따른 Cerebellar infarction의 임상 증상
	- 급성 사지마비, 버둥거림, But 의식에는 문제 없음 → 소뇌 증상
	- 경추마비, neck scratching → COMS 증상과 같이 나타남
	- 망막, 각막, 홍채 등에 아무 문제 없는 환자가 갑자기 시력이 사라짐 → **central blindness**, 대뇌이상

>- 단두종에서 많이 생기는 후두공이형성증. (소형견이 많은 우리나라에서 흔한 편)

### ✔️ Medical therapy

##### 초기 치료 (2주차)

- [[prednisolone|prednisolone]] : 스테로이드, 권장 용량보다 낮게 !! (0.5mg)
	- CSF를 생성하는 choroid plexus의 CSF 분비를 억제

	- N-acetylcystein : 객담 배출, 점안, 항산화 기능
	- pentoxiphylline : 동맥 확장
	- Vitamin E : 항산화제. Ischemic necrosis → 지방 산화 → 염증이 발생하는 과정을 막음.

### 📋 Case 3. Head trauma (`T`)

> ##### 환자 병력 확인

>[!tldr] Signalment
>- 4y, IF, Maltese, 3kg
>- <u>Falling from chair</u>
>- <u>Stupor</u>(멍함), Recumbency + Depression ⇒ 균형을 못 잡음
>- Anisocoria(양쪽 동공 크기 다름) ← <u>mydriasis</u> (L>R; 산동)
>- Head turn to right
>- Left horizontal nystagmus (왼쪽 눈알이 수평으로 빠르게 움직임)
>- <u>Delayed PLR on left eye</u> : 축동(myosis)이 되지 않음

- PLR 검사 ⇒ CN2, CN3 기능 확인 가능. 이 환자는 왼쪽 축동이 안 되는 거니 CN3 손상을 의심.
- Midbrain(CN. Ⅲ), forebrain(CN. Ⅱ), diencephalon, .. 간뇌/중뇌/연수 등 뇌줄기의 전체적인 손상<br>⇒ multifocal한 문제. 예후가 매우 좋지 않음.

한 번 떨어진 것으로는 이런 문제가 생기지 않음. 지속적인 trauma - 학대의 징표.

\* diencephalon(사이뇌) + hemisphere(대뇌반구) + = forebrain. Optic nerve가 여기서 나와서 망막으로 연결됨. 

| 관련 뇌신경                                                       |                        |
| ------------------------------------------------------------ | ---------------------- |
| **CN2. Optic nerve** (시각신경) : sensory, 뇌로 빛 정보를 전달함          | 간뇌 (diencephalon)      |
| **CN3. Oculomotor nerve** (눈돌림신경) : motor (등/배/안쪽곧은근, 배쪽빗근)  | 중뇌 (midbrain)          |

| **CN6. Abducens nerve** (갓돌림신경) : motor, 눈알을 밖으로 당김 (가쪽곧은근)  | 연수 (medulla oblongata) |

>[!check] 축동이 되지 않는 이유?
>- 전기 신호을 전달하는 경로 : optic nerve(Ⅱ) - retina(망막) - pretectal area - oculomotor nerve(Ⅲ)로 전달됨.
>	- retina에서 빛에너지가 전기에너지로 탈분극되면서 정보 전달.
>	- Pretectal area로 진행되면서 Ⅲ. 눈돌림신경으로 전달되어 축동.
>- **Ⅱ. Optic n.** : sensory n.의 기능 / **Ⅲ. Oculomotor n.** : motor n.의 기능
>- 축동이 일어나지 않았다 = 1) <u>망막</u>: retina detachment나 hemolysis, 2) <u>시각신경(Ⅱ)</u>: optic neuritis(시신경염) 3) <u>Pretectal area</u>의 문제: (염증, 종양, 수두증 등) 4) <u>눈돌림신경(Ⅲ)</u>: oculomotor n. 의 문제
>- Midbrain에 문제가 생기면 축동이 일어나지 않는다.

- 이뇨제(furosemide, 1mg/kg, SC) : 뇌압 감소
- 스테로이드(PDS, 1mg/kg BID, PO) : 염증 완화 
- Mannitol (1g/kg, IV for 30m) : 뇌압을 떨어뜨림, 30분 동안 천천히 투여
- PLR 검사 계속 해서 산동 확인 → 뇌 기능이 정상으로 돌아오는지 확인
- NS 계속 투여 : 탈수 치료
- Cushing reflex 검사 계속 (PLR test를 통해 간접적으로 확인)
	- 뇌압이 높아지면 midbrain을 압박해서 PLR response가 없어진다. (중뇌 - Ⅲ이 압박 받아 조리개 X)
	- 학대, trauma → PLR 검사에서 산동/축동 있으면 뇌신경 이상일 가능성 ↑ → cushing reflex 확인.
	- 이에 맞춰 산소 포화도 증가, CO2 감소,, heart rate 조절, ROS 감소 등의 치료 방향 설정



>[!note] Cushing reflex
>- <u>bradycardia, decreased cerebral blood flow, increased ICP</u>
>- 뇌의 ischemia(산소 포화도 저하)가 일어나면 모두 Cushing reflex 가능.
>	- 그 원인으로는 색전증, 종양, ICP 증가 등 다양함.

### ✔️ Types of Tremor

균형 감각과 관련, 대부분 소뇌의 이상.
- Resting tremor : 그냥 가만히 있어도 몸을 떠는것
- **Intention tremor** : 원하는 대상 앞에서 몸을 떠는 것
- Task-specific tremor : 특정 활동할 때 몸을 떠는 것
- Action or kinetic tremor : 수의적 운동 중에 떨리는 것
- Postural tremor : 특정 자세에서 한쪽이 떨리는 것


###  📋 Case 4. CDS (`D`)

>[!note] CDS (Cognitive Dysfunction syndrom, 인지기능장애 증후군)
> - 신경퇴행성 질환(neurodegenerative disorder). DJD (degenerative joint disease) 등이 해당함.
> - Brain pathology, 인지 기능 감소
> - <u>7살 이상</u>(인간 40세)에서 인지 기능 저하. 11세 이상에서 현저하게 발생.
> - 8살 이상의 개의 14%에서 CDS 발병
> - **CDS의 기전**
> 	- 뇌는 특성상 산화적 손상에 취약.
> 		- oxidative metabolism이 많이 있는 곳, 산화적 손상화(ROS)가 많이 발생 가능. 
> 		- lipid가 많음.
> 		- 다치면 regeneration이 안됨.
> 	-  <u>원인</u> ⇒ Brainstem과는 무관, forebrain(주로 대뇌)의 문제 - Forebrain의 aging process에 따라 발생
> 		- limbus, hippocampus에서 특히 많이 생김 → Oxidative damage, Amyloid 축적, 유전적 요인에 의해.
> 	- <u>결과</u> ⇒ 학습과 기억 능력 저하, abnormal behaviors → cognitive dysfunction

Intracranial meningioma : 가드리늄에 의해 T1 이미지에서 조영증강됨. dural tail sign
