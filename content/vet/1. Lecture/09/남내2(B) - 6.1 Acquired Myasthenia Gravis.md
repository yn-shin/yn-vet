# Uncommon Immune-Mediated Disease

# 1. Etiology

- AChR (Acetylcholine receptor)의 기능 이상으로 생기는 신경근 질병
- Acquired MG의 기전은 3가지로 설명

| Ab directed against AChR                                                                        | Receptor internalization                    | Interfere with the interaction |
| ----------------------------------------------------------------------------------------------- | ------------------------------------------- | ------------------------------ |
| AChR에 IgG가 와서 달라붙고,<br>보체가 와서 붙으면서 MAC 발생<br>(Membrane Attack Complex)<br>막이 공격당하면서 AChR 표면적 감소 | 자가항체에 의해<br>AchR이 세포막 안으로 파고들어<br>기능을 하지 못함 | 자가항체가 AChR을 직접 막아<br>기능을 방해    |

# 2. Clinical Features

[Dog]

- **Generalized weakness** (most common)
- **Focal MG**로 인한 regurgitation ⇒ dysphagia, dysphonia, cranial nerve dysfunction (안면신경 반사가 떨어지는) 등 나타남
- <u>Acute fulminating form</u>
	- severe weakness, megaesophagus
	- 합병증으로 aspiration pneumoniae 주의
- Hypothyroidism, hypoadrenocorticism 병발

[Cat]

- 덜 흔함
- generalized weakness <u>without megaesophagus</u>
- cranial mediastinal mass (thymoma) 로 인한 증상

# 3. Diagnosis

-  Acquired MG의 진단은 AChR에 대한 serum **autoantibody** 측정으로 이루어짐 : sensitive, specific
	- 위양성 드묾 (MG 환자의 2%에서만 seronegative)
	- 스테로이드(면역 억제 용량) : 항체가를 낮춰 검사가 부정확해질 수 있음
- Congenital MG는 antibody test 음성
- Antibody test못 한다면 → **short-acting anticholinesterase drug** ([[edrophonium|edrophonium]] chloride test)
	- 약 주사 즉시 dramatic하게 나음 (지속 시간은 짧음)
- 병발 가능한 질환에 대한 additional testing 필요 (hypothyroidism, Addison's dz, thymoma, megaesophagus)

# 4. Treatment

| Treatment                            |                                                                                                                     |
| ------------------------------------ | ------------------------------------------------------------------------------------------------------------------- |
| **Acetylcholinesterase inhibitors**  | - Pyridostigmine : 경구 투약<br>- Neostigmine : IM 주사 (국내에 X)                                                           |
| **Glucocorticoids** (위에 걸로 충분하지 않으면) | - 근육 위축의 부작용 있음<br>- 오연성 폐렴, 당뇨, 위궤양 환자에서는 주의<br>- PDS 2mg/kg/day PO (면역억제 용량) 에서 시작<br>→ 최소 용량에서 멈춤 (0.5mg/kg EOD) |
| Adjunctive immunosuppressive drugs   | azathioprine, cyclosporine, leflunomide, MMF                                                                        |
| Thymectomy (흉선절제술)                   | - MG associated with thymoma 환자에서 고려됨.<br>- 예후는 좋은 편                                                                |
| Spontaneous remission                | 개에서 저절로 관해되는 경우도 있음<br>(예후 good)                                                                                    |
