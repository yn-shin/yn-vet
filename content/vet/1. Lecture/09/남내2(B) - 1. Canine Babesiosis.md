- 우리나라는 주로 *B gibsoni*
- 2022년 *B. canis* 의 국내 최초 보고가 있었다.

# 1. Etiology & Epidemiology

- RBC에 기생하는 원생동물이다.
- Anemia 관련 증상 일으킨다. (혈소판 감소, secondary IMHA, ITP 등)
- 전세계적으로 분포하며(worldwide distribution), species가 다양하다.


![[Pasted image 20250612231118.png|700]]


- 고양이도 걸리기는 하지만, 드물다. (Babesiosis in cats : less common)
	- 집냥이에서는 특히 볼 일이 잘 없다.
	- 우리나라 양주에서 50마리 정도의 길냥이를 대상으로 검사해보니 15% 정도 걸린 고양이가 있긴 했다..
	- 수혈을 통해 전파될 수 있으므로, 수혈 시 donor의 혈액 검사가 필요하다! (screening test)


![[Pasted image 20250612231337.png|800]]



- *B. canis*, *B. vogeli*, *B. gibsoni* (pathogenic strains) 에 감염된 경우,
	- 며칠~몇 주의 incubation period 존재
	- Degree of parasitemia(혈액 속 기생충이 존재하는 정도)는 organism에 따라 다양

---

- 기생충은 RBC 내에서 복제되어 **intravascular or extravascular hemolytic anemia** 를 유발한다. (⇒ IMHA와 감별 필요)
	- 기생충에 대한 면역 반응(immune-mediated reactions)이나, 항원 반응(altered self-antigens)이 용혈성 빈혈을 악화 → [[남내1(A) - 02. 면역매개질병의 진단적 검사#2️⃣ COOMBS test = DAT (Direct Antiglobulin test) -✅IMHA|Coombs test]] 양성을 유발할 수 있다.
	- Macrophage를 자극한다 → <u>fever, hepatosplenomegaly</u>
	- 적혈구를 빠르게 파괴시킨다 → 산소를 잘 운반하지 못하므로 <u>severe hypoxia</u>, 2차적 장기 손상을 유발할 수 있다.
	- 급성 감염된 일부 개에서는 <u>DIC</u>로 인해 혈소판 감소

---

- 심각도는 다음 2가지의 영향을 받는다: 1) *Babesia* 의 species&strain, 2) host's immune status
	- Chronic, subclinical도 흔하다. (증상이 없어 진단이 안 되기도 / 적극적 치료 안 해도 됨)
	- Glucocorticoids, splenectomy 는 비장 종양 등 다른 문제가 동반되었을 때 지시된다. (단일 감염 시 권장되지 않음)
	- *Bartonella* spp. 와 coinfection → 더 심각 (국내에는 잘 없음)

# 2. Clinical Features

미국에서 준임상형(subclinical infection)이 흔함.

| Clinical forms            |                                                                                                                                                                                                              |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Peracute**<br>**Acute** | Anemia & fever 을 유발한다.<br>빈혈 ⇒ pale MMC / tachycardia, tachypnea / depression, anorexia, weakness (비특이적)<br>용혈성 빈혈 ⇒ 황달, petechiae, hepatosplenomegaly (심한 경우 촉진으로도 느껴질 수 있음)<br>: 감염의 단계, DIC의 동반 정도에 따라 다름 |
| <u>acute</u>              | severe anemia, DIC, metabolic acidosis, renal disease<br>South Africa에서의 *B. rossi* 감염이 가장 심각하다.<br>심각한 경우 tissue hypoxemia도 나타난다 : 산소 공급이 중요                                                                |
|                           | Acute babesiosis는 다음과 감별진단 필요 : Primary [[남내1(A) - 04. IMHA\|IMHA]] & Primary [[ITP|ITP]] (PCR 검사)                                                                                                               |
| **Chronic**               | Weight loss, anorexia                                                                                                                                                                                        |
| **Atypical**              | 혈액에는 없는데 복수/심낭수에서 DNA가 검출되기도 함.<br>전형적인 Babesiosis 증상이 없더라도 체액에서 검출되는 경우가 있음.                                                                                                                                |

# 3. Diagnosis

빈혈이 있다 > Blood smear > CBC상의 이상이 실제로 빈혈인지 / 봉입체가 발견되는지 확인!

- **Spherocytic regenerative anemia, thrombocytopenia**
- (비특이적) hyperbilirubinemia, bilirubinuria, hemoglobinuria, metabolic acidosis, azotemia, polyclonal gammopathy, proteinuria, cylindruria
- RBC의 organism 확인! (but 염색상 관찰되지 않을 수 있음.)
	- 모세혈관에서 혈액이 조금 더 정체돼서 잘 보일 수 있음.

- 임상적으로 정상이나 항체 반응 양성인 경우가 많음. > 현재 감염이 아닐 수도 있으니,
	- 항체 검사만으로 확진하기는 어려움.

#### PCR assays

- 미국에서 주로 진행. *B. vogeli* 와 *B. gibsoni* 를 찾아주지만, 종 구분은 불가.
- 위음성 : peracute disease, concurrent immunosuppression, 질병 초기 항원 양이 적거나, 검사 기준의 차이로 인해
- 위양성(seropositive) : 질병에 걸렸다가 치료되어 몸에 항원이 남아있는 경우 ⇒ 항원 자체로는 확정 진단 불가.

[국내 PCR assays 환경]

- *Babesia* 를 감별하는 데 가장 좋음 - RBC에 기생, 전혈 샘플 필요.
	- 혈검 위해 뽑을 때 PCR용도 꼭 남겨놓자 (전혈 1ml)
- 장점) 간단하고, 사람 손을 덜 타는 편.
- 주의점) 스테로이드 치료 시작 시 → 검사 결과가 가려질 수 있음.


- Subclinical carriers가 있는 경우, 양성 결과가 꼭 임상 증상을 나타내지는 않음.
- 하지만 증상이 있을 때의 검사는 임상적 의의가 있음.


[고양이] 

- Acute phase proteins : serum amyloid A (고양이 ↑), haptoglobin, paraoxonase-1
	- 굉장히 소수의 동물에서, subclinical Babesia carrier와 임상 증상이 나타난 동물에서 유의미한 차이가 있음.
	- serum haptoglobin 증가 & paraoxonase-1 감소

# 4. Treatment

2가지로 접근.

- Supportive care : 수혈, 중탄산염 치료, 수액 치료
- Drugs : atovaquone (다른 약물은 중요x)

# 5. Zoonotic Aspects and Prevention

- 개, 고양이 → 사람으로 전파된 케이스는 없지만, 주의는 필요함!
	- *B. microti* 는 사람에게 감염 가능
- Minimal cross-protection : 한 번 걸렸어도 다른 species에 걸릴 가능성 있음
- 면역억제제, 비장절제술 : 과거 감염되었던 개에서는 위험
- 개물림사고 : 조심해라
- 백신? : 있는 나라도 있지만, 미국과 우리나라에는 없음
- Blood donor programs : 꼭 PCR screening을 거치고 나서 공혈
