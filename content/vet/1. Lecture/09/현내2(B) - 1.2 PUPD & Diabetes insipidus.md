> - **Anuria** (무뇨) : 오줌이 제대로 생성되지 않거나, 배출되지 않음

> - **Polyuria** (다뇨) : 비정상적으로 많은 urine 생성 또는 배출 / 주로 상부 비뇨계 문제
> - **Dysuria** (배뇨통) : 배뇨 곤란, 통증
> - **Stranguia** (통증배뇨) : 느리고 고통스러운 배뇨
> 	- 요도, 방광의 근육 경련(spasms) 등에 의함
> - **Pollakiuria** (빈뇨) : 소량씩 자주 배뇨 / 주로 하부 요로계 문제
> - **Urinary incontinence** (요실금) : 비자발적 배뇨 (통제 불가)
> - **Nocturia** (야뇨) : 밤에 배뇨하고 싶어짐, 노령 동물에서 주로 발생
> 	- 다뇨나 요실금의 초기 증상일 수 있음
> - **Hematuria** (혈뇨)
> - **Pyuria** (농뇨)
> - **Malodorous** urine (악취 나는 소변)

## Water homeostasis

- <u>in</u>gestion : **drinking**, food
- <u>ex</u>cretion : **urine**, the respiratory tract, feces
	- ADH 증가 → 소변을 몸 안으로 흡수 → 배뇨량 감소

#### ADH

ADH에 문제가 생긴 경우가 많다.
뇌하수체 후엽을 통해 분비되는 호르몬.
소변에서 빨아들여서 농축된 소변을 유발.
혈관 수축, 이상 행동, 뇌하수체의 전엽에 영향

## Diuresis

- 요붕증 :

- 대부분 용질이뇨
	- 소변 안에 커다란 물질들이 많아서 
	- AQP-2, ADH는 제대로 작용하지만, 용질이 커서 막 흘러가니까 정상적으로 물을 흡수할 시간이 없음
	- Diabetes mellitus : 
	- mannitol
- Water diuresis
	- 물이 많이 들어오면 용질은 적어짐, 묽어짐, ADH 분비 감소, AQP이 제대로 작용하지 않음
	- 물을 빨아들이는 메커니즘이 없어 물이 계속 빠져나감

### Primary polydipsia

- 매우 드문 경우.
- Psychogenic polydipsia (PP) : 뇌, 정신의 병적인 문제로 강박적으로 물을 많이 마시는 것.
- 갈증 중추의 손상, 이론적으로 가능하나 극히 드물다.

### Primary polyuria (with compensatory polydipsia)

- 다뇨가 먼저 생겨 보상성으로 다음.
- 수분이뇨(ADH 자체에 문제)인지, 용질이뇨인지 구분 필요.
- 용질이뇨
	- Diabetes mellitus (당뇨)
	- 신장질환
	- Post-obstructive diuresis (요로가 막혀 며칠 배뇨를 못했다? → 문제가 해소되어도, 배뇨 기능을 회복하기까지 시간이 걸리는데 그동안 용질이뇨가 발생 가능.)
	- Primary renal glycosuria (소변으로 포도당이 질질 새어나감)

# Diabetes insipidus (DI, 요붕증)

- 당뇨가 더 많이 발생 → 당뇨를 그냥 diabetes라고 하기도.
- DI = 아무 맛이 없는 소변이 흐른다
- DM = 달콤한 소변이 흐른다

### 구분

- CDI
	- Complete
	- Partial
	- idiopathic 많음
	- trauma, infection 
	- 뇌에 cyst나 종양이 있는 경우(rare)
	- 
- NDI : 신장 때문
	- Primary : 선천적으로 ADH의 작용을 못 받아들이는 경우, 극히 드물다
	- **Secondary** (가장 많음) : 다른 질환으로 인해 2차적으로 ADH를 받아들이는 신장 기능이 떨어짐

>[!check] Diseases associated with secondary NDI
>- 신장 : Renal failure, Renal medullary solute washeout (삼투압 차가 없어서 그냥 용질이 빠져나가는 현상)
>- 갑상선 : 
>- 간 : Hepatic disease

### History & Clinical signs

- No age, breed or sex predilection

### Definitive Diagnostics

##### Random plasam osmolality

물을 많이 먹으니까 > 
overlapping이 있어서

CDI vs NDI 감별 위해 바소프레신, 수분제한검사

##### Synthetic vasopressin therapy

합성 vasopressin 투약 → 항이뇨 작용이 일어나는지 확인
- **CDI** : ADH 작용 X → vasopressin을 넣어주면 항이뇨 작용이 정상적으로 나타남 → <u>소변량 감소, 다음 해소</u>
- **NDI** : 정상적으로 나오는 호르몬의 작용이 안 되는 것 → <u>뚜렷한 개선 없음</u>
	- 아주 약간 올라가는 차이는 나타남

- 분무 또는 점적
- Regulary washout 때문에 개선 효과가 바로 나타나지 않고, 일주일 정도 걸림

##### Modified water deprivation test (MWDT, 수분 제한 검사)

Vasopressin 투약보다 더 명확
탈수를 세게 유발 → ADH 호르몬의 작용 평가

하지만 이미 요붕증이 있는데.. 탈수 심화, 오줌은 계속 나가니까 나트륨이 쌓임, 
진단적 이점만 있지, 장점은 없고 risk가 있음
탈수 와서 급사 할 수 있고 발작 할 수 있습니다..? 실제로 시행할 때가 많진 않은.

### Treatment

- 치료 방법은 딱히 없지만
- 그나마 항이뇨 호르몬 투여

|               |                                                           |
| ------------- | --------------------------------------------------------- |
| **CDI**           | - Vasopressin 투여<br>- Low-sodium diet (저염식이)              |
| **Primary NDI**   | 치료반응이 좋은 편은..                                             |
| **Secondary NDI** | 기저질환 치료 잘                                                 |
| **Primary PD**    | - 천천히 물을 제한<br>but 광적으로 집착하기에..<br>환경을 바꿔주고, 다른 즐거움을 주는 듯 |

>[!note] DDAVP
>- 합성 바소프레신, natural보다 3배 정도 뛰어난,
>- efficacy는 SC에서 가장 좋은데, 반감기가 짧아서 하루 1번 주사 필요. (반복투약의 단점)
>- 경구, 결막낭으로 투여할 때가 많음.
>- 너무 많으면 → hyponatremia, water intoxication

##### Chlorpropamide

- 혈당을 낮춰주는 약물 
- 요붕증 환자에서도 도움이 됨. ADH 항진
- partial에서 선택적으로 쓸 수

##### Thiazide diuretics ([[약리1(B) - Ch.17 이뇨제|약리1(B) - Ch.17 이뇨제]])

원위세뇨관에서 물의 재흡수를 촉진
Na, K도 같이 나감
보상작용으로 PT에서 물을 빨아들이는 효과, Na도 같이 빨려들어감; K은 딸려가지 X

Thiazide 이뇨제는 요붕증 환자에서 좀 효과를 보이지만, 다른 계열 이뇨제는 X
