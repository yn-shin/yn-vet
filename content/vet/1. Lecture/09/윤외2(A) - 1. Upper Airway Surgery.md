##### Congenital Abnormalities
- Stenotic nares : 비공협착. 콧구멍이 꽉 막혀, 개구호흡밖에 못함.
- **Elongated soft palate** : 연구개 노장 (길어지는 것 + 비후되는 것)
	- 선천적이지만, 시간이 지날수록 악화될 수밖에 없음.
	- 
- **Hypoplastic trachea** : 태어날 때부터 기관이 저형성됨. (단두종 증후군에 포함될지 말지는 계속 논란 중)
	- dorsal band가 없이 태어나는 것. 아주 딱딱하고 기관이 좁음. (사람은 원래 없음)
##### Acquired Abnormalities
- **Everted laryngeal saccules** : 후두낭 외번 [= **후두허탈 1단계**]
	- 콧구멍이 작아지고 음압이 생기면서 후천적으로 발생.
	- 
- **Laryngeal collapse**
	- 쐐기돌기가 무너져서 서로 붙고, 연구개가 노장되어 있음. corniculate은 보이지도 않음.
	  ![[Pasted image 20250307094439.png|200]]

### 마취 시 주의사항
- 조금만 근육이 이완되어도 위험할 수 있는 부위.
- Induction과 recovery 과정에서 특별히 위험
	- induction 전 처치하면서도 호흡이 멈춰버릴 수 있음.
	- extubation 후 아직 근육이 이완되어 있어서 여전히 호흡 통로가 좁아진 채로 사망할 수 있음.
- 수술 전 5분 이상 충분히 산소를 준다.
- intubation은 빠르게, extubation은 느리게. (빼는 속도 X 빼는 시기)

## Congenital (1) - Stenotic nares


|                  Normal                   |                 Stenotic                  |
| :---------------------------------------: | :---------------------------------------: |
| ![[Pasted image 20250314090721.png\|200]] | ![[Pasted image 20250314090905.png\|200]] |

- Alar fold의 연골이 내측으로 무너지는 현상. → 치료하려면 무너진 연골의 회복이 필요.
좁은 통로를 지나가며 음압이 생김 → 주변 영향에 미치는 압력이 커짐.
- "*악의 축*".. 여러 증상들의 근본 원인이 되는 현상.
#### Digansis
- 임상 증상
	- Open mouth breathing 코가 좁아지니 입으로 숨을 쉼.
	- Coughing, gagging 인후두 쪽으로 기침, 토하는 현상
	- Snoring 코골이 현상
	- Hyperthermia 호흡할 때 애를 많이 씀 → 체온이 올라감. 
	- Exercise intolerance 산소 포화도가 떨어져서 운동하면 쉽게 지침.
	- Cyanosis 청색증
	- Restless sleeping 잠을 자도 잔 것 같지 않음.
- Imaging
	- 심장이 커지는 형상 ⇒ 폐수종 발생
	- 오연되기 쉬움 → Aspiration pneumonia
	- 배에 가스가 차는 Stomach bloating
#### Medical Care
- 약물로 근본 원인을 해결할 수는 없지만,
- 수술로는 50%밖에 안 됨. - <u><font color="#ff0000">체중 감량</font></u>이 매우 중요함.
- 운동 제한

- Severe : 일단 처치 
	- 진정제 → 노력성 호흡을 완화 ([[butorphanol|butorphanol]], [[diazepam|diazepam]])
	- **Steroid** 반드시 필요. Edematous한 상태를 완화시키는 데 탁월하므로.
	- 산소 공급, cooling(체온 감소)

#### Surgical Treatment
- ⭐ **Complete Upper Airway Examination First** ⭐
	- <u>비공협착, 연구개노장, 기관저형성 / 후두낭 외번, 후두허탈</u> : 5가지를 통으로 떠올리고, 전체 검사 필요
- 가능한 3-4개월령에 하는 게 권장됨
	- 시간이 지날수록 2차적인 문제들이 심해짐. 다리를 안 쓰고, 심장이 커지고,,
	- 산소포화도가 적은 상태가 지속적으로 유지되면 심혈관계에도 문제가 됨. → 필연적으로 심장병이 심하게 옴.
	- 마취를 견디기에도 좋은 나이.
- Wedge resections
	- **Vertical** : 무너진 연골을 많이 잘라내기 위함. 목적성은 가장 맞음.
	- Horizontal : 끝부분 soft tissue만 살짝 잘라냄.
	- Dorsolateral : 끝부분을 자르고 피부와 봉합하여 넓힘.
- Laser amputation of rebundant alar fold : 레이저로 구멍을 뻥 뚫어줄 수 있으나, 충분히 숙련되어야 함.

#### Postoperative care & assessment
- 기관 튜브의 제거는 가능한 늦게
- 호흡하기 편한 자세로 - sternal recumbency
- Upper airway edema를 줄이기 위해 **corticosteroid** 반드시
- 하루 정도는 금식
- Elizabeth collar : 발로 긁으면 큰일남. 넥카라 중요.
	- 혀는 control 불가. 봉합사가 풀리고 피가 날 수 있으나, 그런 출혈은 일주일면 사라질 것. (연골 제거가 가장 큰 목적)

#### Prognosis
- 연구개 노장, 후두낭 외번, 후두허탈 등이 없으면 예후가 좋으나.. 연구개 노장은 선천적인 부분
- 후두허탈이 없으면 = 병이 심화되지 않음 = 예후가 매우 좋음.

## Congenital (2) - Elongated soft palate
![[Pasted image 20250409003024.png|400]]
- 연구개는 정상적으로 epiglottis를 살짝 덮고 있음.
- **들숨**할 때 훅 빨려들어가면서 막음. → 염증, 부종
- Aspiration(오연) 유발 가능.
- 길이가 정상일 때는 숨을 내쉴 때 빠져나가는데, 길이가 너무 길면 내쉴 때 오히려 nasopharynx를 막음.
#### Diagnosis
- 입을 벌려 눈으로 확인
- Clinical signs
	- Respiratory distress during inspiration 흡기 때 호흡 어려움. 
	- Gagging 토하려고 함
	- Stertor 호흡음 (소리는 같으나, 잘 때 소리가 나는 것을 snoring이라고 함.)
	- Hyperthermia 호흡이 잘 안 되면 반드시 체온이 올라감
	- Exercise intolerance, cyanosis, restless sleeping
- Imaging : 심장 비대, 폐수종, 폐렴

#### Surgical treatment
- Medical care는 동일.
- Young(4-24개월)에 수술 필요. 
	- 마취를 견딜 수 있는 가장 빠른 나이, 4개월.
	- 너무 어릴 때 잘랐다가 다른 기관만 성장하면, 연구개가 오히려 짧아질 수 있음. (⇒ 오연성 폐렴)
	- 수의사의 판단이 매우 중요함. 가능한 빨리 수술하는 것이 좋지만, 다른 곳의 병의 정도가 괜찮을 때는 조금 보류할 수 있음.
	- 어린 나이에 수술하게 되면, 생각했던 것보다는 길게 자르기. 증상은 많이 개선될 것이고, 나중에 한번 더 수술해도 괜찮음.
- **후두 쪽 퇴행성 변화 나타나기 전** 수술 해주는 게 좋음.
- Steroid로 부종 완화- 수술 때문에도 부종 생길 수 있으므로.
- 얼마나 자를까? Endotracheal tube를 삽관하면 잘 보이지 않음. → 자를 위치를 미리 표시해두어야 함.
- 수술하면 출혈 있음. Tube 주변에 거즈 장착.

#### Techniques

- (1) Crush technique : 전통적인 방법
	- 자를 부분을 mosquito로 집음(조직을 crush시킴) → 노장된 부분을 자르고 봉합
	- Blood supply를 막아 shirink되고, 괴사가 발생 → 술자가 목표했던 절제 길이보다 짧아질 수 있음에 주의.
	- **유착**을 유발하는 근육층은 봉합에 포함시키지 않음. Mucosa층만 포함시켜 봉합.
		- 1번째 층 : oropharyngeal mucosa
		- 2번째 층 : 근육층 - 엄청 비대해짐
		- 3번째 층 : nasal mucosa
- (2) **Freehand** technique
	- 가장 많이 쓰임
	- 반만 자르고 봉합, 나머지 반을 자르고 다시 봉합. 출혈을 방지하여 술야 확보.
	- Alice로 버릴 조직을 먼저 잡고 (어차피 버릴 조직이라 crush 괜찮음) 봉합
- (3) Laser Freehand technique
	- 레이저로 봉합하면서 자름
	- 뒤편에 금속판을 대지 않으면, 그 뒤에 어디까지 잘리는지 통제 불가
	- 수술이 엄청 빨라지고, 피를 내지 않고, 염증/부종도 거의 없음 (숙련도가 요구되나, 어마어마한 장점이 있기에 더 활용 많아질 것)


>[!note] 수술 기구
>- Crush한 기구 : Mosquito, Allis tissue forceps
>- Crush하지 않은 기구 : Vascular Debakey, Babcock

# 2. Laryngeal paralysis

- 단두종 증후군과는 구분.

- 후두마비는 아예 움직이지 않는 것(신경의 innervation 이상으로 발생), 들숨 시에도 벌어지지 않음.
	- 후두허탈은 들숨 시에 벌어지려고 움직이긴 하나 계속 무너지는 것.

- 후두마비는 비교적 급성으로 나타나는 거라, 빠르게 내원함
	- 후두허탈은 오랫동안 호흡 문제로 고생한 흔적

#### Etiology
- Congenital : 플랜더스(Flanders), 시베리안 허스키(Siberian Husky)에서 선천적으로 많이 발생
- Acquired : **Recurrent laryngeal n.** 가 손상될 만한 상황이면 다 발생 가능
	- Labrador Retrievers : 후천적으로 신경이 손상될 수 있는 risk factor가 많음.
	- 후두마비는 신경의 이상으로 발생하는 질병이므로, 신경 이상으로 인한 다른 전체적인 이상이 있을 수 있음. Neuromuscular disorders를 갖고 있는 경우가 많기에, 신경계 검사를 쭉 해보는 것이 좋음.
	- 갑상선 기능 저하증, 갑상선 종양
	- Traumatic
	- Idiopathic (제법 많음)
	- Iatrogenic(의인성) : tracheal collapse 수술, 갑상선 제거 수술 시 recurrent laryngeal n.를 만남.
- 대형견에서 호발, Male에서 2~4배 호발

#### Clinical signs

- 한쪽만 후두마비가 있는 경우는 임상 증상이 거의 없음. 
- 양쪽 다 마비가 온 경우 : 목소리가 바뀜, 오연성 폐렴, respiratory distress, hyperthermia
	- 보호자가 임상증상을 호소했을 때는 오연성 폐렴(Aspiration pneumonia)이 있는 경우가 많음. 
	- 인후두 쪽 종양도 recurrent laryngeal n.에 영향을 줄 수 있음

#### Diagnosis

- CBC, serum chemistry : 갑상선 기능 저하증(hypothyroidism) 관련 확인
- 방사선 촬영 : 오연성 폐렴 확인, 종양 확인
	- 갑상선 부근 종양 → 되돌이후두신경에 영향을 주어 후두마비 유발 가능.
- **laryngoscopy** (확진) : 성대가 <u>들숨 때 벌어지지 못하고 가만히 있는지</u> 확인.
- 후두마비가 오래되면 trachea, bronchus 쪽에도 문제 →  tracheoscopy, bronchoscopy를 실시할 수 있음.
- Ancillary test (main 외에 추가로)
	- EMG (Electromyograpy, 근전도 검사) : 의심되는 근육에 electric node를 꽂고, 전기적 자극을 가해서 반응을 확인
	- NCV (Nerve conduction velocity) : 신경 쪽 속도검사 

#### Treatment

- 후두마비의 문제 → 수술적 치료로 바로 해결. <u>성대와 arytenoid cartilage를 열어줌</u>.
	- 호흡 통로가 엄청 좁아짐.
		- 후두허탈은 엄청 고생한 흔적이 많음. 부어 있고, 혈관도.
		- 후두마비는 해부학적으로는 정상이지만, 들숨 때 움직이지 않음.
	- 음식물을 넘길 때도 반응을 못해 오연성 폐렴의 위험이 높다.
- 약물 치료로는 어려움. 어떻게 치료?
	- 외과적으로 성대와 arytenoid cartilage를 열어줌
	- 하지만 오연성 폐렴의 위험도 높아지므로, 한쪽만 적절히 열어서(unilateral) 호흡 통로만 조금 확보
![[Figure. 후두의 구조.png|Figure. 후두의 구조.png]]
- **Arytenoid lateralization** : arytenoid cartilage를 측면으로 벌리겠다.<br> =<u> Cricoarytenoid laryngoplasty</u> : 후두를 성형 (= <u>Tie back</u>: 뒤에 묶는다)
	- 갑상연골을 제껴야 함.
	- Unilateral하게 실시
	- 측면으로 접근, Arytenoid cartilage를 cricoid cartilage로(뒤쪽으로) 당겨서 묶어줌.
	- complication : 너무 많이 벌리면 aspiration pneumonia, 너무 적게 벌리면 dysphagia, dyspnea

- Ventriculocordectomy : 성대를 잘라 호흡 통로를 넓힘, 한쪽만 자름
	- Tie back 실패 시 / 후두허탈일 때도 적용 가능
- Castlellated laryngofissure
	- arytenoid와 thyroid cartilage는 붙어있기 때문에, thyroid cartilage를 옆으로 넓혀주면 arytenoid도 넓어진다고.. 하지만..
	- 쓰는 경우는 못 봄
- Permanent tracheostomy : 영구적으로 구멍을 뚫는 방법.
