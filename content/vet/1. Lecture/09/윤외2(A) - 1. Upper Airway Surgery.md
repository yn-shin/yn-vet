## 1. Brachycephalic Syndrome (단두종 증후군, [[BOAS|BOAS]])

>단두종
>- Dogs
>	- English Bulldog
>	- French Bulldog
>	- Pug
>	- Boston Terrier
>	- Pekingese
>	- Shar-Pei
>	- Shih-Tzu
>- Cats
>	- Persian
>	- Himalayan

##### Congenital Abnormalities
- Stenotic nares : 비공협착. 콧구멍이 꽉 막혀, 개구호흡밖에 못함.
- **Elongated soft palate** : 연구개 노장 (길어지는 것 + 비후되는 것)
	- 선천적이지만, 시간이 지날수록 악화될 수밖에 없음.
	- ![[Pasted image 20250307093845.png|200]]
- **Hypoplastic trachea** : 태어날 때부터 기관이 저형성됨. (단두종 증후군에 포함될지 말지는 계속 논란 중)
	- dorsal band가 없이 태어나는 것. 아주 딱딱하고 기관이 좁음. (사람은 원래 없음)
##### Acquired Abnormalities
- **Everted laryngeal saccules** : 후두낭 외번 [= **후두허탈 1단계**]
	- 콧구멍이 작아지고 음압이 생기면서 후천적으로 발생.
	- ![[Pasted image 20250307094310.png|200]]
- **Laryngeal collapse**
	- 쐐기돌기가 무너져서 서로 붙고, 연구개가 노장되어 있음. corniculate은 보이지도 않음.
	  ![[Pasted image 20250307094439.png|200]]


### 마취 시 주의사항
- 조금만 근육이 이완되어도 위험할 수 있는 부위.
- Induction과 recovery 과정에서 특별히 위험
	- induction 전 처치하면서도 호흡이 멈춰버릴 수 있음.
	- extubation 후 아직 근육이 이완되어 있어서 여전히 호흡 통로가 좁아진 채로 사망할 수 있음.
- 수술 전 5분 이상 충분히 산소를 준다.
- intubation은 빠르게, extubation은 느리게. (빼는 속도 X 빼는 시기)

#### 약물
- steroid 추가 - **dexamethasone** : edema를 잡고 등등
	- 술전, 술후에 steroid 처방. (0.5-2mg/kg IV)

#### Medical Care
- 약물로 근본 원인을 해결할 수는 없지만,
- 수술로는 50%밖에 안 됨. - 체중 감량이 매우 중요함.
- 운동 제한

- Severe : 일단 처치 
	- 진정제 → 노력성 호흡을 완화
	- Steroid 반드시 필요. Edematous한 상태를 완화시키는 데 탁월하므로.
	- 산소 공급, cooling(체온 감소)

#### Surgical Treatment
- ⭐ **Complete Upper Airway Examination First** ⭐
	- 비공협착, 연구개노장, 기관저형성 / 후두낭 외번, 후두허탈 : 5가지를 통으로 떠올리고, 전체 검사 필요
- 가능한 3-4개월령에 하는 게 권장됨
	- 시간이 지날수록 2차적인 문제들이 심해짐. 다리를 안 쓰고, 심장이 커지고,,
	- 산소포화도가 적은 상태가 지속적으로 유지되면 심혈관계에도 문제가 됨. → 필연적으로 심장병이 심하게 옴.
	- 마취를 견디기에도 좋은 나이.
- Wedge resections
	- **Vertical** : 무너진 연골을 많이 잘라내기 위함. 목적성은 가장 맞음.
	- Horizontal : 끝부분 soft tissue만 살짝 잘라냄.
	- Dorsolateral : 끝부분을 자르고 피부와 봉합하여 넓힘.
- Laser amputation of rebundant alar fold : 레이저로 구멍을 뻥 뚫어줄 수 있으나, 충분히 숙련되어야 함.

#### Postoperative care & assessment
- 편한 자세로 - sternal recumbency
- 하루 정도는 금식
- Elizabeth collar : 발로 긁으면 큰일남. 넥카라 중요.
	- 혀는 control 불가, 봉합사가 풀리고 피가 날 수 있으나, 연골 제거가 가장 큰 목적이었고 그런 출혈은 일주일면 사라질 것.

#### Prognosis
- 연구개 노장, 후두낭 외번, 후두허탈 등이 없으면 예후가 좋으나.. 연구개 노장은 선천적인 부분
- 후두허탈이 없으면 = 병이 심화되지 않음 = 예후가 매우 좋음.

### Elongated soft palate
![[Pasted image 20250314093544.png|470]]
- 연구개는 정상적으로 epiglottis를 살짝 덮고 있음.
- 들숨할 때 훅 빨려들어가면서 막음.
- Aspiration
- 길이가 정상일 때는 숨을 내쉴 때 빠져나가는데, 길이가 너무 길면 오히려 내쉴 때 nasopharynx를 막음.

- Respiratory distress during inspiration 흡기 때 호흡 어려움. 
- Gagging 토하려고 함
- Stertor 호흡음 (소리는 같으나, 잘 때 소리가 나는 것을 snoring이라고 함.)
- 호흡이 잘 안 되면 반드시 체온이 올라감
- 심장 비대, 폐수종, 폐렴

#### Surgical treatment
- Medical care는 동일.
- Young(4-24개월)에 수술 필요. 
	- 마취를 견딜 수 있는 가장 빠른 나이, 4개월.
	- 너무 어릴 때 잘랐다가 다른 기관만 성장하면, 연구개가 오히려 짧아질 수 있음. (⇒ 오연성 폐렴)
	- 수의사의 판단이 매우 중요함. 가능한 빨리 수술하는 것이 좋지만, 다른 곳의 병의 정도가 괜찮을 때는 조금 보류할 수 있음.
	- 어린 나이에 수술하게 되면, 생각했던 것보다는 길게 자르기. 증상은 많이 개선될 것이고, 나중에 한번 더 수술해도 괜찮음.
- 후두 쪽 퇴행성 변화 나타나기 전에 수술 해주는 게 좋음.
- 부종 완화할 수 있는 steroid - 수술 때문에도 부종 생길 수 있음.
- 얼마나 자를까? Endotracheal tube를 삽관하면 잘 보이지 않음. → 자를 위치를 미리 표시해두어야 함.
- 수술하면 출혈 있음. Tube 주변에 거즈 장착.
- Techniques
	- Crush : 
	- Freehand : 
	- Laser Freehand : 
	- 너무 많이 자르는 것을 주의. 
	- 많이 자르면 오연성 폐렴의 위험, 짧게 자르면 다시 수술해야 할 수도. 그래도 짧게가 낫다!
	- Epiglottis의 1-2mm를 덮는 게 정상 길이 but epiglottis 계속 움직임.
