## Canine Babesiosis

![[Pasted image 20250617013210.png|450]]

| Babesia              |                                                                                                                                                                                                                                                                                  |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Etiology             | RBC에 기생하는 원생동물 → anemia 관련 증상.<br>다양한 species가 있으며 우리나라는 주로 *B. gibsoni*<br>적혈구에서 복제되어 혈관내/혈관외 용혈성 빈혈을 일으킨다.<br> + 발열, 간비장비대, 저산소증, DIC                                                                                                                                          |
| Clinical<br>Features | - **Peracute** : 빈혈 (빈호흡, 빈맥, 우울, 쇠약) / 용혈성 빈혈 (황달, petechiae, 간비장비대)<br>- **Acute** : 심각한 빈혈, DIC, 신장 질환<br>- **Chronic** : 체중 감소, 기력 부진<br>- **Atypical** : 복수에서 DNA 검출                                                                                                        |
| Diagnosis            | 빈혈, 혈소판감소증 / 혈액 도말해서 RBC의 organism 확인 / PCR로 감별<br>[*gibsoni* - 작은 점 모양 / *canis* - 조롱박 모양]<br>항체 반응 양성이어도 현재 감염 아닐 수 있음.<br>고양이에서 APPs 확인 (serum amyloid A 증가, paraoxonase-1 감소)                                                                                                |
| Treatment            | 수혈, 중탄산염 치료, 수액 치료 + 약물 사용<br>[US] *vogeli* → Imidocarb diproprionate / *gibsoni* → azithromycin + atovaquone (w/ 아이스크림)<br>[Asia] 다양한 약물 조합이 제시됨. (100%는 없음)<br>1st : AA protocol (azithromycin + atovaquone)<br>2nd : MCD protocol (metronidazole + clindamycin + doxycycline) |
| Prevention           | 한 번 걸렸어도 다른 species에 감염될 수 있음.<br>개물림 사고 조심, 공혈 시 PCR screening 꼭.                                                                                                                                                                                                               |

## Canine Leptospirosis

![[Pasted image 20250617010301.png|400]]

| Lepto                |                                                                                                                                                                                                                                |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Etiology             | <u>사람과 동물 모두에 감염</u>되는 운동성 있는 나선형 세균.<br>다야한 혈청형, 풍토병 (지역별 차이), 습지나 농촌, 여름에 잘 발생.<br>Adapted → 증상 없을 수 있음. / 면역 없는 개체에서는 증상이 명확.<br>Urine을 통해 전파됨 - 토양, 음식, bite, 수직 감염 등                                                      |
| Clinical<br>Features | 다양한 조직에서 복제하지만 특히 <u>간, 신장</u>에서 높은 역가.<br>- **Peracute** : 발열, 우울, 빈호흡, 구토, 발열, 빈맥, 급사 가능.<br>- **Subacute** : 경미한 바열과 우울, 간과 신장 질환<br>- **Chronic** : PU/PD, 복수, 간성 뇌병증                                                      |
| Diagnosis            | 농촌 등 history + 신장 등에 이상 ⇒ Top Ddx는 아니지만 고려해야 함.<br>- USG 감소, bilirubinuria, cast 확인 / Renomegaly, hepatomegaly<br>- [키트] MAT(Microscopic Agglutination test)  : titer가 정상의 4배 이상<br>**Definitive Dx.** : PCR, 뇨/혈액/조직의 감염체 확인. |
| Treatment            | - 수액 처치 : 신장에 영향 많이 주므로 기본 수액 처치.<br>- 항생제 : ampicillin,amoxicillin, doxycycline                                                                                                                                               |
| Prevention           | 인수공통 조심, 배설물/물/그릇 조심, 몇 개 혈청형에서만 유효함.<br>DHPPL의 L, 지금은 빠졌으나 코어 백신 될 수도 있음.                                                                                                                                                     |

## Canine Distemper Virus

![[Pasted image 20250617004435.png|450]]

| CDV                  |                                                                                                                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Etiology             | *Paramyxoviridae* 에 속하며 strain에 따라 병원성이 다양하게 나타나는 바이러스.<br>호흡기를 통해 들어와 관련 증상을 유발했다가, 신경계까지 퍼져서 신경 증상을 유발한다(8-9일).<br>어린 개에서 심각하게 나타남. (그래서 마음 아픈 케이스들이 많다..)<br>면역이 형성되면 그리 심각하지 않음. |
| Clinical<br>Features | 우울, 무기력, 기침, 구토, 설사, 신경 증상.<br>- 편도 비대, 기관지 폐렴<br>- 무도증(얼굴 근육 마음대로 떨림), 발작 통제 안 됨.<br>- 포도막염, 시신경염, 망막맥락염 등 안과 질환도 동반<br>- 치아 표면이 울퉁불퉁, 코와 발바닥의 과각화                                  |
| Diagnosis            | - 임상 증상 (호흡기 → 신경계) + 나이 로 잠정 진단.<br>- CSF 검사 → 단핵구 증가, IgG 상승<br>**Definitive Dx** : DFA - 직접형광염색, 조직 검사, PCR 등으로 감염체 확인!                                                           |
| Treatment            | 항생제, 항경련제 등 but 예후 안 좋음.                                                                                                                                                             |
| Prevention           | 체외 생존 시간 짧고 소독제에 쉽게 사멸됨.<br>생독(modified live) 백신 : 6-16주령에 3-4주 간격으로 3회 이상 접종.                                                                                                       |
