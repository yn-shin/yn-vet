# 1. Kidney

- 신장의 크기 (VD상)
	- 개 : L2의 2.5~3.5배
	- 고양이 : L2의 2.4~3배
	- hydronephrosis나 보상성으로 정상보다 커질 수 있다.
	- 중성화 수컷 고양이에서는 신장의 크기가 작다.

##### Gastric axis
- gastric axis가 정상이면 늑골과 나란함.
- Caudo-dorsal 변위 = hepatomegaly(간 비대) ⇒ Cushing's dz, diabetes, 우심부전 등
	- Hepatomegaly : 간의 caudal margin이 뭉툭하고, rib cage를 벗어나 있다.
- Cranio-ventral 변위 = microhepatica(간 위축) ⇒ [[PSS|PSS]](portosystemic shunt), chronic hepatitis, liver cirrhosis 등
	- 감별 방법 : BUN 정상 → cirrhosis / BUN 감소, NH3 높음, 어린 나이 → PSS 의심

##### 복부 방사선상 비정상 신장 소견
- C-NHAN (cyst, nodule, hematoma, abscess, neoplasia) + 결석, 염증, 종양 
- Renomegaly : 개) L2의 2.5-3.5배 이상일 때, 고양이) L2의 2.4-3배 이상일 때
- Renal calculi
- Small/irregular calculi
- hydronephrosis (수신증) : renal pelvis dilation, 
#### Kidney
>[!note] CC) 포도 섭취, 구토 ⇒ Dx) Acute renal failure
> - Hepatomegaly : 간의 caudal magin이 뭉툭하고, rib cage를 벗어남. gastric axis가 caudodorsal로 변위됨. 
> - Renomegaly : 좌신의 장축이 L2의 4배

>[!note] CC) 건강검진
>- Hepatomeglay : 간이 뭉툭하고, rib cage 밖으로 빠져나옴.
>- Gall bladder sludge
>- Lt. renal calculi, small and irregular kidney : 신장 음영이 작고 결석 관찰
>- Rt. ureteral dilation & caliculi : 요관은 원래 초음파상 보이지 않지만, 요관 확장/obstruction/요관 결석/ectopic ureter로 요가 차 있으면 확인됨.

#### Renal mass
>[!note] CC) 1주일째 켁켁, 구토, 식욕부진 ⇒ DDx) Renal cell carcinoma, Metastatic lung tumor
>- Hepatomegaly
>- 폐실질에 severe nodular pattern : multiple nodule은 metastasis를 의심.
>- Bilateral renal calculi
>- 좌신에 연부조직 밀도의 mass effect
>- 초음파 | 내부의 mineralization, 수피질 경계가 불규칙하다 못해 아예 뒤바뀜, cyst.

>[!note] CC) 발열, 복통, 기력저하, 식욕부진, 구토 ⇒ Dx) Renal TCC (transitional cell carcinoma)
>- 좌신이 보이지 않고 커다란 mass effect가 꽉 채우고 있음
>- 장내 gas가 차 있는데, 그 중에서도 덜 lucent한 영역이 있음 : background에 mass effect를 의미

>[!note] Dx) Renal adenocarcinoma & lung metastasis
> - 우측 renomegaly, 경계 불규칙
> - 소장의 좌측 변위
> - 폐엽 전반에 multiple, diverse-sized, round한 연부조직 음영 
> - 초음파 | pelvis 확장, hydronephrosis 확인

>[!note] Dx) 요관 파열에 의한 수신증+복막염 ⇒ Tx) Nephrectomy
>- Mass effect, serosal detail 저하, 소장이 앞쪽으로 deviation : 복강 내 액체 저류 의심
>- 결장 내 gas 축적
>- Lt. 신우 확장 → hydronephrosis(수신증), ureter 보이지 않음, diverticulum(collecting duct) 확장
>- 신장 주변으로 뇨가 누출되어 perirenal peritonitis


# 2. Ureteral disease

>[!note] CC) vomiting, diarrhea ⇒ Dx) Rt. renomegaly & hydronephrosis & caliculi, Rt. Hydroureter
>- Calculi : 우측 요관, 좌우측 신장, 방광
>- Rt. Renomegaly : L2 body length의 4.2배
>- Dilated renal lpelvis : 지저분함, cell debris 의미, pyonephrosis

>[!note] CC) 우울, 기력저하, 구토, 복통 ⇒ Dx) 좌측 요관 결석  & 수신증, Lt. renomegaly & calculi, hydroureter
>- Lt. kidney renomegaly
>- Lt. renal pelvis calculi : 결석이 중앙에 위치해 있으므로 신우결석 (주변에 있으면 실질에 생긴 결석)
>- Hydronephrosis : 신우결석보다는 요관폐색에 의해 가장 잘 생기므로, 요관 결석 확인 필요
>- Lt. ureteral dilation : 요관 확장이 끝나는 지점에 결석 있을 것

>[!note]  Dx) Right ureteral rupture & Peritonitis
>- Lt : Renomegaly, 신우 결석, serosal detail 저하
>- Rt : 요관 확장, 신우 확장 : 요관(원래 2mm)이 확장되고 조영제 축적, 방광으로 가는 요관 안 보임 ⇒ <u>요관 파열</u>
>- 초음파 | **복막염** 소견
>	- Hyperechoic mesenteric fat : 장간막이 고에코성으로 밝게 보임 
>	- Small intestine corrugation : 복막염으로 장벽이 자극되어 표면이 쭈글쭈글해짐



# 3. Urinary bladder & Urethra

>[!note] CC) 3일 전부터 소변 못 봄, 구토 ⇒ Dx) [[FULTD|FULTD]] (고양이 하부 비뇨기계 질환), Urinary bladder edema
>- Bilateral renomeglay : 신장이 L2의 3.3배 (고양이는 2.4~3배 정도)
>- 방광에 뇨가 가득 참 : 복부에서 원형 돌 덩어리처럼 만져짐.
>	- 고양이는 뇨 내에 mucin 성분이 축적됨.
>	- 요 생산이 이루어지는데 못 빠져나감 → 방광 팽대, 요관 확장, 신우 확장, 수신증 유발될 수 있음. 
>	- 요 축적이 계속되면 방광 점막 비후, 방광 벽으로 요가 침투, 방광벽 부종 (UB edema) ⇒ 원형 돌처럼 느껴짐.
>	- 요도가 막혔을 가능성 생각해야 함.


>[!note] CC) panting, 복부팽만 ⇒ Dx) 방광&신우 결석, Cystitis
>- Hepatomegaly
>- 요관 결석, Rt. 신우결석
>- 방광 결석(cystic calculi) ⇒ 방광염(cystitis) 가능
>- 방광이 L4-L5 수준으로 displacement : 복강 내 gas 등에 의해

>[!note] CC) Hematuria ⇒ Dx) TCC
>-  방광 후반부 결석 : 자세상 방광 중간 쪽으로 결석이 쏠려야 하는데, 뒤쪽에 머물러 있다면 전립선 요도(prostatic urethra)에 있을 가능성. 
>- 중성화한 개에서 전립선 비대, mineralization ⇒ 악성 종양 가능성 높음.

>[!note] CC) HBC
>- Traumatic diaphragmatic hernia : 흉강의 cardiac effacement, 간 경계 확인 안 됨, 간 경계 너머에 연부 조직 밀도의 tubular 구조
>- <u>Urinary bladder rupture</u>, Urethera rupture : 요도에 넣은 조영제가 복강으로 다 새어나감.

 >[!note] 방광, 요도 결석
>- 방광, 요도 결석 : 방광 결석이 요도 직경(2-3mm)보다 큼 → 요도 점막 꽉 끼고, 혈액 공급 안 되어 괴사, 궤양, 파열, 출혈, 염증, 등등..
>- Hydroureter, hydronephrosis

##### Pelvic 복합 골절이 확인되는 교통사고 환자에서 비뇨기계 손상을 의심할 수 있는 방사선 소견, 진단명과 뒷받침 소견, 확진을 위한 검사
- 비뇨기계 손상을 의심할 수 있는 방사선 소견
	- 하복부 serosal detail 저하 및 soft density 증가 (방광, 요관 파열, 출혈 의심)
	- 조영제의 복강 누출 (방광, 요도 파열 의심)
	- 방광 경계가 뚜렷하지 않음
- 진단명 설정 : 방광 파열(UB rupture)
- 검사 : 복부 방사선 1st, 초음파 2nd, 이후 조영술(EU, RGU), 복강 천자, 혈액 검사, CT
- 뒷받침 소견 
	- 초음파: 방광벽의 연속성 소실, 방광 내 sludge, 방광 collapse
	- RGU: RGU로는 방광과 요도를 확인 가능한데, 방광에서 조영제 누출 소견 확인됨.
	- 복강천자: 천자액의 BUN, creatinine이 혈액 내 수치보다 높음
