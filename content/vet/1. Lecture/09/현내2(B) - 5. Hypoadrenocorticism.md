- **Cortisol** : H-P Axis의 영향을 많이 받음.
- **Aldosterone** : RAAS 시스템에 의해 조절되는 부분이 더 큼.
- 에디슨 : 알도스테론 분비를 하지 못함

# Hypoadrenocorticism (HOAC), adrenocortical insufficiency

- **Primary hypoadrenocorticism** (= <u>Addison's disease</u>) : 부신피질 기능 자체에 문제
	- aldosterone & cortisol 둘 다 감소
	- 원인은 명확하게 밝혀지진 않았지만,
		- immune-mediated destruction
		- neoplasia, tuberculosis, fungal dz
		- granulomatous or infectious
		- trilostane : 피질 괴사 유발 X, 합성 자체만 억제. but 아주 소수의 환자에서는 부신 피질 괴사 유발
		- mitotane :`
- **Secondary hypoadrenocorticism** : 뇌하수체의 문제로 ACTH가 분비되지 않아 이차적으로 부신 피질 기능 저하
	- Aldosterone은 RAAS 시스템의 관여 ↑ → 정상 범위 유지
	- Cortisol과 성호르몬만 저하됨
	- 원인
		- 선천적인 뇌의 damage
		- 대부분은 <u>iatrogenic</u> : 스테로이드의 장기간 복용
- Tertiary hypoadrenocorticism
	- 시상하부의 문제 → CRH도 감소하는 것이 특징
	- 사람에서도 드물고, 개에서는 발생 보고가 없다.

### Clinical features

##### Predisposition

333
- Age : young ~ middle-aged
- Breed : Poodle에서 호발
- Sex : 암컷에서 2~2.5배까지 호발

##### Common clinical features

: 엄청 비특이적. 증상만 보고 예측하기는 쉽지 않음.

- Lethargy, anorexia, weakness, weight loss, shivering/shaking (저혈당으로 인한 전신 증상)
- Vomiting, diarrhea, abdominal pain (cortisol의 위장관 흡수 감소)
- PUPD (흔하지 않음, 수화되면서 나타나기도 함)


스트레스 이벤트에 악화되는지 / 스테로이드나 수액 처치에 반응성이 좋음

>[!warning] Addison crisis : 에디슨으로 인한 응급 상황
>- Addison crisis, acute adrenal insufficiency
>- 평생 1번 정도는 경험할 수 있으니, 보호자에게 사전 고지 매우 중요!! 바로 병원 데려오도록.
>- 쿠싱/갑기저에 비해 emergency 발생 가능성 ↑
>- Coma로 빠르게 진행
>##### Clinical features
>- Vomiting, diarrhea
>- 심한  GI bleeding
>- 심한 탈수, collapse 상태나 shock 발생 가능

### Diagnosis

- 심한 탈수, 저혈량증 때문에 맥박 저하
- 전해질 불균형이 흔하지만, 너무 비특이적
	- **Na:K ratio < 27**
	- 하지만 이것도 비특이적, 낮을 수 있는 Ddx가 ~~~

- 저혈량증 지속 → 조직으로 가야 할 혈류가 충분X → Multi Organ Failure 발생 가능
- 탈수 교정하면 masking된 소견 나타날 수도
- 중증 상태에서도 stress leukogram 나타나지 않는 게 특징. (Absence of a stress leukogram)
	- WBC 올라갈 때 
- 요비중 감소하는 이런 질환들과도 유사. <br>![[Pasted image 20250530144444.png|250]]
- ACTH stimulation test : iatrogenic에 대한 감별 필요.
- 에디슨 진단에서는 IV로 하는 게 권장된다. 탈수+저혈량증, 흡수도 분포도 잘 안 되기 때문에 그나마 IV를 해야 유리.
- 검사 결과에 영향을 줄 수 있는 factors
	- 검사 정확도를 높이기 위해 
- Baseline endogenous ACTH :

##### Chronic Treatment (Long-term 관리)

- **PDS** : 천천히 tapering

- DOCP : 들어가면 aldosterone으로 전환, mineralocorticoid 보충 효과만
	- 몇 시간 내로 증가, 2주쯤 peak, 한달 정도 지속
	- SC로 주사
	- Peak 때 전해질 농도, 환자 body condition 체크해서 용량 조절 ⇒ 25일째에 결정해서 주사
	- 부작용 : 체중 증가, 얼굴 부음, 울혈성 심부전
