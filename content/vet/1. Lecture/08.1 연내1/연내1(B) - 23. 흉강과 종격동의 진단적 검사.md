## 14주차 : Clinical Manifestations and Diagnostic Tests of Pleural Cavity and Mediastinal Disease

## Clinical signs

- 흉강에 공기가 차면서 폐가 쪼그라듦. → 심장이 위로 들림.
- 펼쳐진 폐가 쪼그라들면서 하얗게 보임.

- mediastinum 쪽 thymus 종양
- 청진 시 폐음이나 심음이 먹먹하게 들림.

### General Diagnostic Approach

- 액체가 찼을 때 **흉강 천자(thoracocentesis)**
	- 옆으로 뉘고 ICS 사이 찌르면 보통 공기가 먼저 나옴. (공기가 위로 들리기 때문)
	- 갈비뼈 바로 뒤쪽에는 동정맥과 신경이 지나가기 때문에, ICS 중 cranial part를 찌르는 게 좋음. 갈비뼈 바로 뒤쪽은 피하기.

#### Analysis of effusions

- 색, 투명도 확인
- 유핵세포수 측정 (TNCC) / 단백질량 측정 (TP)
- 생화학 수치 측정
	- Creatonine : 오줌에 있어야 할 게 나온 거니, 방광이나 콩팥이 터졌을 때.
	- Bilirubin : 담낭이 터졌을 때.
	- TG : 높아지면 유미흉. 
	- Glucose, Lactate, Tchol
- Cytology 세포모양관찰

- 단백질도 낮게, 유핵세포수도 낮게 → 흉수

### FLUID CYTOLOGY (Pleural Effusions에 대한 진단적 접근)

| Transudates                  |                                                                 |
| ---------------------------- | --------------------------------------------------------------- |
| pure transudates             | TP < 2.5, TNCC < 500                                            |
| 정수압 증가                       | RCHF(우심부전), pericardial disease에 의해.                            |
| 삼투압 감소                       | 저알부민혈증에 의함.<br>- 간부전, 사구체신염, PLE(Protein-losing enteropathy) 환자 |
