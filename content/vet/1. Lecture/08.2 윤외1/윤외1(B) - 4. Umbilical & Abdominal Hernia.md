# 1. Definition
- 하나의 연계성이 있는 복벽에서 defect가 발생해서 내부 장기가 튀어나오는 것.
- 내부 장기가 나오면서 다음과 같은 구조가 생김.
	- Hernial sac
	- Hernial neck
	- Hernial ring
	- Hernial contents
- True hernia : 연속성 있는 복벽에서 장기가 튀어나와 해부학적으로 hernial sac 내부에 있음.
- False hernia : hernia sac이 없음. (==Trauma, surgical incision==)
# 2. Principles
- 재발이 없도록 하는 것이 핵심.
- Sac에 있는 조직 중 불필요한 조직은 제거
- 심각한 유착이 있을 때를 제외하면, ==신선창을 만들지 않는다==. (유합은 잘 일어나나 ring의 구조 불안정해짐)
- Local anatomy 숙지 → ring 확보
- sufficient holding power로 suture해야 함.
- Tension free closure : 조직이 부족해서 봉합할 때 tension이 심하다면,
	- ==**Flaps**==
	- Mesh
	- SIS (Swine Intestinal Submucosa) 사용 가능

### Herniorrhaphy
- Closed : Sac과 내부 장기를 한꺼번에 ring 안에 집어넣음. 현실적으로 어려움.
- Open : sac을 열고 일부분을 제거한 뒤, 내부 장기를 ring 안으로 집어넣음.

# 3. Hernias
## Umbilical Hernia (제대 탈장)
- True hernia
- Congenital
- 복벽의 융합에 문제
- Herniorrhagphy : edge trim하지 말 것, ==흡수성== 봉합사 + interrupted pattern
- Omphaloceles : 복벽+피부까지 문제 발생. 복강 size 작음. 쉽게 찢어지는 amniotic tissue로 덮여 있음. 거의 사망으로 이어짐. 

## Inguinal Hernia (서혜 탈장)
- 비만인 강아지, 고양이와 구분 필요.
- 탈장이 일어난 곳 바로 위에서 봉합하는 것을 권장.
- 양쪽 ring을 한번에 봉합해야 함.
- 너무 tight하게 봉합하면 신경, 혈관의 압박

- internal canal의 경계
	- [cranial] internal abdominal oblique muscle의 caudal edge
	- [medial] retus abdominis muscle
	- [lateral & caudal] inquinal ligament
- external canal의 경계
	- external abdominal oblique muscle의 longitudinal slit
## Scrotal Hernia 
- Indirect hernia (vaginal ring을 통함)
- DDx : Testicular torsion, hydrocele(음낭수종), orchitis(고환염), tumor
- Strangulation (세게 조이면) X. 재발 위험 때문에 castration이 권장됨.

## Femoral Hernia
