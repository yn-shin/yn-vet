# Cytogram 

| 기본적인 Erythrocyte cytogram                 |                                           |
| ----------------------------------------- | ----------------------------------------- |
| ![[Pasted image 20241114004711.png\|270]] | ![[Pasted image 20241114124415.png\|320]] |

그래프를 통해 Hgb 양과 세포 크기 (MCHC, MCV 등) 확인할 수 있다.

| 기본 예시                                                          |                                                                                                                                                                                                                                                          |
| -------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **정상 feline**<br>![[Pasted image 20241114004940.png\|200]]     | (1) 정상 → Normochromic, normocytic에 몰려 있음.<br>(2) 고양이의 normocytic region이 개에 비해 아래쪽에 위치.<br>: RBC 크기가 더 작기 때문.                                                                                                                                            |
| **약한 빈혈의 canine**<br>![[Pasted image 20241114005115.png\|200]] | 정상이면 Normochromic, normocytic에 많아야 함.<br>(1) 정상에 비해 Macrocytic & hypochromic region (i영역)에 많이 퍼져 있음.<br>⇒ <font color="#0070c0">**RDW 증가, Reticulocytosis**</font><br>: 개가 정상적인 재생성 반응을 나타낸다는 전제 하에,<br>macrocytic, hypochromic RBC = Reticulocytes를 의미. |

| Case A                                    | 12y, mixed breed<br>mild anemia, epistaxis 3개월째                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 병력 청취 및 신체 검사                             | - 3개월 코피 지속 ⇒ 출혈성 빈혈<br>- mucous membrane color에서도 나타날 것                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ![[Pasted image 20241127165636.png\|150]] | - PCV : 29% ⇒ mild anemia<br>- MCV : 감소 (microcytic, 가운데보다 약간 왼쪽으로 치우침)<br>- 따라서 RDW 증가.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ![[Pasted image 20241127171125.png\|200]] | - <font color="#0070c0">**Microcytic**</font> : 두드러지게 나타나지는 않음. (margin에 걸친 정도)<br>- <font color="#f79646">**Normocytic, macrocytic**</font> / <font color="#f79646">**hypochromic**</font> cells 존재 : Reticulocytes에 해당.<br><br>⇒ 재생성 반응이 있었으나,<br>만성 출혈로 철분이 결핍되어 <font color="#0070c0">**Microcytosis**</font>가 나타나기 시작. + Reticulocytosis는 아니지만, 아직 <font color="#f79646">**reticulocytes**</font>가 남아있는 상태.<br><br> ❔Normocytic + Hypochromic cell들은 왜 있나요?<br><font color="#7f7f7f">정상적으로 normo/normo → 빈혈이 지속되면서 normocytic/hypochromic<br>→ 재생성 반응 있으면 macrocytic/hypochromic (이 단계 중에 있음)</font><br>\* rubriblast ~ reticulocyte까지는 Hgb 합성이 완성된 상태가 아니므로 hypochromic함. |

| Case B                                    | 12y, miniature schnauzer<br>very mild anemia                                                                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ![[Pasted image 20241127171543.png\|150]] | - PCV : 37% (많이 감소하지 않음)<br>- MCV : 살짝 큰 편 (약간 오른쪽으로 치우침)<br>- RDW : 증가 (MCV가 정상 범위에 있지 않으니 RDW는 증가함)                                                                                                                                                                                                                                                                                          |
| ![[Pasted image 20241127172206.png\|200]] | - Hypo & Normochromic <font color="#8064a2">**macrocytes**</font> 관찰됨.<br>- 앞 케이스에 비해 **reticulocytosis가 더 활발**함. (수치와 그래프상 RBC volume이 더 큼)<br><br>\*  앞 케이스는 macrocytes가 나타났지만, MCV는 감소함.<br>재생성 반응이 없는 것은 아니지만 활발하다고 할 순 없음. (재생성 반응 활발하면 macrocytosis 나타나는 것이 정상)<br>지속된 빈혈로 생체내 Hgb 합성 능력이나 재생성 반응이 떨어진 상태.<br>⇒ 이 케이스는 MCV도 높게 나타나고, macrocytic cells도 많이 나타난 걸 보아 reticulocytosis가 활발함. |
