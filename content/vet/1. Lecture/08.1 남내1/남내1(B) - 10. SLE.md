# Uncommon Immune-Mediated Disease  7) Systemic Lupus Erythematosus

# 1. Etiology

- 흔하지 않으나, 이름은 많이 등장하는 질병..
- Multisystemic immune disorder
- **Type Ⅰ 제외하고** Type Ⅱ, Ⅲ, Ⅳ 모두 해당할 수 있음.
	- Type Ⅱ hypersensitivity : Hematologic, IgG & IgM
	- Type Ⅲ hypersensitivity : immune complex 침착
	- Type Ⅳ : 근염을 동반하는 경우
- 원인이 잘 밝혀지지 않음.
	- CD4/CD8 ratio의 증가, T-cell의 과발현, lymphopenia 등이 나타나는 것을 보아 ==**T-suppressor cell의 결손**으로 추정됨==.
	- MHC type과도 관련 있을 것으로 예상.
	- environmental factor, 특정 감염체나 약물에 대한 노출도 risk factor
	- ==Heritable== (유전성 존재) - but 단순 상염색체 유전은 X
- 품종 소인 : German Shepherd dog, Shetland Sheepdog, ==Collie, Beagle, Poodle==.. 좀 큰 애들 `저먼셰퍼드와 푸들은 ITP와 겹침 / English 말고 Shetland / 비글이랑 콜리 추가`

# 2. Clinical Features
- 개에서 가끔 있음(uncommon), ==고양이에서 거의 없음==(rare)
- 개에서,
	- 노령성 질병은 아님. Middle-aged (1-11년령)
	- 성별 소인 X
	- 2개 이상의 장기에 영향을 미치고, 그에 따라 임상 증상이 다양.

>[!note] 임상 증상
>- **Fever** : 열은 무조건
>- **관절** : nonerosive polyarthritis 와 같이 - Lameness, joint swelling
>- **신장** : 사구체신염 → Proteinuria, 콩팥 기능 부전 증상 (체중 감소, 구토, 다음 다뇨)
>- **피부** 증상 : photosensitization, alopecia, erythema, ulceration, crusting, hyperkeratosis, pemphigus(천포창) 의심 소견.. 다양
>- 용혈성 빈혈, PRCA, thrombocytopenia, leukopenia, myositis, pleuropericarditis, laryngeal paralysis, CNS signs
>
>어느 장기에서 걸렸는지에 따라 증상이 다르게 나타남. 열, 관절염 > 신장, 피부 관련 

- 어떤 장기에서나 다 생길 수 있다는 점. 증상이 광범위하고, 퍼센트가 다양하다.
- 신경근육 관련 증상 → 적혈구, 혈소판 관련 증상.
- 고양이에서) ==피부와 관련된 병변이 비교적 흔하게== 발생.

# 3. Diagnosis
- ==2개 이상의 장기에 면역매개로 인한 질병의 증거==가 있어야 한다.
- 영향을 받은 장기 관련 다양한 검사 - 환자마다 매우 다름.
- 고양이에서 감별해야 하는 바이러스
	- <span style="background:#e0e5fc">FeLV</span> (feline leukemia virus)
	- <span style="background:#e0e5fc">FIV</span> (feline immunodeficiency virus)
	- <span style="background:#e0e5fc">FIP</span> (feline infectious peritonitis)

![[Pasted image 20241206110048.png|600]]

- ==ANA test, LE test가 유용==
	- [2 major sign + ANA/LE 양성] 이거나, [1 major + 2 minor + ANA/LE 양성] 이면 확진
	- [1 major or 2 minor + ANA/LE 양성] 이거나, [2 major + ANA 음성] 이면 의심
-  의심되는 상황에 대해 약물 치료 등을 쓰면서 치료

>[!check] ANA test
> - 상대적으로 민감함.  (진단 자체가 쉽지 않은데, 그나마..)
> - 위양성 결과 생각보다 높음. (10-20%)


>[!check] LE test
> - 매우 특이적이나, 임상적으로 잘 이용되지 않는다.
> - 분엽이 잘 된 핵들이 가장자리로 치우쳐 있는 세포.<br>![[Pasted image 20241206111824.png|300]]


# 4. Treatment
- PDS (1-2mg/ko PO q12h)
- 보조 면역억제제 (Azathioprine, cyclosporine, mycophenolate)
- 예후가 좋지는 않다.
	- 치료가 잘 되어도 재발이 흔함.
	- 평생 약을 먹어야 함.
	- A, B 장기에 생겼다가, 재발 시 C 장기에 생기는 식.
	- 점점 관리가 어려움. 안락사할 때도 있고, 혈전으로 급사할 수도 있음.
- 궤양 생기고, 진물이 나고, 잘 못 걷고, 피 계속 나고.. 보호자의 마음이 아픈 질병..
