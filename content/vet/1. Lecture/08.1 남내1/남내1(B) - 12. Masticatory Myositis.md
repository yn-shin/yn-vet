# Uncommon Immune-Mediated Disease 9) Masticatory Myositis (저작근 근염)

# 1. Etiology

- 저작근 : Temporalis(관자근), Masseter(깨물근), Digastricus(두힘살근)
![[Pasted image 20241213113825.png|250]]
- 저작근은 Muscle fiber type (type 2M) 을 포함하고 있다. → 여기에 특이적인 항체
# 2. Clinical features
- ==개에서 흔하고, 고양이에서 드물다==.
- 어리거나 대형견에서 많이 발생한다.
- 품종/성별 소인은 없다.
	- 단 ==Cavalier King Charles Spaniel 에서 유년기 저작근염이 잘 발생==한다.

### 증상
- Acute phase : swelling pain
- Chronic phase : severe muscle atrophy, inability to open jaws (trismus)
	- 덜 심각하면 혀는 움직일 수 있어 액체 마시는 건 가능
- 열, 우울, 체중 감소, dysphagia, dysphonia, exophthalmos (안구돌출) from swelling of pterygoid muscles

# 3. Diagnosis
- ==2M Antibody 검사 : 80% 이상에서 양성==. 
- Creatine kinase 측정 : 근육에서 방출되는 효소.
	- 일부 개에서 증가하나, 두드러지게 나타나지는 않음.

# 4. Treatement
- 진단이나 치료 시 억지로 입을 벌리면 부러지거나 턱이 빠질 수 있음.
- Steroid를 먹고 임상 증상이 호전되면 천천히 tapering한다.
- 증상의 호전과 CRP를 확인, 
- 추가적인 면역 억제제 투여를 고려할 수 있다.
- 심한 → 장기간 복용에 주의.
