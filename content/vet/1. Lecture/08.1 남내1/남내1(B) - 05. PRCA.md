# Uncommon Immune-Mediated Disease 2) Pure Red Cell Aplasia

# 1. Etiology
- ==Erythroid precursors이 아예 없거나, 두드러지게 감소하여 나타나는 심각한 **nongernerative anemia**==
- Spherocytes가 보이거나, Coombs test 양성일 수도 있다. (대부분의 Coombs test는 음성)
- IMHA의 큰 범주 안에 PRCA가 있다고 여겨지고 있다.

>[!check] 일반적인 IMHA와 구분되는 점
>- 비재생성 IMHA : maturation arrest가 나타남 - rubricyte나 metarubricyte level에서 멈춰 있음.
>- PRCA : 어느 단계에서부턴가 전부 없음. **Erythroid precursor가 거의 아예 없어지는 것이 특징!**

>[!tldr] Fig. IMHA의 범주에 대한 경향성
>![[Pasted image 20241214190116.png|450]]
>- **Classical IMHA** : 재생성, spherocytes, Coombs 양성, 황달 등
>- **NRIMA/PIMA** (비재생성 IMHA/precursor 상태에서 IMHA) : 비재생성, <u>maturation arrest</u>, myelofibrosis가 조금 더 흔함, 다른 세포에 영향 존재
>- **PRCA** : 비재생성, myelofibrosis 덜 흔함, <u>Erythroid precursor가 거의 없음</u>, 다른 세포에 영향 거의 없음

- Erythroid precursor에 대한 항체의 친화력(affinity)가 병의 심각도를 결정한다.
- Secondary causes
	- Dogs<br>(1) 신장 기능이 떨어져 Epo 활성 감소 → 재조합 human [[erythropoietin|EPO]]를 반복 주사하다가 면역 반응이 발생<br>(2) ==Parvovirus infection== → 개의 적혈구에 영향을 미침
	- ==Cats== : Feline leukemia virus subtype C (==고양이에서도 속발성으로 나타날 수 있음==)
- Dogs with PRCA : IMHA와 비슷한 증상
- Cats with primary PRCA: 개보다 더 어린 나이에서 나타난다. (8개월~3년령)
- 심각하고 비재생성인 빈혈을 나타내나, 다른 검사 수치에서 거의 이상을 나타내지 않는다. (platelet count, leukogram, biochemical panel, urinalysis 등 / 말초 용혈이나 염증 소견도 없음)
- 종종 적은 수의 spherocytes도 나타난다.
- Coombs tests는 대부분 음성이나, 양성일 수도 있다.
- Diagnosis : bone marrow aspirate & core biopsy
	- Erythroid precursor 없거나 rare
	- M:E ratio가 높음 (>99:1)
	- ==심한 myelofibrosis는 드물다.==

# 2. Treatment
- IMHA와 유사하나, PDS + 보조 면역억제제를 사용했던 IMHA와 달리 PRCA에서는 ==PDS 단독 치료에 대한 반응이 좋은 편==.
- 보통 **PDS 단독 치료** → ==종종== 반응 안 좋을 때 ==병용 약물 이용==.
- Adjunctive immunosuppressive : azathioprine, <u>cyclophosphamide</u>, cyclosporine, mycophenolate 를 사용할 때도 있음.
- 치료는 2-6개월 소요되고, 증상 진행이 느리듯 개선 반응도 느리다.
	- 약물이 듣지 않는 건지, 골수가 치료에 반응하기까지 시간이 더 필요한지 판단을 요구함.
- ==**골수 검사**==가 필요하지만, 현실적으로 쉽게 하지는 못함 : 원칙적으로 약물 변경하려고 할 때 골수 검사를 시행해야 한다. 2개월 이상 개선이 없으면 골수 검사가 권장된다.
- 항응고제(anticoagulant)는 지시되지 않음 : PRCA에서는 IMHA에서와 달리 TE(혈전색전증) 위험 증가나 systemic inflammation은 나타나지 않는다.
- 예후
	- Dogs : ==mortality <20%, IMHA보다는 예후가 좋다==. 치료 비용이 높아 안락사되는 경우가 많다 😿
	- Cats : 개랑 비슷하지만, 조금 더 반응이 빠르고(1.5~5주), 재발 가능성도 좀 더 높다.
		- Azathioprine 고양이에서 금지 → 보조 면역억제제로 cyclophosphamide와 cyclosporine 사용한다!
