### 각막

- basement membrane
- 기질층 : 중간에 두껍게 자리잡음
- 내피세포 : 한개의 층, 매우 얇게 존재

>[!question]- (국시 기출) 각막의 층은 총 몇 개?
> → 총 4개 층 (anterior epithelium, connective tissue stroma, Descemet's membrane, endothelium)
> - anterior epithelium과 basement memb.은 따로 구분하지 않는다.

| 각막의 층 구조                   |                                                                                                                                                                                                                           |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1) **Corneal epithelium**  | - outer layer는 2-3층의 squamous cells<br>- middle layer : polyhedral, wing cells<br>- inner layer : 아래쪽에 basement membrane 존재. hemidesmosomes로 부착됨. (tight junction)<br>- 층의 turnover가 많음 → 회복 속도가 매우 빠름. 다 벗겨내도 2-3일이면 회복. |
| 2) **Stroma**              | - 가장 두꺼운 층<br>- corneal keratocytes, 분열 거의 안 함<br>- 매우 느린 turnover rate를 가짐 → 웬만하면 새로 X                                                                                                                                   |
| 3) **Descemet's membrane** | - 각막 내피세포가 평생 동안 만듦. ==점점 두꺼워짐!==                                                                                                                                                                                         |
| 4) **Endothelium**         | - Na-K pump 많음. 물이 빠지면 Na-K pump 드러남.<br>- 가장 얇은 세포층이다.<br>- 나이가 들수록 얇아진다.<br>- 높은 metabolic activity, 단일 세포층, 각막 투명도를 유지함.                                                                                               |
| **Sensory innervation**    | - Trigeminal nerve (Ⅴ번).<br>- 각막 외피, 앞쪽 기질층 손상이 심층 손상보다 더 아픔. (basement memb.에 신경 있어서)                                                                                                                                    |
