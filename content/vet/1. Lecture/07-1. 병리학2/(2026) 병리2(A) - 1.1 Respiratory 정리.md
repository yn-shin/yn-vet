# 1. Structure and Function

|            | Conductive System                                                    | Transitional System | Exchange System             |
| ---------- | -------------------------------------------------------------------- | ------------------- | --------------------------- |
| Structures | nasal cavity, paranasal sinus, pharynx, <br>larynx, trachea, bronchi | bronchioles         | alveolar ducts, alveoli     |
| Cells      | Pseudostratified ciliated columnar epithelium & goblet cells         | Club (Clara) cells  | Type Ⅰ / Type Ⅱ pneumocytes |

#### Conductive System

>[!check]
>- Lining : 

코부터 기관지까지는 (외부 물질을 걸러내는 역할을 하는 곳이다).  **pseudostratified columnar epithelium**으로 이루어져 있으며, 특징으로 'ciliated cell'과 'goblet cell'을 기억하자.

(상피세포는 기본적으로 바닥(basement membrane)에 붙어 있다.)

Ciliated cells가 이물을 배출하고, goblet cells(술잔세포)는 점액을 배출한다. (이때 섬모는 '배출하는 방향으로' 움직인다; escalating)

#### Transitional System

>[!check]
>- Structures : Bronchioles (Primary, Secondary, Terminal // respiratory bronchiole)
>- Key cells : **Club (Clara) cells** → Secretion, detoxification, progenitor functions
>- Distal bronchioles : Decrease in ciliated cells, but increase in Club cells

세기관지가 이룬다. 분지하는 쪽과 멀어질수록 terminal이라 한다. Pseudostratified ciliated columnar epithelium이 이어지고 있다. 분비세포인 Club cell(Clara cell)이 여러 가지 역할을 한다.

(이렇게 볼록 튀어나온 동그란 세포가 club cell이다.)

- Antibacterial (세균 방어)
- Surfactanat (윤활 작용; 매끄럽게 함)
- Antioxidant (항산성 유지)
- Stem cell (분화해서 손상된 세포를 다시 채움)

Bronchiole의 끝으로 갈수록 ciliated cells는 줄어들고, 대부분 club cells로 채워진다.

#### Exchange System

>[!check] 
>- Structures : Alveolar ducts, Alveoli
>- Cell types :
>	- Type Ⅰ pneumocytes (gas exchange)
>	- Type Ⅱ pneumocytes (surfactant production, regeneration)

실질적인 기체 교환을 담당하는 exchange system은 alveolar ducts, alveoli가 이룬다. Terminal bronchiole에서 alveoli로 넘어갈 때 미세한 duct들이 있고, 말단에서 수백만 개의 alveoli로 연결된다.

폐의 주요 세포는 type Ⅰ, type Ⅱ pneumocyte가 있다.
- **Type Ⅰ pneumocytes** : 실질적인 기체 교환을 담당한다. 매우 납작하게 생겼고, 손상에 취약하다.
- **Type Ⅱ pneumocytes** : Surfactant 생성과 regeneration을 담당한다. Type Ⅰ보다 훨씬 통통하고 round(polygonal)하게 생겼다. Type Ⅰ pneumocyte가 손상되면 type Ⅱ가 세포분열을 해서 채운다.


Alveolar wall의 구조를 확대해서 보면, 혈관내피세포 1-2개로 둘러싸인 작은 혈관에 type Ⅰ pneumocyte가 맞닿아 있다.

이렇게 alveolar epithelium, baslal lamina, capillary endothelium으로 구성된 alveolar wall을 "**Blood-Air Barrier**"라고도 한다.
