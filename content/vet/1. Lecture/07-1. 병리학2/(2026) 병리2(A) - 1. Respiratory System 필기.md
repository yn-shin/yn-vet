# 1. Structure and Function

- 코 ~ 기관 ~ 폐
- 세 가지로 나눌 수 있습니다.
	- Conductive system : nostril ~ bronchi (nostrils, nasal cavity, sinuses, nasopharynx, larynx, trachea, bronchi)
		- hilar lymph node : 면역 반응 시 activation된다(커진다)
	- Transitional system : bronchiole
	- Exchange system : alveolar ducts, alveoli
- System에 따라 lining cell이 달라진다.

2) Bronchiectasis (기관지확장)
영구적 확장을 말함
정상적인 내강보다 훨씬 넓어짐 
 bronchus를 이루는 세포들이 회복 불가능한 수준으로 손상될 때

계속된 염증 → 세포의 재생 불가, 염증 세포가 점유, 구조적 변화 발생

연골에서는 chondromalacia : 연골이 녹아버림.
bronchus의 연골이 계속된 염증으로 인해 cartilage의 괴사도 일어남. ⇒ 원형 모양이 유지되도록 틀을 잡아주는 역할이었는데, 연골이 죽으면서 모양이 유지되지 못하고 늘어짐, 확장.

(proteolytic enzyme이 smooth muscle, cartilage 등을 녹여버림 ⇒ bronchiectasis
이것은 비가역적 변화임. 다시 복구될 수 없음. 다시 재생될 수 없는 영구적 손상.)

saccular type : 여러 개의 낭으로 생김 - 주로 국소적 손상 (그림은 국소다발적)
cylindrical type : 하나의 큰 실린더처럼 생김 - 주로 광범위 손상

# 3. Defense Mechanisms

- conducting
	- mucociliary clearance 중요; 점액으로 붙잡고, 섬모운동으로 내보내는 (escalation)
	- 항체, lysozyme
- transitional : club cell
- exchange system : alveolar macrophages, intravascular macrophage (간질에도 있고 alveolar 안쪽에도 )

바이러스 감염 ⇒ 직접적인 손상 말고도, defense mechanism을 무너뜨려서 2차적인 세균 감염이 일어나게 함.
(바이러스 감염된 개체에서는 세균이 잘 자람)


[ Conducting System ]
- mucociliary escalator
	- *Rhodococcus equi* : mesentery lymph node를 쨌을 때 농이 가득 찬, pyogranulomatous lymphadenitis; 호흡기로 들어와서 소화기로 넘어가는 세균 
	- 호흡기 유입된 균을 밖으로 내보내는데, trachea까지만 배출되거나, 침을 삼키거나 할 때 소화기로 넘어감.

- cilia : 1000회/1분 속도로 쳐냄

- M cell, APCs -> 항원을 BALT로 전달함. (염증 반응을 시작하는 신호 역할)

#### Bronchopneumonia 

(사진 위주 설명)

- 주로 세균, mycoplasma-(세균도, 바이러스도 아닌)가 원인체
- bronchiole에 영향을 주고 퍼져나감 
	- bronchial, bronchiolar, alveolar lumen이 영향을 받음
	- 헷갈리면 bronchus 쪽 내강의 병변 확인 (호중구 등 괴사소가 많음)
	- 경로를 생각했을 때 alveolar까지 병변이 퍼졌으면, bronchus 쪽도 심할 것
- cranioventral pattern인 경우가 많음 (반대로 바이러스는 작고 가벼워서 diffuse, caudodorsal pattern 많음)
	- firm, 붉거나 회색빛을 띈다.

#### Interstitial pneumonia

간질 = 대부분 alveolar wall (모세혈관, macrophage 존재)
- alveolar septa에 영향을 주고,
- 간질에는 모세혈관이 많기 때문에 원인체가 혈행성으로 전파될 수 있다. (toxin, 감염성 원인체, 다른 antigen 등)
- 미만성 패턴 
- surfactant protein이 제대로 기능 X → 호흡곤란이 더 심함 (호흡할 수 있는 공간 넓혔다 좁혔다 불가)

양에서 nasal adenocarcinoma는 비교적 흔하다.
tubular, papillary 모양을 띄고 있다. (glandular한 게 확장되고 자라다 보면 tubular, papillary 모양으로 바뀜)
adenocarcinoma라고 하는 게 정확한 표현.


\* 비강 종양은 일단 접근성이 어려워 치료가 어려움. 열어서 들어낼 수 없어서 punch biopsy로 조직을 보내는데, 그러면 괴사 부위, 염증 부위 등 달라서 한 지점을 찔러서 biopsy해오면 진단이 어려움.
\* 제거하기 어려운데, 연골을 잘 깨는 특징이 있다. 비갑개 연골이 얼마나 깨져있는지 = grade를 나누는 가장 중요한 지표! 비갑개 연골이 깨지면 바로 중추신경계로 연결된다. 연골을 뚫고 올라가면 바로 뇌전이(뇌침습)이다. 제거 어려움 + 위치상 중추신경계와 가까움 ⇒ 굉장히 위험한 종양!
