### Cardiac remodeling

- Preload → Volume overload → eccenteric hypertrophy
	- MVI, TVI, MMVD
	- PDA, VSD, ASD
	- "enlargement"
- Afterload → Pressure overload → concentric hypertrophy
	- PS, AS, MS(Mitral)
	- Systemic hypertension
	- Pulmonary hypertension (HW, PTE)
	- "bulging"

# 2. Heart diseases

### Congestive Heart failure

| CHF (Congestive Heart Failure, 울혈성 심부전)                                                                                                                                                                          |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1) **LCHF**                                                                                                                                                                                                      |
| LV → LA → Pulmonary vein hypertension → lung → PA                                                                                                                                                                |
| **Radiographic signs**<br>- <u>Pulmonary edema</u><br>- PV enlargement<br>- (cats) pleural effusion, patch alveolar pattern<br>- LA enlargement (backpack sign, double wall sign)<br>- alveolar pattern          |
| ![[Pasted image 20241219064905.png\|250]]![[Pasted image 20241219064917.png\|172]]                                                                                                                               |
| 2) ==**RCHF**==                                                                                                                                                                                                  |
| lung, PA → RV → RA → CVC → peritoneum                                                                                                                                                                            |
| **기전**<br>우심실에서 폐동맥으로 혈액을 충분히 내보내지 못해, 우심방과 체순환 정맥에 혈액이 정체됨.                                                                                                                                                     |
| **원인**<br>- 좌심부전 → 폐고혈압 → 우심실 과부하 → 우심부전 발생<br>- PS, PH, TVI 등으로 인해 발생 (=**대표 질병**)                                                                                                                              |
| **Radiographic signs**<br>- RA, RV enlargement<br>- mPA bulging (reversed D)<br>- Bilateral <u>pleural effusion</u> (interlobar fissure, scallop sign.. 등)<br>- Ascites, Edema, CVC dilation, Hepatosplenomegaly |

### MVI = MMVD (Myxomatous mitral valve disease)

| ==MMVD== (Myxomatous mitral valve disease, 이첨판폐쇄부전증)                                                                                                                                                                   |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1) LAE(left atrium enlargement), LAp 증가<br>	- 좀더 정확하게는 eccentric hypertropy<br>2) 연쇄적으로 LVE (left ventricle enlargement) <br>3) PVp 증가, pulmonary edema 발생<br>4) Tachycardia, preazotemia<br>5) Pulmonary hypertension |
