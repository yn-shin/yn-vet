# 기본
- IVDS는 정상적으로 **T10-11, L5-6**이 주변보다 좁다.
- Disc calcification은 연골이형성종(닥스훈트 등)에서 호발한다.
	- DISC 석회화로 인해 탄성이 감소하면 파열 가능성이 커지지만, 석회화 자체로 질병을 의미하지는 않는다.
# 선천적인 문제
## 1. Block vertebra
- 척추뼈가 붙어 하나의 덩어리처럼 보임. → IVDS 소실
- 주로 경추에 많이 발생.
- 척추사이 공간 조금 남아있으면 partial, 완전히 붙어 사라지면 complete
## 2. Cervical rib
- 말 그대로 경추에 갈비뼈가 있는 것. (suprenumerary rib)
- C7에서 가장 많고, C6이나 C5까지도 생길 수 있음. 
- Bilateral > Unilateral
- C7 에 날개 하나 더 생기는 식이라고 보면 됨.
- 보통 임상적 의미 없고, 통증 없음.

## 3. ==Hemivertebra==
- 선천적인 문제로 인해 척추뼈가 정상적인 직사각형의 모양을 갖지 못함.
- Brachycephalic dogs, screw-tall breed(돼지꼬리) 종에서 흔함.
- alignment가 어긋나며 위에 있는 spinal cord를 압박할 수 잇음.
	- 아래 scoliosis, kyphosis, lordosis를 유발하여 spinal cord 압박, 신경 병증 유발.
- X-ray상 외측에서 **wedge shape**, VD/DV에서 **butterfly sign**으로 나타남.
- DDx : vertebral body fracture
## 4. Vertebral alignment/posture
- Kyphosis : 등쪽으로 휘는 것
- Lordosis : 배쪽으로 휘는 것
- Scoliosis : S자로 휘는 것
## 5. Transitional vertebra
- 흉추가 요추, 경추가 흉추 같은 모양을 나타냄.
- Lumbosacral region에서 중요 → cauda equina syndrome
## 6. Spina bifida
- 척추뼈의 발생 과정에서 fusion이 덜 되어 두 개로 갈라지는 현상. spinal cord가 밖으로 빠져나올 수 있음.

# 퇴행성 질환
## 1. Spondylosis
- DISC에 퇴행성 변화가 생기면서 인근 DISC space에 osteophyte 생김.
- 다양한 곳에 발생 가능, intervertebral foramen을 좁아지게 할 수도 있고, spinal nerve에 영향을 줄 수도 있음.

# AAI (Atlantoaxial instability)
- Axis(C2)가 Atlas(C1)에 비해 등쪽으로 변이되면서 spinal cord가 눌림.
	- dorsal lamina가 평행하지 않고 각을 이룸.
	- dense에 의해 vertebral canal이 좁아짐.
- 증상 : 심한 통증, 걷지 못하고 한쪽으로 치우쳐짐, 고개 움직이지 못함.
	- 혹은 걷는 게 어눌하고, 옆으로 휘청거림, 목 통증.
- 소형견, toy breed에서 다발 / 선천적 이형성이나 외상에 의해 발생.
- ==영상 촬영 시 주의== : Flexed view로 찍으면 spinal cord가 더 심하게 눌릴 수 있으므로 최대한 neutral position으로 촬영.

# IVDD (Intervertebral disc disease)
- 외상, 퇴행성 변화 등에 의해 수핵이 탈출되어 spinal cord를 압박 → 신경 문제 발생.
