# 2. 흉부 방사선 판독 시 주의 포인트

- 갈비가 양쪽으로 겹치게 찍어야 함.
- 흉부는 **흡기**에 찍어야 함. (아니면 횡격막과 심장이 겹칠 수 있음.)
	- 복부는 호기에 촬영.

# 3. Positioning

# 4. Normal variation

| Inspiration                               | Expiration                                |
| ----------------------------------------- | ----------------------------------------- |
| ![[Pasted image 20241215014824.png\|250]] | ![[Pasted image 20241215014858.png\|250]] |
|                                           | - 폐 부분이 더 opaque (뿌얘보임)                   |
| ![[Pasted image 20241215014947.png\|250]] | ![[Pasted image 20241215015041.png\|250]] |
|                                           | - 횡격막이 더 위로 올라와 심장과 붙어 있음.                |

| Normal variation                                                                                                                       |
| -------------------------------------------------------------------------------------------------------------------------------------- |
| **Thymus (sail sign)**<br>![[Pasted image 20241215015339.png\|450]]<br>- 어린 동물에서 thymus가 남아있는 모양을 "sail sign"이라고 함. (모양은 다양)           |
| **Skin fold**<br>- 기흉과 헷갈리게 생겼지만, 자세히 보면 바깥이랑 이어지고 있음.                                                                                 |
| **CCJ (costso-chondral junction)** <br>![[Pasted image 20241215015502.png\|200]]<br>- 폐결절과 오인하지 않도록.<br>- 닥스훈트에서는 꺾어서 관절하기 때문에 더 두드러짐. |
| **Retrosternal fat accumulation**<br>: 심장 근처에 지방이 찰 수 있다.                                                                              |
| **Pulmonary osseus metaplasia** : 폐의 석회화 (노령성)<br>![[Pasted image 20241215015551.png\|200]]<br>(폐전이에 비해 너무 작고, opacity가 bone에 더 가까움)   |
| **Knob sign** <br>![[Pasted image 20241215020014.png\|200]]<br>- 나이 많은 고양이에서 대동맥이 구불해지고, 심장이 누움.                                       |
