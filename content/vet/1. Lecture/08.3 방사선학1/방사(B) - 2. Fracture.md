## Relative displacement of the bone fragments
- ==Avulsion fracture== : 인대, 힘줄이 붙는 자리에서 일어난 골절 - 골절편이 인대에 의해 당겨지는 traction이 발생함. 가장 다발하는 부위는 tibial tuberosity. 성장판 열려 있는 개체에서 호발. 치료법은 TTT(Tibial Tuberosity Transposition).
- Impacted fracture : 잘 없음. 부러진 골절편이 compression되어 낌.
- **Compression fracture** : 척추뼈에서 주로 일어나며, 앞뒤로 compression이 일어나 vertebral body 작아짐 - Hemivertebra 조심, vertebral canal 쪽 신경 압박해 신경증상 유발.
- Depression fracture : 주로 두개골에서 발생하며, 움푹 꺼지는 골절.

## Fracture complication
Bone healing이 정상적으로 이루어지지 않는 경우.

| 다양한 fracture complications                                                                                                                                              |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1) **Disuse atrophy → osteopenia(골다공증), muscle atrophy(근육 위축), joint stiffness(관절 굳음)**                                                                                 |
| 골절된 다리를 사용하지 않아 발생할 수 있는 증상들.<br>뼈가 radiolucent하게 관찰되고, 주변 연부조직이 얇고 울퉁불퉁한 것이 전형적인 증상.                                                                                   |
| 2) **Device failure**                                                                                                                                                   |
| 3) **Implant failure**                                                                                                                                                  |
| 골절, gap이 생겨 흔들리거나 plate가 부러질 수 있음.                                                                                                                                      |
| 4) **Premature closure of growth plate (성장판 조기폐쇄)**                                                                                                                     |
| 성장판이 조기 폐쇄되면서 뼈가 휘어짐, 관절의 아탈구, 관절염까지도 이어질 수 있음.<br>(angular deformity, malunion..)<br>Distal ulnar에서 가장 흔하게 발생.<br>치료 - 닫힌 성장판을 열기는 어렵고, 막힌 쪽 뼈를 절단해서 길이 맞춤 (osteotomy) |
| 5) **Infection**                                                                                                                                                        |
| Open fracture, 외고정 시 감염 가능.<br>pain, sudden lameness, edema.<br>방사선상으로 <u>periosteal reation</u>, <u>radiolucent</u>한 영역 생김(bone lysis 일어나서)                            |
| 6) **Delayed union**                                                                                                                                                    |
| 정상적인 기간보다 유합이 늦어짐. 보통 수술 후 3달이 지나도 유합이 안될 때.<br>No sclerosis<br>**원인**<br>- 심한 soft tissue 손상<br>- 분쇄골절<br>- implant에 의한 혈류 공급 방해<br>- 적절하지 못한 수복<br>- old dogs (당뇨 등)  |
| 7) **Nonunion**                                                                                                                                                         |
| 붙지 않음. fracture ends 사이에 gap 넓게 형성, 골절면이 크고 둥글어짐(smooth).<br>골절면 끝에 sclerosis (하얗게 경화됨 (opacity 높음) → 혈관이 가지 못하고, 골화되었다는 의미. 수술할 때 여기를 잘라줌.)<br>                        |
| 8) **Atrophic nonunion**                                                                                                                                                |
| 뼈가 연부조직으로 대체되어 완전히 재흡수되며 소실. radiolucent하게 나타나고, 소실되어 수술 불가.                                                                                                            |
| 9) **Malunion**                                                                                                                                                         |
| 붙긴 붙었는데 이상하게 붙은 경우.<br>- Functional (기능상 이상 X)<br>- Non-functional (뼈나 관절 문제로 체중 부하에 문제)<br>- Angular / torsional deformity (뼈가 돌아서 붙음)                                 |
| - Valgus : 외측으로 휘어짐<br>- Varus : 내측으로 휘어짐                                                                                                                               |
