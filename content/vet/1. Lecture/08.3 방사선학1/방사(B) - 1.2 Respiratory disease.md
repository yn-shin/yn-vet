# 3. Lung

- 우 4엽, 좌 2엽으로 총 6엽이다.<br>
  ![[Pasted image 20241215043924.png|300]]
- 외측상은 겹쳐 보여 선호되지 않고, DV나 VD - 특히 **VD**가 선호된다.
	- DV : 안정된 자세, 기흉 촬영
	- VD : 흉수 촬영
	- 호흡곤란, 기흉 의심될 때는 DV 촬영 (자세 뒤집으면 힘들어함)


| ✳️ **Bronchial pattern** (기관지 패턴)                                                                                                                                                                                                                         |                                                                                        |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| 염증 세포 침윤으로 기관지벽 두꺼워짐<br>- Doughnut sign : end-on<br>- Tramline sing : side-on<br><br>**원인**<br>- Bronchitis (만성)<br>- Allergic (<u>Feline asthma</u>, 알러지성 폐렴)<br>- Immune-mediated (PIE)<br><br>**주의**<br>- 보인다고 해서 무조건 질병은 X<br>- 말단에서 다수 관찰되면 질병 가능성 ↑ | ![[Pasted image 20241215045311.png\|300]]<br>![[Pasted image 20241215045355.png\|300]] |
| 관련 질병 1) **CCB (Canine Chronic Bronchitis)**<br> - 만성적인 dry cough<br>- 내강이 좁아져 호흡 곤란                                                                                                                                                                      |                                                                                        |
| 관련 질병 2) **Bronchiectasis**<br>- 기관지가 확장되다가 탄력을 잃고 흐물해짐<br>- 만성적이며, 영구적. 치료 불가.<br>- 폐렴 발생하기 쉬움, 치명적.                                                                                                                                                     | ![[Pasted image 20241215045956.png\|300]]                                              |
| 관련 질병 3) **Asthma**<br><br>[**Feline Asthma**]<br>- Hyperlucent lung fields; "air-trapping" (까만 폐)<br>- Tenting diaphragm<br>- Bronchial patterns (기관지 두꺼워짐, 염증 ↑)                                                                                        |                                                                                        |

| ✳️ **Alveolar pattern**                                                                                                                                                          |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Fluid, blood, cell 등이 균질하게 차 있음<br>\* interstitial pattern에 비해 진하고 균질함.                                                                                                          |
| **Radiographic sign**<br>- Consolidation (폐경화; 공기 외 다른 게 참)<br>- Lobar sign (엽과 엽 사이 경계가 드러남)<br>- Air bronchogram; "Trees in the fog" (폐엽이 차서 기관지가 드러남)<br>- Cardiac effacement |
