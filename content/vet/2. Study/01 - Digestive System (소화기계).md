## 3.1 구조와 구성

소화기계는 다음과 같이 구성된다.

- 구강 → 인두 → 식도 → 위 → 소장 (십이지장, 공장, 회장) → 대장 (맹장, 결장, 직장) → 항문
- 부속기관: 간(liver), 담낭(gallbladder), 췌장(pancreas) ➡️ [[02 - Hepatobiliary System (간담도계)|02 - Hepatobiliary System (간담도계)]]에서 따로 보기로.

---
 
### 기능적 구분

| 구분 | 구성 | 역할 |
|------|------|------|
| 상부 소화기 | 구강, 식도, 위 | 섭취, 저장, 초기 소화 |
| 소장 | 십이지장, 공장, 회장 | 소화 및 영양 흡수 |
| 대장 | 결장, 직장 | 수분 흡수, 배설 |
| 부속기관 | 간, 담낭, 췌장 | 효소/담즙 분비, 대사 |



👉 구조–기능 연결  
- 소장 → 긴 길이 + 융모 → 흡수 효율 ↑  
- 대장 → 수분 재흡수 → 대변 형성  

---

## 3.2 세포/조직학적 특징



### 기본 구조 (벽층)

- mucosa (점막)
- submucosa
- muscularis externa
- serosa



### 주요 세포

#### 위 (stomach)
- parietal cell → HCl 분비
- chief cell → pepsinogen
- mucous cell → 점액 보호

#### 소장 (intestine)
- enterocyte → 흡수
- goblet cell → 점액
- Paneth cell → 항균

#### 간 (liver)
- hepatocyte → 대사, 해독
- Kupffer cell → 대식세포



👉 조직–기능 연결  
- villi + microvilli → 표면적 ↑ → 흡수 ↑  
- 점액 → 물리적 방어  
- 간세포 → 대사/해독  



## 3.3 생리와 기능

### 1. 소화 (Digestion)
- 기계적 + 화학적 분해

👉 이상 시:
- 효소 부족 → 소화 장애


### 2. 흡수 (Absorption)
- 소장에서 영양소 흡수

👉 이상 시:
- villi 손상 → 흡수 ↓ → 설사, 체중 감소


### 3. 장운동 (Motility)
- 연동운동 (peristalsis)

👉 이상 시:
- motility ↓ → 정체
- motility ↑ → 설사


### 4. 분비 (Secretion)
- 위산, 효소, 담즙

👉 이상 시:
- 과다 → 궤양
- 부족 → 소화불량



### 5. 장벽 기능 (Barrier)
- 점막, 면역

👉 이상 시:
- 장벽 손상 → 세균 침입

## 3.4 병리적 변화와 방어 기전



## 1. 염증 패턴
- 위염, 장염

👉 결과:
- 점막 손상 → 흡수 ↓ → 설사



## 2. 흡수 장애 (Malabsorption)
- villi 손상

👉 결과:
- 영양 결핍
- 체중 감소



## 3. 폐쇄 패턴 (Obstruction)
- 이물, 종양

👉 결과:
- 내용물 정체 → 구토



## 4. 궤양
- 점막 손상

👉 결과:
- 출혈, 통증



## 5. 간 질환
- hepatocyte 손상

👉 결과:
- ALT ↑
- 해독 ↓



## 6. 종양
- mass 형성

👉 결과:
- 폐색 / 기능 저하

## 3.4.1 Vomiting & Diarrhea (핵심 흐름)



### Vomiting

자극 (염증/독소/폐색)  
→ 구토 중추 자극  
→ 위 내용물 역류  

---

### Diarrhea

점막 손상  
→ 흡수 ↓  

OR

분비 ↑  

→ 수분 증가  

---

---

## 3.4.2 Malabsorption

villi 손상  
→ 표면적 ↓  
→ 영양 흡수 ↓  
→ 체중 감소  

---

## 3.4.3 Hepatic Disease

hepatocyte 손상  
→ ALT ↑  
→ 해독 ↓
