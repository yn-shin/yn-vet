>[!info] Dopamine D2 receptor
>- 십이지장 쪽에 분포함
>- 뇌에서 도파민이 너무 많이 분비되면 vomiting : 운동 능력이 촉진되어 과도한 장 운동
>- 도파민 차단제 : **metoclopramide**, **domeperidone**
>	- dopamine과 metoclopramide 동시에 쓰면 소용 없겠지~?
>- Metoclpramide는 <u>항구토제</u>로 매우 많이 쓰임 : 도파민 D2 수용체를 차단해서 구토가 이루어지지 않게 하는 것

>[!info] Serotonin 3 (5-HT₃) / Serotonin 4 (5-HT₄) receptor
>- Serotonin = 5-hydroxytryptamine, 5-HT
>- Serotonin 3 & 4 수용체 : 위장관 쪽 분포, 특히 4는 식도 쪽에 많이 분포
>	- Serotonin 3 차단제 = <u>항구토제</u> : **ondansetron**, **metoclopramide** (세로토닌과 도파민 수용체 둘 다 차단)
>	- Serotonin 4 차단제 = 식도 운동성 촉진 : **cisapride**, **mosapride**
