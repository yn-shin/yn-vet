Toll-like receptor. (막관통 단백질)
<u>선천 면역의 시작점</u>이 되는, 선천 면역계의 대표적인 PRR.

TLR이 ligand 인식 → NF-κB 등의 경로가 활성화 → cytokine (TNF-α, IL-6 등)과 Interferon 분비

| 종류   | 인식하는 ligand (병원체 종류)                      |
| ---- | ----------------------------------------- |
| TLR4 | LPS (그람 음성균)                              |
| TLR2 | peptidoglycan, lipoteichoic acid (그람 양성균) |
| TLR3 | dsRNA (바이러스)                              |
| TLR9 | DNA (세균, 바이러스)                            |
| TLR5 | Flagellin (운동성 세균)                        |
>[!check] 왜 "Toll-like" 일까?
>- 초파리의 Toll 유전자에서 유래된 이름.
>- 초파리에서 Toll 유전자는 감염에 대한 면역 반응에 관여하는데,
>- 포유류에서 유사한 수용체가 발견되며 Toll-like receptor라고 이름붙게 되었다.
